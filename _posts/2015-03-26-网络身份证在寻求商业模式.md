         <h1 class="page-title">
          网络身份证在寻求商业模式
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/1036" class="username" datatype="" href="/author/1036" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        王海涛
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <strong>
           （泡泡特约）
          </strong>
          今年第11期得《财新周刊》刊发了报道《
          <a href="http://weekly.caixin.com/2015-03-20/100793090.html" rel="nofollow">
           网络身份证来了
          </a>
          》，尽管它写得很晦涩，既认为网络身份证（eID）用于金融行业带有急迫性，又含糊地在它的安全性存疑与现实需求之间摇摆。通读下来，这篇文章是赞成网络身份证扩大的，即使在金融行业有风险。
          <br/>
          <br/>
          这篇报道的立意暂且不去考虑，阅读它的另外一个意外收获是，了解到网络身份证的技术发展状况。也就是说，
          <strong>
           网络身份证在技术上已经成熟，而且在360等网络公司已经有应用，除了金融行业，在快递、物流等行业上马使用，政府丝毫没有考虑到公民感受
          </strong>
          。
          <br/>
          <br/>
          通过闪烁其词的财新报道，可以知道网络身份证的使用流程及地点，可以得出这样的判断：这是一个可以实现对网络行为进行实时监控并留存记录的身份监测系统，它
          <strong>
           可以实现点对点的网络监测
          </strong>
          。
          <br/>
          <br/>
          尽管强调的是它的正当用途，但谁都知道，那只是它的一部分功能。
          <br/>
          <br/>
          也就是说，网络身份证的民用与国安用途，不是一个截然分开的功能。而且在二代身份证的使用中，这种区隔也不存在。更关键的是，以政府现在的行事风格、对法治“不是挡箭牌”的肆意侵害
          <strong>
           ，一般人是无法指望政府严格区分网络身份证的民用或国安用途
          </strong>
          。
          <br/>
          <br/>
          在全世界，没有国家将网络身份证用于金融行业，这并不是说技术不如中国，而是说它们在扩大网络身份证时面临一系列棘手的法律问题——但这个障碍，也可以说是保护公民权益的屏障，在中国是不存在的。
          <br/>
          <br/>
          当马云在向默克尔炫耀这种技术时，暴露了权贵精英的傲慢与蛮横。
          <br/>
          <br/>
          <strong>
           金融行业以及阿里巴巴之所以鼓动网络身份证在商业领域的使用，完全是为了服务其商业竞争的目的，争夺客户，从而在互联网金融这一块切出新的“蛋糕”。
          </strong>
          对当局而言，它们之所以忌惮这么做，绝不是因为保护国民隐私，不过是为了怕生出麻烦。
          <br/>
          <br/>
          韩国网络身份证的泄密、被劫持等负面案例，在财新的报道不是被用来证明“网络身份证的不可行”，反而被用来证明需要“立法推动”，这是报道中非常没有说服力、倾向性很明显的方面。
          <br/>
          <br/>
          <strong>
           在银行、阿里巴巴与国民权利三方之间，国民处在最弱势的地位。
          </strong>
          <br/>
          <br/>
          在这样的状况下，不仅财新的报道很可疑，公安部研发的网络身份证也更加可疑。实际上，
          <strong>
           在现有的网络监测系统下，没有网络身份证，照样可以准确地找到特定人，而且网民的上网记录都会被服务器上载——从这个角度看，网络身份证的用途实在可疑
          </strong>
          。
          <br/>
          <br/>
          也有可能是，网络身份证这个技术在现有的严密的网络监控前，并无国安上的必要用途。那么，将它用于商业金融及可以帮助商业竞争，还能盘活这项束闲置已久的技术。但是为了打消网络身份证对现有网络监测利益的触动，就声称它不会用于敏感用途。
          <br/>
          <br/>
          如果按照这个逻辑，就可以解释财新这个非常奇怪的报道，也能解释它的别扭之处。但归根结底，无论是以声张避开国安用途、专注商业开发，还是可能的给网络再加多一个监控网络，数以亿计的公民权益是没有被当做第一位讨论的，实际上被代表、被出卖。
          <br/>
          <br/>
          尽管网民在互联网使用中，早就近似一个透明人，政府可以轻易地进行监控和记录，所以
          <strong>
           网民对实名制的抵触已经从敏感降为麻木
          </strong>
          ——但网民的不在乎（实际上是无法在乎、没有能力争取）并不代表网络身份证就是天然合法的，可以一声不吭地使用。
          <br/>
          <br/>
          大陆近期曝光的国安部副部长马健勾结商贾，利用手中的国安技术授权，监控监听竞争对手、打击异己，在权贵争夺战中强取豪夺的腐败事迹。这个事例有力地证明对网络身份证负有恰当使用责任的强力部门，根本不具有被信托的资格，但这恰恰是网络身份证在内的国民ID的真实处境。
         </p>
        </div>
       </div>
      </div>

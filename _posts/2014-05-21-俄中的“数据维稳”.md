         <h1 class="page-title">
          俄中的“数据维稳”
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/11" class="username" datatype="" href="/author/11" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        阿蔼
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          全世界都在大谈大数据(Big Data)的力量。
          <br/>
          <br/>
          过去，大数据主要用于资讯科技行业，譬如说搜寻软件和互联网的广告——网络服务供应商会透过爬行软件，搜集和分析用户习惯，然后在用户的邮箱出现与其好爱相关的广告：假如你是一个家庭主妇，邮箱旁的广告大概会跟家居用品相关。
          <br/>
          <br/>
          近年大数据变成了商机，企业相信可以透过分析互联网与社交媒体的数据，推出迎合市场的产品，通向成功。
          <br/>
          <br/>
          更有媒体和政治团体，通过互联网热点话题的分析，或借势制造议题，或调整政团的政治取态。笔者就认识不少媒体人，对大数据主宰新闻议题设定的情况，表示担忧，因为这意味着新闻随市场的口味转动；相反，媒体老板可能会因为记者主动去调查的新闻，没有足够的“赞”，而不愿多投放资源。
          <br/>
          <br/>
          可是，由于大数据是由无数的用户上网习惯、兴趣而生成，若不小心处理相关的数据，可能会侵犯用户的隐私。
          <strong>
           而当国家力量介入，通过控制数据去制造舆论，又或以分析社交网络数据去瓦解反对派政治力量时，大数据将会成为最有强大的政治操控和维稳工具
          </strong>
          。
          <br/>
          <br/>
          <strong>
           俄罗斯互联网数据法
          </strong>
          <br/>
          最近，莫斯科政府就打算要求所有社交媒体在俄罗斯境内储存俄罗斯的用户数据 ​​。相关法案以“反恐”为名，要求所有每天用户访问量超过3000的个人博客平台，作媒体登记，以便管理；规定同时又强制海外的互联网公司，保存六个月的用户数据 ​​，供当局索取。该法案已在下议院通过，并将会在今年的8月1日实施。
          <br/>
          <br/>
          不符合规定的公司和网站，除了将被罚款外，网站会被查封，又或被过滤。这意味着，不愿意保存六个月用户数据、并向俄罗斯政府提交用户资料的境外网站，将会被封锁。而俄罗斯总统普京也在4月24日的记者会上，以国家安全之名，要求存有俄罗斯数据资料的境外数据中心搬到俄罗斯。
          <br/>
          <br/>
          当然，在云端数据技术下，普京的说法不可能实现，但一些俄罗斯的博客认为，
          <a href="https://advocacy.globalvoicesonline.org/2014/05/16/countdown-to-the-end-of-twitter-in-russia/" rel="nofollow">
           该法规会为俄罗斯设立封锁境外网站的防火墙提供基础
          </a>
          。
          <br/>
          <br/>
          这条法规当然引起俄罗斯网民反弹。
          <a href="https://advocacy.globalvoicesonline.org/2014/05/05/russian-internet-on-the-way-to-pyongyang/" rel="nofollow">
           著名博客Anton Nosik指出
          </a>
          ，新的法例是走向朝鲜的列车，把俄罗斯网络一下子切成内联网，中间没有停经北京。
          <br/>
          <br/>
          <strong>
           莫斯科 vs. 北京
          </strong>
          <br/>
          Anton Nosik的说法好像暗示，互联网的极权管治，
          <a href="https://pao-pao.net/node/26" rel="nofollow">
           终站在朝鲜
          </a>
          。其实，他想像中的直线并不存在，莫斯科、北京与朝鲜之别，只是极权管治模式之差异。
          <br/>
          <br/>
          若比较莫斯科与北京在互联网数据的管理，前者尝试以法去管治，在没有相关的法律基础下，Twitter、Facebook并没有被封锁，尽管这两大社交媒体已成为关键反对力量的大本营。目前Twitter全球约有两亿个登记用户，当中有一百万在俄罗斯，而全球拥有十三亿用户的Facebook，则有两千一百四十万在俄罗斯。
          <br/>
          <br/>
          <strong>
           北京的管治模式则以党为中心，通过政治力量培养本地大型的互联网公司，建立伺从关系(patron-client relation)，协助其从属企业赶走竞争者，让它们垄断国内市场
          </strong>
          。党政机关不单可以直接下达审查的命令，亦能自由进出使用数据，进行分析。
          <br/>
          <br/>
          故此，北京不用一刀切封锁所有境外网站，它只需要策略性地封锁一些网站，如Youtube、Twitter和Facebook，一方面帮助从属的互联网公司建立垄断的位置，一方面有助政治上的维稳。因为在法律以外操作，每当外交部回答记者有关中国互联网治理的问题时，仍坚持“中国政府没有审查互联网”、“中国互联网是自由的”等等。这样的“策略”思维，使中国互联网的管理上存在很大灰色地带，让不同力量进行搏奕。
          <br/>
          <br/>
          当然，我们不能美化西方国家的数据管治模式，尤其是斯诺登前年出走后爆出，美国国家安全部门在主要互联网服务供应商均设有后门，能分析其用户的通讯，以进行反恐。
          <br/>
          <br/>
          <strong>
           面对国家以稚稳与反恐之名入侵用户数据 ​​，民间反应确显得非常迟顿，大家欢天喜地的说大数据是力量，但这力量会由谁控制？受控制的会是谁？如何制约大数据入侵个人隐私的力量？
          </strong>
         </p>
        </div>

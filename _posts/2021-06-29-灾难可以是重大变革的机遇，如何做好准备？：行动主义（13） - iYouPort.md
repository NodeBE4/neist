  <p class="graf graf--p">
   冠状病毒防疫在全世界阻碍了反抗行动，当局仅以 “违反社交距离” 为由就可以禁止任何聚集（比如英国就是以防疫为由拒绝批准声援 Julian Assange 的守夜活动）。这让全世界的活动家们感到很沮丧。
  </p>
  <p class="graf graf--p">
   但政治和社会的动荡本身就可能是机会：历史上有许多例子表明，由于一个惊心动魄的历史事件改变了条件，从而促进了积极的变化。美国的大萧条和英国的二战都发生了这种情况，两个社会此后都在社会、经济和文化的进步变化方面实现了飞跃。
  </p>
  <p class="graf graf--p">
   鉴于其巨大的痛苦和折磨，并不是说谁都希望这种动荡。但是，大规模的历史事件并不征求我们的意见。它们就是如此。问题是，我们该如何看待它们？在英国剧作家克里斯托弗·弗莱的反战剧作《囚犯的睡眠》中有一句对白：“事务现在是灵魂大小。”
  </p>
  <p class="graf graf--p">
   恐惧可能会是压倒性的，导致人们可能会忽略不断扩大的机会。这就是为什么行动者需要保持彼此的连接，以扩大 “灵魂”，注意到机会的扩张。和您的队友一起学习过去的活动家们利用困难时刻的方法，可以帮助获得灵感。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     民权运动面对的挫败
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   1962年，在佐治亚州，奥尔巴尼运动痛苦地输掉了取消种族隔离的斗争，尽管群众广泛地参与了这场抗争。当时马丁·路德·金已经成为全国性人物，他和南方基督教领袖会议（SCLC）投入到支持学生非暴力协调委员会（SNCC）倡议的斗争中。
  </p>
  <p class="graf graf--p">
   后来，奥尔巴尼的警长劳里·普里切特一直在吹嘘 “击败金博士”，他在南方各地巡回演讲时向那些白人警察局长们介绍如何摧毁民权运动，这是他的自夸。
  </p>
  <p class="graf graf--p">
   金知道是时候重新充电了，重新评估并回到基本的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" rel="noopener" target="_blank">
    战略图谱中
   </a>
   。
  </p>
  <p class="graf graf--p">
   首先，他们发现了一个战略错误：选择了错误的目标（奥尔巴尼的政客而不是商界）。他们花了很多时间重新研究，也给组织重新充电。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    大胆地花时间重新充电 — — 这是处于挫折中的人们都需要的一个心得
   </strong>
   。是的，1962年南方种族主义的暴力一如既往地紧迫，但SCLC成功抵制了一个又一个绝望的打击。
  </p>
  <p class="graf graf--p">
   相反，
   <strong class="markup--strong markup--p-strong">
    他们想要的是在恰当的时刻再次有力地站出来，这意味着重新组合和磨练自己的战略技巧
   </strong>
   。赌注太大，不能被动地满足于道德见证，而道德见证也并不能改变什么。他们对历史的解读是，
   <strong class="markup--strong markup--p-strong">
    胜利是可能的，但行动者需要增加学习曲线，提高自己的游戏水平。
   </strong>
  </p>
  <p class="graf graf--p">
   SCLC的下一个大的战略是1963年在阿拉巴马州的伯明翰，在那里，他们的要求升级得如此强大，以至于他们对平等住宿的要求迫使华盛顿做出了回应。他们赢得了一项突破性的民权法案。
  </p>
  <p class="graf graf--p">
   伯明翰作为战斗地点是与传统思维相悖的，因为要想得到总统和国会的行动，你就得去华盛顿，对吧？
   <strong class="markup--strong markup--p-strong">
    但传统并不能帮助现实，高明的战略有时就是需要做到前所未有。
   </strong>
  </p>
  <p class="graf graf--p">
   相反，SCLC 及其盟友加入了伯明翰的基层活动家。他们用一场旷日持久的和戏剧性的运动使
   <strong class="markup--strong markup--p-strong">
    当时的一个主要工业城市
   </strong>
   失调，触动了当权者的命脉，足以迫使华盛顿采取行动。
  </p>
  <p class="graf graf--p">
   能够维持一个漫长甚至血腥的运动，意味着行动组织需要足智多谋和具有充分的创造性，作为一个强大的团队来运作。马克和保罗·恩格勒以伯明翰的戏剧性反抗故事写下了这本书《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.patreon.com/posts/wu-fa-yu-ce-yun-45358392" href="https://www.patreon.com/posts/wu-fa-yu-ce-yun-45358392" rel="noopener" target="_blank">
    这是一场起义
   </a>
   》，您可以免费下载阅读。
  </p>
  <p class="graf graf--p">
   因为历史给人们提供了更大的变革机会，可以利用这个时机，准备好采取真正的行动。这意味着要磨练行动者的战略技巧，放弃一次性见证事件的分心。这
   <strong class="markup--strong markup--p-strong">
    意味着要更加明确行动的愿景，这样就可以通过对行动者想要的东西的常识性描述来赢得更多的人加入您的反抗事
   </strong>
   业。而这意味着建立更有效的组织，让您的团队合作更有力，让你们的整体能力更加强大。
  </p>
  <p class="graf graf--p">
   此刻的资源，可以加强行动主义团体的力量。
  </p>
  <p class="graf graf--p">
   变革培训组织最近重新出版了 “
   <a class="markup--anchor markup--p-anchor" data-href="https://ebook.pdfbookslibs.com/1084715.pdf" href="https://ebook.pdfbookslibs.com/1084715.pdf" rel="noopener" target="_blank">
    变化时代的组织指南
   </a>
   ”，这本书可以作为帮助行动者提高效力的基础，寻求真正有效的方法。几十年来，“变革培训” 一直在为基层组织和劳工团体提供咨询和行动组织者研讨会，并分享其被称为直接教育的活动家友好型教学法。培训师发现，
   <strong class="markup--strong markup--p-strong">
    许多与他们合作的行动团体不仅需要技能，还需要新的结构来提高行动效力。
   </strong>
  </p>
  <p class="graf graf--p">
   这本书是由 George Lakey、Berit Lakey、Janice Robinson 和 Rod Napier 以高度集体的方式编写的。他们首先对在民权运动、反战运动、环境运动、LGBTQ团体、以及劳工组织和基层邻里组织中工作过的各种不同的行动者进行密集采访；询问他们，对于新生行动组织和历史较长的行动团体来说，什么是有效的方法。
  </p>
  <p class="graf graf--p">
   然后把采访的内容编织在一起，并加入了自己的经验。Berit Lakey 是性别平权运动和反种族主义运动的先驱，Janice Robinson 在哈林区建立了一个社区健康中心，Rod Napier 帮助创新学校、革新教育体制，George Lakey 是老牌活动家，擅长建立直接行动团体。他们也都做过咨询工作，帮助行动主义组织处理分歧冲突，发展更好的组织结构，以适应变革目标。
  </p>
  <p class="graf graf--p">
   这本书中充满了行动者的经验故事。虽然合作这本书的行动团体大多在美国，但是这本书的译本在埃及、塞尔维亚、泰国等许多其他国家出版，调动了全球行动者的灵感。例如，在泰国，一些基层团体使用这本书进行了超越传统等级制度的创新。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     行动组织可以学会处理更多的动荡
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   从黑人历史学家文森特·哈丁的著作《美国黑人争取自由的斗争》中得到启发，这本书的中心比喻是
   <strong class="markup--strong markup--p-strong">
    将反抗行动比作泛舟 — — 利用橡皮艇或者竹筏在时而湍急时而平缓的水流中顺流而下的一种挑战竞技。参赛者会遇到各种未知的风险，一切胜败都取决于快速敏锐的反应能力
   </strong>
   。就如我们在《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bf%a1%e6%81%af%e5%8f%aa%e6%98%af%e6%88%98%e6%9c%af%ef%bc%8c%e6%9c%89%e6%95%88%e8%a1%8c%e5%8a%a8%e6%89%8d%e6%98%af%e6%88%98%e7%95%a5%e7%9b%ae%e6%a0%87%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8/" href="https://www.iyouport.org/%e4%bf%a1%e6%81%af%e5%8f%aa%e6%98%af%e6%88%98%e6%9c%af%ef%bc%8c%e6%9c%89%e6%95%88%e8%a1%8c%e5%8a%a8%e6%89%8d%e6%98%af%e6%88%98%e7%95%a5%e7%9b%ae%e6%a0%87%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8/" rel="noopener" target="_blank">
    信息行动主义
   </a>
   》系列中介绍战略部署时使用的登山比喻，帮助您更清晰地预想行动过程。
  </p>
  <p class="graf graf--p">
   借此比喻，这本书各章的内容分为 “接近河流，强化皮筏， 利用水流转向， 和面对巨石”。这本书包含了各种典型的组织结构，并超越了它们，描述了替代方案。邀请读者在各种方案中进行选择，找到适合您的团体特点的方案。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    这里有具体的工具支持行动团队成员直接表达他们的分歧，不管是战略上的分歧还是身份上的分歧。目标是发展一个有凝聚力和团结的团队，能够应对压制或渗透
   </strong>
   。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    不要错过《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e7%9a%84%e5%9b%a2%e9%98%9f%e9%99%b7%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%e5%92%8c%e8%a2%ab%e6%b8%97%e9%80%8f%e7%9a%84%e6%81%90%e6%85%8c%e4%b8%ad%ef%bc%8c%e5%8f%aa%e6%98%af/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e7%9a%84%e5%9b%a2%e9%98%9f%e9%99%b7%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%e5%92%8c%e8%a2%ab%e6%b8%97%e9%80%8f%e7%9a%84%e6%81%90%e6%85%8c%e4%b8%ad%ef%bc%8c%e5%8f%aa%e6%98%af/" rel="noopener" target="_blank">
     如果您的团队陷在被监视和被渗透的恐慌中，只是因为您缺少了一件东西 — 什么是安全文化
    </a>
    》
   </li>
  </ul>
  <p class="graf graf--p">
   书中还提供了如何促进有效组织会议的技巧，在节省时间的同时做出决策，以及如何组建任务小组，在完成工作的同时培养学习曲线。太多的活动家忽视了士气的损失，因为重复 “老一套” 的做法只能产生平庸的结果。例如，提高士气的一个简单方法是每次会议用很短的时间去汇报工作。久而久之，就会形成一条学习曲线，会议就会有所改善，人们就会注意到士气开始提高。
  </p>
  <p class="graf graf--p">
   这本手册把组织实践放在建立社会运动的大背景下，可以激发人们的信心以进行重大变革。这意味着要让一个团体的注意力向外延展，如乔纳森·马修·斯马克所说的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%bf%90%e5%8a%a8%e5%a6%82%e4%bd%95%e5%88%a9%e7%94%a8%e6%88%8f%e5%89%a7%e6%80%a7%e6%8a%93%e4%bd%8f%e5%85%ac%e4%bc%97%e7%9a%84%e6%83%b3%e8%b1%a1%e5%8a%9b%ef%bc%9a%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89/" href="https://www.iyouport.org/%e8%bf%90%e5%8a%a8%e5%a6%82%e4%bd%95%e5%88%a9%e7%94%a8%e6%88%8f%e5%89%a7%e6%80%a7%e6%8a%93%e4%bd%8f%e5%85%ac%e4%bc%97%e7%9a%84%e6%83%b3%e8%b1%a1%e5%8a%9b%ef%bc%9a%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89/" rel="noopener" target="_blank">
    “超越合唱团”
   </a>
   ，同时也要通过对领导力发展的深思熟虑的关注向内审视。
  </p>
  <p class="graf graf--p">
   大流行病的动荡正在加快所有社会的变革步伐。这意味着机会，当然，仅仅是对于那些懂得利用机会的人来说。纳奥米·克莱因曾指出 “灾难资本主义” 的概念，
   <strong class="markup--strong markup--p-strong">
    您的对手正试图通过像这样的动荡来最大化他们的利润，反抗者急需倾注时间和精力尽快学习如何将机会拉向自己一边。
   </strong>
  </p>
  <p class="graf graf--p">
   这本书就支持这一点，您可以和您的团队成员一起阅读，并安排交流对话，也许能带来更多灵感，以增强你们的力量。⚪️
  </p>
  <p class="graf graf--p">
   <em class="markup--em markup--p-em">
    —— 未完待续 ——
   </em>
  </p>

         <h1 class="page-title">
          许志永与中国网络公民的十年
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/11" class="username" datatype="" href="/author/11" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        阿蔼
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <strong>
           在农历新年前夕，新公民维权运动的骨干许志永被指扰乱公众秩序，判刑四年。这个案子，好像替始于2003年到2013年的公民运动定了罪。
          </strong>
          <br/>
          <br/>
          当年，全国的大学生透过互联网为黄静和孙志刚鸣冤，三名法学博士：许志永、俞江和滕彪，上书中央，促使当时新上任的领导人胡锦涛与温家宝，废除了收容遣送制度，民间与政府良性的互动，带来了新的希望。
          <br/>
          <br/>
          十年过去，许志永为了争取非户籍孩子于城市平等的教育权利，在教育部门外请愿，被法院判了四年。然而今天中国的互联网上，再难听到人们帮许先生发声，在所谓代表中国草根舆论平台的微博上，均是官方的立场，标题如：“许志永对当今世界的基本判断是错误的”、“支持中国异见人士，西方的‘阳谋’”等，至于他那万言表述新公民运动自辩书，当然遍寻不见。
          <br/>
          <br/>
          <strong>
           互联网上的结社自由
          </strong>
          <br/>
          自辩书里，他没有奢求要改变一党专政的政治制度，只求公民能有自由追求公义与爱的社会。众所周知，中国并没有结社自由，“人民”组织被党组织所控制。2003年许志永有份积极参与的一系列的公民维权事件，正是借互联网的力量，突破出社会控制的缺口，践实公民结社自由。
          <br/>
          <br/>
          当年，因为SARS 的袭击，校园实施隔离，全国的大学生透过互联网讨论地方政府隐瞒疫情所造成的灾难，同时对黄静与孙志刚之死议论纷纷，正如维权律师滕彪所描述的一样，当时几乎每个有影响力的维权事件，都以互联网作为平台，而接着的五年，是互联网公共领域的迅速发展期。
          <br/>
          <br/>
          其实自1990代末，已经有很多较大型的门户网站出现，如新浪、腾讯、百度等。不过，知识份子主要仍透过BBS的议政，大学内的如“水木清华”、“未名”等，校外的如“天涯社区”，这些平台凝聚了知识界对社会议题的观点，并酝酿接二连三的全国性事件，如1999年中国驻南斯拉夫领事馆后的全国反美示威，以及2003年的连串维权事件。
          <br/>
          <br/>
          2004年开始，更多商业性的门户网站出现，Google等互联网公司于2005进入中国，本地相对独立的搜孤及网易等商业入门网站也相继成立，此外，内聚性较强的社交媒体，如“校内网”（后改为“人人网”）和“豆瓣网”也开始风行。
          <br/>
          <br/>
          自2002年，第一个博客平台“博客中国”出现后，大大小小的个人和集体网络杂志纷纷出现，著名博客老虎庙就于2004年开始，通过博客记录身边发生的人事和物，罗永浩于2006年建立了集体书写的“牛博网”，翟明磊于则自2007年起通过"壹报"延续《民间》杂志的出版工作。
          <br/>
          <br/>
          直至2007年第一个微博平台“饭否”出现，把手机与互联网出版连接起来后，社会事件的疆界由地方瞬间传片全国，2007年6月的厦门反PX事件及当天的网络直播，就是“微博”威力的最佳展现。 2008年汶川地震的民间救援工作，微博也发挥了重要的统筹力量。
          <br/>
          <br/>
          <strong>
           网络打压加强
          </strong>
          <br/>
          2008年奥运结束后，知识界发起了《08宪章》联署，要求政治改革，触发一连串的政治打压。先是2009年初的反低俗运动，多个小型网站被迫关闭，“豆瓣网”的组群功能也被移除，到年中开始出现局部地方封网的情况，
          <a href="https://pao-pao.net/article/20" rel="nofollow">
           新疆更因乌鲁木齐的“七．五骚乱”被封网近一年
          </a>
          。 “饭否”就在骚乱后两天被封，其公共领域的位置被“新浪微博”取代。
          <br/>
          <br/>
          2010至2011年，中东和北非的苿莉花革命，再次触动了当权者的神经，并拘捕了大批网络异见份子。尽管打压激烈，但手段仍以关键词过滤、舆论倡导、事后惩罚为主，网民继续通过微博直播拆迁等社会事件，介入基层选举（如2011年的乌坎事件），又以调笑、擦边球的方法表达意见，更有实名检举官员贪污腐败。本来用于监控及制造寒蝉效应的实名制，反过来造就了一批具有影响力、相信公民力量的网络意见领袖。
          <br/>
          <br/>
          许志永的“新公民运动”就是承着过去十年网络培养出来、思想独立的主体而发展出来。“新公民”主体始于互联网，打击也从互联网开始。
          <br/>
          <br/>
          2013年4月，国家互联网信息办副主任任贤良指挥地方官员说："一些网络舆论被操纵，微博大V该关就关。"四个月后，网络大V被迫签规范言论的“七条底线”，接着开始拘捕网络造谣者，被捕者以千计。
          <br/>
          <br/>
          2013年9月，最高法院通过造谣诽谤一系列法令；其后“新公民运动”其中一位骨干王功权被捕，及后多名大V如薜蛮子等被捕。到10月底，舆情部门宣布打击网络造谣运动取得胜利，批评政府的声音大跌。微博因帖量大减而失去共公性，濒临死亡。
          <br/>
          <br/>
          不过，打击并没有宣示胜利的一刻停止，且有从网上延至网下之势。踏入2014年，“维吾尔在线”的创办人伊力哈木·土赫提于1月15日被捕，官媒指他发表分裂国家言论，许志永则于1月26日被判入狱四年。
          <br/>
          <br/>
          回首十年，足见这次被判罪的，不单是许志永，还有十年间通过互联网觉醒的网络公民。
         </p>
        </div>

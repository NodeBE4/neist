         <h1 class="page-title">
          茂名反PX人士白宫请愿：禁忌的解放
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/18" class="username" datatype="" href="/author/18" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        长平
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <strong>
           （泡泡网特约撰稿）尽管只能用苦涩的玩笑应付雾霾天气，但是越来越多的中国人走上街头，抗议政府在家门口建设大型化工项目。有一些项目在国外可能是安全的，但是他们不信任政府的管理能力，更担心官僚集团普遍的贪污腐败导致偷工减料及管理失控。而项目计划阶段的种种猫腻，往往已经给了他们这种担心足够的证据。
          </strong>
          <br/>
          <br/>
          然而，这些抗议行动随时面临中国政府的残酷打压。不给当局打压的借口，成为组织者颇费心机的事情。为此，2007年厦门市民上街抗议PX项目时，创造性地使用“散步”来替代“游行”（按照1989年“六四”事件后颁布的法律，游行需要批准，但从来不会获得批准），一直沿用至今。“扰乱社会秩序”也成为政府出警的常用理由，因此组织者都特别强调理性、纪律和克制。即便他们明明在抗议政府决策缺少民主程序，通常也会声明“只问民生，不谈政治”，甚至打出标语“拥护中国共产党”。
          <br/>
          <br/>
          <strong>
           茂名反PX挑战禁忌
          </strong>
          <br/>
          “勾结境外势力”更是一顶沉重的帽子。在中共的宣传中，无论什么理由，这种行为都罪大恶极。因此，
          <strong>
           哪怕国内媒体被宣传部门通令噤声，接受境外媒体采访也是一个禁忌
          </strong>
          。境外的环保NGO也会避免表达直接的关注和支持，哪怕那本身就是他们的使命。
          <br/>
          <br/>
          最新爆发、仍在进行中的茂名反PX抗议活动，向这种规则发出了大胆的挑战。4月1日，广州茂名同乡发起上千人的声援游行。游行队伍先到省政府门口表达抗议，然后到南方报业大厦前呼吁媒体公开报道，最后转道去美国广州领事馆求助。
          <br/>
          <br/>
          <strong>
           白宫请愿
          </strong>
          <br/>
          不仅如此，4月2日，有人在
          <a href="https://petitions.whitehouse.gov/petition/please-help-them-maoming-city-china-under-military-massacre-dozens-people-dead/wfP9qHYx" rel="nofollow">
           美国政府白宫网站发起联署请愿
          </a>
          。请愿书称，茂名人民因为抗议PX建设，于3月30日遭到政府武力镇压。“军事坦克和武装力量封锁了城市，数十人遭到杀害。城市的互联网被切断，中国媒体被强令沉默”。请愿书呼吁奥巴马政府及国际社会提供帮助，介入调查，追究凶手责任。目前，已有超过1300人在这份联署上签名。
          <br/>
          <br/>
          中国人到白宫网站请愿并非头一遭。去年发生的昆明抗议PX项目及成都反对彭州化工项目，均有人发起这项活动，但是言辞温和谨慎。其中有发起人被警方“喝茶”，公开求援删除白宫请愿书。清华大学“朱令案”重新调查的请愿，更在白宫网站上超过10万人签署。《环球时报》为此发表评论文章，称“
          <a href="http://news1.uschinapress.com/china/20130507/52860.html" rel="nofollow">
           白宫做不了中国的‘洋信访办
          </a>
          ’，它会很快被‘压垮’”。
          <br/>
          <br/>
          中国人没有把税交给美国政府，白宫的确做不了中国的“洋信访办”。除了基本的政治姿态，美国政府也很难实际处理中国的环保个案。但是，《环球时报》没有看到的是，
          <strong>
           白宫请愿是中国人冲破禁忌的自我解放
          </strong>
          。
          <br/>
          <br/>
          <strong>
           “勾结境外势力”的政治效果
          </strong>
          <br/>
          本来，当重庆市前副市长、公安局长王立军夜逃成都美领馆之后，当若干政协委员、人大代表都拥有外国绿卡或国籍风传之后，当拥有巨额财富的中共高层家族无一不在国外开设公司、转移财产、移民家属被曝光之后，“勾结境外势力”已成为一个滑稽的笑话。然而，官方媒体仍然在一本正经地使用它来恐吓民众，长期宣传积累的心理恐惧仍在在有效地阻止人们与世界进行正常的交流。
          <br/>
          <br/>
          在很多抗议活动中，政府仍然在威胁当事人说，一旦事情“被境外敌对势力利用”，变成了“政治事件”，就没有解决的余地了；并欺骗他们说，只要不家丑外扬，一切都好商量。不少抗议组织者或者受害者家属仍然在帮着政府提防外国记者，谴责那些“扩大事态”的“别有用心者”。他们仍然不能随时记得，
          <strong>
           那些没有国际舆论关注的反抗事件，往往遭到政府肆无忌惮的无情镇压
          </strong>
          。
          <br/>
          <br/>
          我曾经总结
          <strong>
           2011年乌坎抗议成功的经验，其中之一就是不受政府暗示，大胆接受境外媒体采访，让世界知道中国发生了什么
          </strong>
          。新媒体为打破传统的禁忌提供了方便。在过去，对于中国普通民众来说，“白宫请愿”是难以想象的事情。
          <br/>
          <br/>
          <strong>
           白宫喊冤因国内请愿无门
          </strong>
          <br/>
          网络不仅让中国民众可以向美国总统“喊冤”，更让他们看到，本应为他们申冤的中南海竟无请愿之门。民众“勾结境外势力”，更容易看清自家的真相，这也是专制政府阻止交流的一个重要原因。
         </p>
        </div>
       </div>

         <h1 class="page-title">
          断网？监控软件？占中的通讯问题
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/11" class="username" datatype="" href="/author/11" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        阿蔼
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <span>
           <span>
            在9月28日，警方使用催泪弹后，曾传出一个消息指警方已要求手机网络供应商，于晚上8点过后，暂时切断金钟一带的网络服务。
           </span>
           <span>
            学联即时呼吁，叫示威者在通讯切断后，终止行动。
           </span>
           <span>
            另一方面，又有网民呼吁下载
            <a href="https://pao-pao.net/article/194" rel="nofollow">
             Firechat
            </a>
            ，透过网状网络（mesh network），以便保持通讯。
           </span>
          </span>
         </p>
         <h2>
          <span>
           <span>
            断网？
           </span>
          </span>
         </h2>
         <p>
          <span>
           <span>
            究竟香港有没有可能如大陆的新疆，出现中断网络的情况呢？
           </span>
           <span>
            Firechat等网状网络又是否安全呢？
           </span>
           <span>
            此外，最近经常有新闻指监控软件入侵iOS和Andriod手机系统，我们又如何可以保护自己的通讯安全呢？
           </span>
           <br/>
           <br/>
           <span>
            资讯科技界的立法会议员莫乃光认为，
            <strong>
             香港出现终断网络的情况微乎其微。
            </strong>
           </span>
           <br/>
           <br/>
           <span>
            <strong>
             在法律的层面，有两条条例限制切断网络的情况出现
            </strong>
            ，包括《电讯条例》106章13条指，若行政长官认为已发生紧急情况，可签署手令接管或使用电讯电台，或对其加以
           </span>
           <span>
            控制，手令有效期不多于一周，但可以连续每周发出。
           </span>
           <span>
            条例没有列明何谓"紧急情况"。
           </span>
           <span>
            另一条《紧急情况规例条例》241章2条指，"在行政长官会同行政会议认为属紧急情况或危害公安的情况时，行政长官会同行政会议可订立任何他认为合乎公众利益的规例"
           </span>
           <span>
            。
           </span>
           <br/>
           <br/>
           <span>
            换言之，不管是全港性的封网，或者是局部地区封网，香港都需要进入“紧急状态”才能要求互联网服务提供者断网。
           </span>
           <br/>
           <br/>
           <span>
            此外，莫乃光指出，即使政府真的行使这两条条例，手令极难执行，因为香港不论是流动还是固网服务，均是开放市场。
           </span>
           <span>
            据香港贸易发展局的数据，香港共有20家本地固定电讯网络营运商（2013年6月），5家流动网络营运商（2012年12月），194家互联网服务供应商（2013年6月）
           </span>
           <span>
            ，若手令一旦发出，影响之大可想而知。
           </span>
           <span>
            而且，手令要直接下发给供应商的行政或技术总裁，换言之，海外的经营者在未收到手令前，不会断网。
           </span>
           <br/>
           <br/>
           <span>
            莫乃光补充，若政府真的要断网，也许可以在3G或4G服务供应商的层次出手，但一旦执行，连带电话服务也会终断，影响是无法衡量的。
           </span>
          </span>
         </p>
         <h2>
          <span>
           <span>
            FireChat的安全漏洞
           </span>
          </span>
         </h2>
         <p>
          <span>
           <span>
            虽然断网看来机会很微，但每逢有大型的示威或集会，都会出现网络大塞车的情况，9月28日当警察发放完催泪弹后，不少示威者均表示，他们手机的互联网连
           </span>
           <span>
            不上线，结果要从金钟跑到中环或湾仔才能发出讯息。
           </span>
           <br/>
           <br/>
           <span>
            故此，在9月28日当天和之后，不断有示威者下载Firechat以便在网络中断或塞车时使用。
           </span>
           <br/>
           <br/>
           <span>
            Firechat这应用程式所采用的技术是mesh network。
           </span>
           <span>
            因为每一部手机都发放出微弱的讯号，mesh network就是把手机变成一个节点，成为一个通讯网络，在大型的群众集会上可以互相通报讯息。
           </span>
           <br/>
           <br/>
           <span>
            不过，互联网私稳安全专家却警告指，
            <a href="https://pao-pao.net/article/194" rel="nofollow">
             这个软件有安全性漏洞
            </a>
            ，使用者可能会变成被攻击的对象，通话也可能被截听，而示威者的所在地也会被追踪。
           </span>
           <br/>
           <br/>
           <span>
            那么究竟有没有一些较Firechat安全，又可于网络大塞车时用的工具呢？
           </span>
           <br/>
           <br/>
           <span>
            香港专业资讯保安协会会长范健文直接地说：“没有”。
           </span>
           <span>
            他呼吁用户不要随便下载应用程式，亦不要Jail Break iphone 或ipad，因为这些动作都可能带来安全问题。
           </span>
          </span>
         </p>
         <h2>
          <span>
           <span>
            加密、不记录通讯
           </span>
          </span>
         </h2>
         <p>
          <span>
           <span>
            其实，大概两三个星期前，就有黑客冒充支持香港社运的科技人组织Code4HK的名义，
            <a href="https://pao-pao.net/article/173" rel="nofollow">
             发放Andriod系统的应用程式，该应用程式已被证明是监控软件
            </a>
            ，能盗取用户手机的资料，
           </span>
           <span>
            并于百度地图上追踪使用者的位置。
           </span>
           <br/>
           <br/>
           <span>
            昨天，路透社又引述网络安全专家指，有一个针对iphone 和ipad 的木马程式Xsser 正在散播，并怀疑发展该程式的黑客来自中国，并针对占中示威行动。
           </span>
           <br/>
           <br/>
           <span>
            范指出，其实很多程式都有安全漏洞，平时多加小心保护通讯工具。
           </span>
           <span>
            此外，若网络连接顺畅时，他会建议使用Telegram。
           </span>
           <br/>
           <br/>
           <span>
            Telegram是类似Whatsapp的通讯，而且所有通讯会被加密，用户可以选择永久删除通讯记录，以免手机或电脑被没收时，被翻查记录。
           </span>
           <br/>
           <br/>
           <span>
            这一点，有机会被捕的朋友要多加注意，因为没收手机已变成了警察对付示威组织者的惯技，更甚的是没收电脑。
           </span>
           <span>
            除了用Telegram，若使用Google Chat聊天，也可以按不记录（Turn off history）的选项。
            <br/>
            <br/>
            本文经授权转载自
            <a href="http://www.inmediahk.net/20141002a" rel="nofollow">
             香港独立媒体
            </a>
           </span>
          </span>
          <br/>
         </p>

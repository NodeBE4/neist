  <p class="graf graf--p">
   备份是确保在您的设备一旦被警察没收或丢失的情况下不会意外删除、损坏或失去您的数据和重要文档的关键。
  </p>
  <p class="graf graf--p">
   在断网或网速被变慢的期间，您可能无法运行常规的云备份或将文档发送到安全的异地位置。
  </p>
  <p class="graf graf--p">
   卸载到台式机或笔记本电脑上的确是一种备份方法，但是，由于人们通常无法访问单一的备份方法，因此，本文将介绍一些选项和提示，以便您可以
   <strong class="markup--strong markup--p-strong">
    在没有计算机的互联网关闭期间从手机备份您的重要媒体文档 —— 这些文档很可能是您揭露当权者罪行的证据
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    使用OTG或无线驱动器
   </strong>
  </h3>
  <p class="graf graf--p">
   OTG是一种USB驱动器，可与许多（虽然不是全部）Android兼容。您可以将OTG拇指驱动器直接插入手机，或使用 OTG-to-USB 将手机与常规USB硬盘驱动器连接。使用OTG，您的手机即可为驱动器供电。
  </p>
  <p class="graf graf--p">
   OTG 驱动器的流行品牌包括 SanDisk、金士顿和三星，尽管还有很多其他品牌。根据存储容量的不同，它们的价格通常在8–25美元之间。
  </p>

  <p class="graf graf--p">
   无线拇指驱动器/硬驱与常规硬盘驱动器相似，不同之处在于它们不需要电缆。这样一来，您就可以连接到通常不连接硬驱的设备，例如手机。
  </p>
  <p class="graf graf--p">
   无线驱动器优于OTG的特点是，您可以一次将多个用户连接到同一无线驱动器。例如，在您和团队合作拍摄抗议活动的过程中，这会很有用 —— 每个人的镜头都可以备份到其中一个团队成员随身携带的硬驱上。
  </p>
  <p class="graf graf--p">
   请注意，由于无线驱动器没有从设备中汲取电力，因此它们依靠电池供电，需要充电。
  </p>
  <p class="graf graf--p">
   SanDisk 可能是最受欢迎的无线拇指驱动器品牌，尽管还有其他品牌。无线拇指驱动器通常比OTG更贵，根据存储容量的不同，价格在25美元至100美元之间，还有更大的要150美元。具体取决于容量。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    备选：使用旧的没用过的手机
   </strong>
  </h3>
  <p class="graf graf--p">
   如果您没有OTG或无线驱动器，但是您的旧手机仍然可以继续工作只是不再使用了，那么您也可以将其重新用于备份。
  </p>
  <p class="graf graf--p">
   只要两部手机都在物理范围内，您就可以使用蓝牙、WiFi Direct 、或近场通信（NFC）/ Android Beam 将媒体从一个手机连接并复制到另一个手机。
  </p>
  <p class="graf graf--p">
   蓝牙和 Wifi Direct 都是无线技术，可以 “配对” 两个设备，而中间无需其他接入点。与蓝牙相比，WiFi Direct 可提供更广泛的范围和更快的数据传输，但会消耗更多电量。
  </p>
  <p class="graf graf--p">
   同时，NFC 的范围比蓝牙或 WiFi Direct 短得多（〜4cm），传输速度慢得多，但连接速度更快，耗电更少，因此在手握两个设备时进行快速的小型传输时非常有用。
  </p>

    <img alt="" class="size-full wp-image-13933" data-recalc-dims="1" height="768" sizes="(max-width: 1024px) 100vw, 1024px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4.jpeg?resize=1024%2C768&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4.jpeg?w=1024&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4.jpeg?resize=300%2C225&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4.jpeg?resize=768%2C576&amp;ssl=1 768w" width="1024"/>
   </noscript>
   <figcaption class="wp-caption-text" id="caption-attachment-13933">
    Send files offline over Files by Google.
   </figcaption>
  </figure>
  <p class="graf graf--p">
   ⚠️
   <strong class="markup--strong markup--p-strong">
    重要提示
   </strong>
   ：这些服务提供的易于连接的缺点是它们不安全。它们被可用于跟踪您的位置或探测设备以获取信息。
  </p>
  <p class="graf graf--p">
   渗透者可能会尝试与您的设备配对，向您发送不需要的文件，甚至在设备容易受到攻击时控制您的设备。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    为了更安全，请在不使用这些服务时将其关闭，而仅在您处于安全区域且必需时再将其打开，将应用程序权限限制为您所需的/仅需的对象，并练习良好的操作安全性，例如及时运行更新和增强性密码
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    包括任何单独的描述/元数据
   </strong>
  </h3>
  <p class="graf graf--p">
   将媒体文件复制到OTG、无线驱动器或旧电话时，包括任何可能与媒体文件分离的描述性信息或元数据，都非常重要。
  </p>
  <p class="graf graf--p">
   例如，许多文档应用程序都会生成CSV或JSON文本文档，其中包括从设备中提取的元数据（例如地理位置、时间、日期）以及用户手动输入的任何描述。请确保也将这些元数据文档导出并包括在备份中。
  </p>
  <figure aria-describedby="caption-attachment-13934" class="wp-caption aligncenter" id="attachment_13934" style="width: 1075px">
   <img alt="" class="size-full wp-image-13934 jetpack-lazy-image" data-lazy-sizes="(max-width: 1075px) 100vw, 1075px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5.jpeg?resize=1075%2C859&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5.jpeg?w=1075&amp;ssl=1 1075w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5.jpeg?resize=300%2C240&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5.jpeg?resize=1024%2C818&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5.jpeg?resize=768%2C614&amp;ssl=1 768w" data-recalc-dims="1" height="859" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5.jpeg?resize=1075%2C859&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1075"/>

   <figcaption class="wp-caption-text" id="caption-attachment-13934">
    Additional metadata files generated using ProofMode app.
   </figcaption>
  </figure>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    密码保护驱动器
   </strong>
  </h3>
  <p class="graf graf--p">
   许多无线驱动器可以通过驱动器随附的移动应用程序进行密码保护。请注意，密码保护与加密不同（参见下文）。
  </p>
  <figure aria-describedby="caption-attachment-13935" class="wp-caption aligncenter" id="attachment_13935" style="width: 1024px">
   <img alt="" class="size-full wp-image-13935 jetpack-lazy-image" data-lazy-sizes="(max-width: 1024px) 100vw, 1024px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6.jpeg?resize=1024%2C983&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6.jpeg?w=1024&amp;ssl=1 1024w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6.jpeg?resize=300%2C288&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6.jpeg?resize=768%2C737&amp;ssl=1 768w" data-recalc-dims="1" height="983" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6.jpeg?resize=1024%2C983&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1024"/>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    考虑加密文件
   </strong>
  </h3>
  <p class="graf graf--p">
   如果您需要更安全地存储文件，则可以考虑对备份进行加密。
  </p>
  <p class="graf graf--p">
   虽然您可能无法用手机加密大多数无线或OTG驱动器，但是可以在将文件移动到驱动器之前对其进行加密。
  </p>
  <p class="graf graf--p">
   可以在Android上加密文件的某些应用包括
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=ru.zdevs.zarchiver&amp;hl=en" href="https://play.google.com/store/apps/details?id=ru.zdevs.zarchiver&amp;hl=en" rel="noopener noreferrer" target="_blank">
    ZArchiver
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://play.google.com/store/apps/details?id=com.rarlab.rar" href="https://play.google.com/store/apps/details?id=com.rarlab.rar" rel="noopener noreferrer" target="_blank">
    RAR
   </a>
   。请注意，您必须记住您的加密密码。如果丢失密码，则无法恢复加密的文件。
  </p>
  <figure aria-describedby="caption-attachment-13936" class="wp-caption aligncenter" id="attachment_13936" style="width: 1080px">
   <img alt="" class="size-full wp-image-13936 jetpack-lazy-image" data-lazy-sizes="(max-width: 1080px) 100vw, 1080px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/7.jpeg?resize=1080%2C1276&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/7.jpeg?w=1080&amp;ssl=1 1080w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/7.jpeg?resize=254%2C300&amp;ssl=1 254w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/7.jpeg?resize=867%2C1024&amp;ssl=1 867w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/7.jpeg?resize=768%2C907&amp;ssl=1 768w" data-recalc-dims="1" height="1276" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/7.jpeg?resize=1080%2C1276&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1080"/>

   请记住，某些国家/地区可能有限制甚至禁止使用加密的法律。使用加密技术来防止当局访问您的数据可能被视为 “破坏证据或妨碍调查”，并可能构成犯罪。
  </p>
  <p class="graf graf--p">
   您可能需要先了解这点，参见这个地图
   <a class="markup--anchor markup--p-anchor" data-href="https://www.gp-digital.org/world-map-of-encryption/" href="https://www.gp-digital.org/world-map-of-encryption/" rel="noopener noreferrer" target="_blank">
    2017 map
   </a>
   。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在不同的位置进行2次备份
   </strong>
  </h3>
  <p class="graf graf--p">
   单个备份并不总是可靠的。例如，您可能会丢失、损坏备份设备，或者可能只是随机发生故障。
  </p>
  <p class="graf graf--p">
   IT专家通常建议人们在不同位置保存的不同设备上进行2次备份（即 共3份副本）。这有助于减轻任何一个特定副本的各种风险。
  </p>
  <figure aria-describedby="caption-attachment-13937" class="wp-caption aligncenter" id="attachment_13937" style="width: 1024px">

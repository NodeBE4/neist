  <p class="graf graf--p">
   如果你担心政府有能力对任何参与政治集会和抗议的公民进行
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e7%94%a8%e5%93%aa%e4%ba%9b%e7%9b%91%e8%a7%86%e6%96%b9%e5%bc%8f%e7%9e%84%e5%87%86%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e7%94%a8%e5%93%aa%e4%ba%9b%e7%9b%91%e8%a7%86%e6%96%b9%e5%bc%8f%e7%9e%84%e5%87%86%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f/" rel="noopener" target="_blank">
    全方位监控
   </a>
   ，甚至只是个人活动
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e7%94%a8%e5%93%aa%e4%ba%9b%e7%9b%91%e8%a7%86%e6%96%b9%e5%bc%8f%e7%9e%84%e5%87%86%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f/" href="https://www.iyouport.org/%e8%ad%a6%e5%af%9f%e7%94%a8%e5%93%aa%e4%ba%9b%e7%9b%91%e8%a7%86%e6%96%b9%e5%bc%8f%e7%9e%84%e5%87%86%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f/" rel="noopener" target="_blank">
    也会被这样监视
   </a>
   ，那么你应该知道，政府监控的技术能力即将成倍地扩大。
  </p>
  <p class="graf graf--p">
   美国空军的研究实验室（是的，它有自己的实验室）最近签署了一份合同，测试一家名为 SignalFrame 的公司的新软件，这是一家华盛顿特区的无线科技公司。该公司的新软件能够接入智能手机，并通过你的手机接入附近的其他任何无线或蓝牙设备。
  </p>
  <p class="graf graf--p">
   引用《华尔街日报》的文章，智能手机被用来 “作为数亿台电脑、路由器、健身追踪器、现代汽车、和其他任何联网设备的使用窗口，统称 ‘物联网’”。
  </p>
  <p class="graf graf--p">
   📌 简单说，你的智能手机实际上成为了政府的监听设备，可以检测并访问所有附近的无线或蓝牙设备，或者任何有MAC地址的设备。 “附近” 到底有多近，该公司和空军都没有透露，他们都拒绝发表评论。但随着5G无线网络的扩展，应该认为它
   <strong class="markup--strong markup--p-strong">
    不仅仅
   </strong>
   是离你的智能手机几步之遥。
  </p>
  <p class="graf graf--p">
   我们可以想象一下，这种能力掌握在政府窥探者手中，会出现多少可怕的场景。
  </p>
  <p class="graf graf--p">
   政府不仅会知道你的地理位置通过GPS信号追踪到你的手机，他们还会知道你在做什么、以及和谁在一起。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一个政治集会可以让政府看到抗议或示威活动附近所有其他手机的主人，看到有多少人聚集在一条特定的街道或地点，以及人们可能前往的方向，或者是否有一个组织在某个大厅或房间里开会，以及谁可能会参加。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    只要你把手机带在身上，就会这样。
   </strong>
  </p>
  <p class="graf graf--p">
   当你在蜿蜒曲折的道路上行驶时，
   <strong class="markup--strong markup--p-strong">
    他们可以
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%80%e4%ba%9b%e7%ae%80%e5%8d%95%e7%9a%84%e6%8e%aa%e6%96%bd%e4%bb%a5%e9%98%b2%e6%ad%a2%e6%82%a8%e7%9a%84%e6%b1%bd%e8%bd%a6%e8%a2%ab%e9%aa%87%e5%ae%a2%e6%8e%a7%e5%88%b6/" href="https://www.iyouport.org/%e4%b8%80%e4%ba%9b%e7%ae%80%e5%8d%95%e7%9a%84%e6%8e%aa%e6%96%bd%e4%bb%a5%e9%98%b2%e6%ad%a2%e6%82%a8%e7%9a%84%e6%b1%bd%e8%bd%a6%e8%a2%ab%e9%aa%87%e5%ae%a2%e6%8e%a7%e5%88%b6/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     入侵你的汽车的电路板，控制你的动力转向
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。或者他们可能会直接关闭你的汽车的电力系统，远程锁定你所有的车门，把你关在里面。警察已经不再需要骑着摩托车在路上追你了。
   </strong>
  </p>
  <p class="graf graf--p">
   新技术将允许政府在你慢跑时访问你的 fitbit 设备上的数据，或者更糟糕的是，甚至可以干扰你的心脏起搏器设备上的信号。
  </p>
  <p class="graf graf--p">
   这项技术可能会被用来访问你的智能手机，并从那里打开你的家庭 Alexa 类的设备，在你不知情的情况下监听和记录你说的一切。或者入侵你的笔记本电脑监视你参加的任何在线会议
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e6%ad%a2%e6%98%af-zoom%ef%bc%8cgoogle-meet%e3%80%81microsoft-teams-%e5%92%8c-webex-%e9%83%bd%e4%b8%8d%e5%ae%89%e5%85%a8/" href="https://www.iyouport.org/%e4%b8%8d%e6%ad%a2%e6%98%af-zoom%ef%bc%8cgoogle-meet%e3%80%81microsoft-teams-%e5%92%8c-webex-%e9%83%bd%e4%b8%8d%e5%ae%89%e5%85%a8/" rel="noopener" target="_blank">
    比如zoom
   </a>
   。或者更糟糕的是，关闭或绕过你家的安全系统。
  </p>
  <p class="graf graf--p">
   而
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/5g%e7%bd%91%e7%bb%9c%e7%9a%84%e5%8f%af%e6%80%95%e6%bd%9c%e5%8a%9b/" href="https://www.iyouport.org/5g%e7%bd%91%e7%bb%9c%e7%9a%84%e5%8f%af%e6%80%95%e6%bd%9c%e5%8a%9b/" rel="noopener" target="_blank">
    通过5G
   </a>
   ，当权者可以追踪的范围可能会远远超出蓝牙设备的范围。
   <strong class="markup--strong markup--p-strong">
    在 SignalFrame 的技术中加入5G，再将其与机器学习和人工智能的能力结合起来，你就可以对任何目标人物或任何集会的大量数据进行实时处理!
   </strong>
  </p>
  <p class="graf graf--p">
   政府对公民自由活动的监控这个问题并不新鲜。它采取了一个巨大的飞跃 ——
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e6%96%af%e8%af%ba%e7%99%bb%e5%88%b0%e4%b8%ad%e5%9b%bd/" href="https://www.iyouport.org/%e4%bb%8e%e6%96%af%e8%af%ba%e7%99%bb%e5%88%b0%e4%b8%ad%e5%9b%bd/" rel="noopener" target="_blank">
    “后911”时代
   </a>
   ，与
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8d%e8%ae%ba%e6%94%bf%e5%ba%9c%e5%85%b3%e9%97%ad%e4%b8%8e%e5%90%a6%ef%bc%8c%e8%ad%a6%e5%af%9f%e5%9b%bd%e5%ae%b6%e9%83%bd%e5%b0%86%e7%bb%a7%e7%bb%ad%e8%93%ac%e5%8b%83%e5%8f%91%e5%b1%95%e3%80%82/" href="https://www.iyouport.org/%e4%b8%8d%e8%ae%ba%e6%94%bf%e5%ba%9c%e5%85%b3%e9%97%ad%e4%b8%8e%e5%90%a6%ef%bc%8c%e8%ad%a6%e5%af%9f%e5%9b%bd%e5%ae%b6%e9%83%bd%e5%b0%86%e7%bb%a7%e7%bb%ad%e8%93%ac%e5%8b%83%e5%8f%91%e5%b1%95%e3%80%82/" rel="noopener" target="_blank">
    所谓的爱国者法案
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%94%bf%e5%ba%9c%e9%97%b4%e8%b0%8d%e5%a6%82%e4%bd%95%e7%9b%91%e5%90%ac%e5%b9%b6%e8%87%aa%e5%8a%a8%e8%af%86%e5%88%ab%e7%94%b5%e8%af%9d%e9%80%9a%e8%af%9d%e4%b8%ad%e7%9a%84%e6%95%8f%e6%84%9f%e8%af%8d/" href="https://www.iyouport.org/%e6%94%bf%e5%ba%9c%e9%97%b4%e8%b0%8d%e5%a6%82%e4%bd%95%e7%9b%91%e5%90%ac%e5%b9%b6%e8%87%aa%e5%8a%a8%e8%af%86%e5%88%ab%e7%94%b5%e8%af%9d%e9%80%9a%e8%af%9d%e4%b8%ad%e7%9a%84%e6%95%8f%e6%84%9f%e8%af%8d/" rel="noopener" target="_blank">
    大规模收集电话数据
   </a>
   的国安机构监视结合在一起。斯诺登的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" href="https://www.iyouport.org/6%e5%b9%b4%ef%bc%8c%e6%94%b9%e5%8f%98%e4%ba%86%e4%bb%80%e4%b9%88%ef%bc%9f/" rel="noopener" target="_blank">
    揭露之后
   </a>
   它应该已经停止，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%e6%9b%9d%e5%85%896%e5%b9%b4%e5%90%8e%ef%bc%8c%e5%a4%a7%e8%a7%84%e6%a8%a1%e6%8b%a6%e6%88%aa%e7%9a%84%e7%8e%b0%e7%8a%b6/" href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%e6%9b%9d%e5%85%896%e5%b9%b4%e5%90%8e%ef%bc%8c%e5%a4%a7%e8%a7%84%e6%a8%a1%e6%8b%a6%e6%88%aa%e7%9a%84%e7%8e%b0%e7%8a%b6/" rel="noopener" target="_blank">
    但它并没有停止
   </a>
   。窥探者继续无视国会决议和法院关于侵犯公民隐私的判决。
  </p>
  <p class="graf graf--p">
   最新的空军实验室测试很可能只是最近的 “冰山一角” 的启示。如果空军在这样做，请放心，陆军、海军、国安局、中情局、联邦调查局和其他所有政府窥探者，也都在这样做。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    当然，这种技术绝非只有美国政府会用。只要美国有的东西，你可以打赌其他政府也有 ——
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e4%bd%9c%e6%81%b6%e8%80%85%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%b8%ae%e4%b8%ad%e5%9b%bd/" href="https://www.iyouport.org/category/%e4%bd%9c%e6%81%b6%e8%80%85%ef%bc%9a%e8%b0%81%e5%9c%a8%e5%b8%ae%e4%b8%ad%e5%9b%bd/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     尤其是中国
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    、
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bf%84%e7%bd%97%e6%96%af%e6%ad%a3%e6%a8%a1%e4%bb%bf%e7%be%8e%e5%9b%bd%e6%a3%b1%e9%95%9c%e8%ae%a1%e5%88%92%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f%e2%80%8a/" href="https://www.iyouport.org/%e4%bf%84%e7%bd%97%e6%96%af%e6%ad%a3%e6%a8%a1%e4%bb%bf%e7%be%8e%e5%9b%bd%e6%a3%b1%e9%95%9c%e8%ae%a1%e5%88%92%e5%bb%ba%e7%ab%8b%e5%a4%a7%e8%a7%84%e6%a8%a1%e7%9b%91%e8%a7%86%e7%b3%bb%e7%bb%9f%e2%80%8a/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     俄罗斯
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    、
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e5%88%9b%e4%b8%9a%e5%85%ac%e5%8f%b8%e7%ad%b9%e9%9b%86%e4%ba%861250%e4%b8%87%e7%be%8e%e5%85%83%ef%bc%8c%e8%ae%a9%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e7%a0%b4%e8%a7%a3/" href="https://www.iyouport.org/%e4%bb%a5%e8%89%b2%e5%88%97%e5%88%9b%e4%b8%9a%e5%85%ac%e5%8f%b8%e7%ad%b9%e9%9b%86%e4%ba%861250%e4%b8%87%e7%be%8e%e5%85%83%ef%bc%8c%e8%ae%a9%e6%94%bf%e5%ba%9c%e5%8f%af%e4%bb%a5%e7%a0%b4%e8%a7%a3/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     以色列
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    ，可能还有一些欧洲国家
   </strong>
   。
  </p>
  <p class="graf graf--p">
   与2001年不同的是，在2020年，SignalFrame 的技术将政府监控提升到了一个
   <strong class="markup--strong markup--p-strong">
    全新的水平
   </strong>
   —— 借助于
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%ef%bc%9a%e6%99%ba%e8%83%bd%e6%89%8b%e6%9c%ba%e6%9c%ac%e8%ba%ab%e5%b0%b1%e6%98%af%e4%b8%8d%e5%ae%89%e5%85%a8%e7%9a%84%e4%b8%9c%e8%a5%bf%ef%bc%88video%ef%bc%89/" href="https://www.iyouport.org/%e6%96%af%e8%af%ba%e7%99%bb%ef%bc%9a%e6%99%ba%e8%83%bd%e6%89%8b%e6%9c%ba%e6%9c%ac%e8%ba%ab%e5%b0%b1%e6%98%af%e4%b8%8d%e5%ae%89%e5%85%a8%e7%9a%84%e4%b8%9c%e8%a5%bf%ef%bc%88video%ef%bc%89/" rel="noopener" target="_blank">
    智能手机
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" href="https://www.iyouport.org/category/%e5%85%b3%e4%ba%8e%e6%99%ba%e8%83%bd%e9%97%b4%e8%b0%8d%e5%ae%b6%e5%b1%85-%e5%9f%8e%e5%b8%82/" rel="noopener" target="_blank">
    物联网（IOT）设备
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%80%e4%ba%9b%e7%ae%80%e5%8d%95%e7%9a%84%e6%8e%aa%e6%96%bd%e4%bb%a5%e9%98%b2%e6%ad%a2%e6%82%a8%e7%9a%84%e6%b1%bd%e8%bd%a6%e8%a2%ab%e9%aa%87%e5%ae%a2%e6%8e%a7%e5%88%b6/" href="https://www.iyouport.org/%e4%b8%80%e4%ba%9b%e7%ae%80%e5%8d%95%e7%9a%84%e6%8e%aa%e6%96%bd%e4%bb%a5%e9%98%b2%e6%ad%a2%e6%82%a8%e7%9a%84%e6%b1%bd%e8%bd%a6%e8%a2%ab%e9%aa%87%e5%ae%a2%e6%8e%a7%e5%88%b6/" rel="noopener" target="_blank">
    依赖数字电路板的汽车
   </a>
   、以及现在所有具有MAC无线接入地址的家用设备的普遍性。
  </p>
  <p class="graf graf--p">
   并且，与2001年左右和臭名昭著的《爱国者法案》的通过（及其在每年的NDAA立法中的延续）不同的是，我们现在拥有
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" href="https://www.iyouport.org/%e4%ba%ba%e5%b7%a5%e6%99%ba%e8%83%bd%e6%96%b0%e5%86%b7%e6%88%98%e2%80%8a-%e2%80%8a-ai%e5%92%8c5g%e5%a6%82%e4%bd%95%e5%8f%98%e6%88%90%e4%b8%80%e5%9c%ba%e5%99%a9%e6%a2%a6/" rel="noopener" target="_blank">
    人工智能
   </a>
   、
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%8f%aa%e6%9c%89%e4%b8%a4%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%b6%b3%e4%bb%a5%e5%8e%8b%e5%88%b6%e5%8f%8d%e5%8f%9b%ef%bc%9aai-%e5%a4%a7%e6%95%b0%e6%8d%ae%e5%92%8c%e5%8f%8d%e5%8f%9b/" href="https://www.iyouport.org/%e5%8f%aa%e6%9c%89%e4%b8%a4%e4%b8%aa%e5%9b%bd%e5%ae%b6%e8%b6%b3%e4%bb%a5%e5%8e%8b%e5%88%b6%e5%8f%8d%e5%8f%9b%ef%bc%9aai-%e5%a4%a7%e6%95%b0%e6%8d%ae%e5%92%8c%e5%8f%8d%e5%8f%9b/" rel="noopener" target="_blank">
    机器学习
   </a>
   、无处不在的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8c%96%e6%8e%98ai%ef%bc%9a%e6%9c%ba%e5%99%a8%e5%ad%a6%e4%b9%a0%e8%ae%ad%e7%bb%83%e9%9b%86%e4%b8%ad%e7%9a%84%e5%9b%be%e5%83%8f%e6%94%bf%e6%b2%bb/" href="https://www.iyouport.org/%e6%8c%96%e6%8e%98ai%ef%bc%9a%e6%9c%ba%e5%99%a8%e5%ad%a6%e4%b9%a0%e8%ae%ad%e7%bb%83%e9%9b%86%e4%b8%ad%e7%9a%84%e5%9b%be%e5%83%8f%e6%94%bf%e6%b2%bb/" rel="noopener" target="_blank">
    神经网
   </a>
   络，以及庞大的政府数据处理能力。
  </p>
  <p class="graf graf--p">
   简而言之，
   <strong class="markup--strong markup--p-strong">
    科技正在成为政府手中越来越强大的工具和权力，用来
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%88%91%e4%bb%ac%e5%8e%8b%e6%a0%b9%e4%b8%8d%e6%83%b3%e8%a6%81%e8%bf%99%e4%ba%9b%e7%a8%8b%e5%ba%8f%ef%bc%9ajill-lepore-%e6%8c%87%e5%87%ba%e6%8a%80%e6%9c%af%e6%97%a0%e6%b3%95%e6%8b%af%e6%95%91/" href="https://www.iyouport.org/%e6%88%91%e4%bb%ac%e5%8e%8b%e6%a0%b9%e4%b8%8d%e6%83%b3%e8%a6%81%e8%bf%99%e4%ba%9b%e7%a8%8b%e5%ba%8f%ef%bc%9ajill-lepore-%e6%8c%87%e5%87%ba%e6%8a%80%e6%9c%af%e6%97%a0%e6%b3%95%e6%8b%af%e6%95%91/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     阻挠民主
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    、压制公民的宪法权利 —— 以及侦查、逮捕和 “处理” 那些抗议和反对政府的人。
   </strong>
  </p>
  <p class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%b8%8b%e4%b8%80%e4%b8%aa%e5%8d%81%e5%b9%b4%e5%8f%af%e8%83%bd%e4%bc%9a%e6%9b%b4%e7%b3%9f/" href="https://www.iyouport.org/%e4%b8%8b%e4%b8%80%e4%b8%aa%e5%8d%81%e5%b9%b4%e5%8f%af%e8%83%bd%e4%bc%9a%e6%9b%b4%e7%b3%9f/" rel="noopener" target="_blank">
    未来的十年不仅经济上越来越困难，政治上越来越不稳定
   </a>
   ，尤其是，它将被证明是一个技术越来越威胁到基本公民权利以及民主制度本身之根基的时期。⚪️
  </p>
  <p class="graf graf--p">

         <h1 class="page-title">
          Facebook和中国"亲密接触"
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/381" class="username" datatype="" href="/author/381" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        Kristo Helasvuo
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <strong>
           Facebook公司创始人兼首席执行官扎克伯格（Mark Zuckerberg）日前再次现身中国，22日在清华大学演讲更以中文答问。加上此前有报道Facebook在中国租办公室，不得不让人猜测Facebook将在中国有何大动作。扎克伯格如此高调出席中国精英学府， Facebook是否已和中国政府达成某些协议？
          </strong>
          <br/>
          <br/>
          社交媒体网站Facebook在中国被封已经有五年之久了。然而去年有报道称，上海自由贸易区将解禁Facebook和Twitter的事情，不过这些传闻很快就消散了。上海自由贸易区内并没有开放特殊的网络权限。然而，七月份，彭博社报道说Facebook在北京租了一间办公室。
          <br/>
          <br/>
          Facebook是否顺从了中国的审查要求，或是与监控者有其他合作，蓄势待发，准备向中国网络大众提供服务。可惜Facebook并未回应泡泡网记者的采访要求，具体细节还是无从知晓。
         </p>
         <h2>
          审查制度的要求
         </h2>
         <p>
          政府一旦掌握了屏蔽公共域名服务器请求的能力，便开始屏蔽掉一些他们觉得政治或者社会相关的网址。这不仅是中国独有的问题：譬如，芬兰也正在出台新的法规，希望在无需请示法院命令的情况下，就可以屏蔽或限制某些网络的IP地址。其他许多国家也在采用相似的屏蔽手段，比如泰国就屏蔽了一个貌似无辜的英国新闻网站。
          <br/>
          <br/>
          有人认为，社交媒体网站被屏蔽的原因是，它们拒绝服从政府的审查要求。还有人说是因为Facebook和其他几个社交网站过多的使用cookie来追踪用户浏览偏好（不管用户是否已登录）。审查同时也与不同政府治理文化和传统的本质有关。很多时候，审查成为了某个政党的政治工具，用来推进国内事务。每个政权都会审查，只不过形式不同罢了。
          <br/>
          <br/>
          网络监控和审查创造了电子世界的“阴影空间”，而这些空间的存在有助于推进社会的发展。Facebook是否会按照中国的方式，来定义阴影空间，如果是的话，会到什么程度。如果Facebook允许当局监控、删除及操纵内容，那确实可能获接入中国网络，解除屏蔽。Facebook在中国的一系列动作，也表明他们在按照这个方向筹划着。
          <br/>
          <br/>
          香港的占中运动在很大程度上是依赖Facebook来分享信息和组织集会，但扎克伯格此次在清华与中国学生“亲密接触”之后，香港方面有媒体担忧，是否以后这些信息将受到审查，亦或用户的实名信息将被分享给当局？
         </p>
         <h2>
          市场竞争
         </h2>
         <p>
          中国当局封锁Facebook也可能是纯粹经济的原因。如果他们也有自己的网络产品，那为什么将外国产品放进自己的市场？芬兰国际事务机构研究员 Elina Sinkkonen指出，一半中国人口都上网，尽管对政府的批评在某种程上是被容忍的，但是所有使用社交网络组织“线下集体活动”的活动都被系统性地审查了。从这个结论可以看出，社交媒体、乃至整个互联网控制的问题，实际上和商业或市场的关系都不大。如果从市场和经济的角度来看事物，Sinkkonen指出，中国“维稳”的预算甚至超过了国防的预算。但她并没有细说，维稳预算中究竟有多少是用来控制网络的。
         </p>
         <h2>
          用户的观点
         </h2>
         <p>
          政府约束网络服务，而广大用户对此怎么看？这个问题也和不同政体下人民和政府的关系有关。毕竟不同人对自由的定义有异。
          <br/>
          <br/>
          “对于某个被屏蔽的网站来说，如果不是很出名，人们甚至都注意不到它的消失”，致力调查泰国审查制度和群众反映的自由活动家Arhit Suriyawongkul如是说。“目前为止，（在泰国）仅有两个网站被封锁之后激怒了广大网民：Facebook和维基百科。”他指出，他并不知道人们是怎样发现网站被屏蔽的，翻墙是不是违法等。这两个网站在泰国后来都被解禁了，但有了前车之鉴，人们始终担忧这类事情会再发生。
          <br/>
          <br/>
          像他那样的专业活动家都可能不知道封锁的技术操作、或者相关的法律框架。这更说明，封锁更多是展示政府的能力，而不是一套完整的解决方案。
          <br/>
          <br/>
          至于火墙内的网民是否“想要”用Facebook，则是另外一个问题了。
         </p>

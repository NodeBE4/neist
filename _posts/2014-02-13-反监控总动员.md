         <h1 class="page-title">
          反监控总动员
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/17" class="username" datatype="" href="/author/17" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        荀春离
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          去年以来，美国国家安全局（NSA）便一直麻烦不断。由“棱镜门”引发的公众信任危机尚未结束，2月11日，一场全球反大规模监控抗议行动又在线上和线下同时展开，矛头直指NSA和非法网络监控。
          <br/>
          <br/>
          <strong>
           示威：逾
          </strong>
          <strong>
           6000
          </strong>
          <strong>
           家网站反对大规模监控
          </strong>
          <br/>
          “亲爱的互联网，我们厌倦了抱怨NSA。我们希望有新的法律能够限制在线监控。”
          <br/>
          <br/>
          如果你在2月11日这天访问Mozilla, Tumblr, Reddit等网站，一定没法不注意到这张位于网页上方的蓝黑色标语。从11日中午十二点开始，逾6000家网站的首页贴上了这个标语。
          <br/>
          <br/>
          这是一场名叫“
          <a href="https://thedaywefightback.org/international/" rel="nofollow">
           反击日（The day we fight back）”的示威行动
          </a>
          。行动的发起者是一群网络安全领域的活跃分子，这场行动的目的是为了纪念一年以前自杀的美国黑客奇才
          <span>
           阿伦·斯沃茨（Aaron Swartz）
          </span>
          ，和两年前针对美国国会众议院SOPA（反网络盗版法）以及参议院PIPA（知识产权保护法）的抗议行动。在2012年1月的那场抗议行动中，包括维基百科在内的数千家网站关闭24小时，促使美国国会宣布暂停SOPA和PIPA这两项法案的立法辩论。
          <br/>
          <br/>
          “我们相信这样的抗议可以带来现实上的改变。
          <span>
           阿伦·斯沃茨
          </span>
          向我们展示了，一个科技工作者在今天的社会可以拥有的力量、以及应该承担的责任。”美国青年Xander向泡泡网记者这样表示，他所在的Restore the Fourth in Los Angels是一家致力于捍卫公民权利的组织，他们为这次的“反击日”准备了近三个月，除了推广线上的示威活动，还在洛杉矶举行了线下的主题演讲会。
          <br/>
          <br/>
          <strong>
           联署：逾十万个跨国签名
          </strong>
          <br/>
          这场行动的矛头虽然直指美国国家安全局，但波及范围却不只局限于美国。
         </p>
         <p>
          对于美国境外的网民，网站会呼吁他们签署一份请愿书，反对大规模监控。目前已经搜集了超过十万个签名。
          <br/>
          <br/>
          各类线下活动也在世界各地纷纷展开。泡泡网记者在抗议网站上看到，来自哥伦比亚、哥斯达黎加、巴西、加拿大、丹麦、爱尔兰、印度、德国、波多黎各、塞尔维亚、南非、瑞典、奥地利、英国等14个国家的17个城市均有相关组织开展线下活动。
          <br/>
          <br/>
          斯德哥尔摩市的主题游行吸引了一百余人。游行组织者、同时也是世界上第一支
          <a href="http://www.piratpartiet.se/international/english" rel="nofollow">
           盗版党（Pirate Party）
          </a>
          的建立者Rick Falkvinge在接受泡泡网记者采访时表示，在游行文化并不流行的瑞典来说，一百人已经算是比较多的，这也展示了人们对网络安全问题的关注。“瑞典的网络安全状况近年来在走下坡路，大量无证窃听的存在影响了公众对政府的信任。我们希望通过这样的抗争行成国际舆论压力，在帮助美国网民的同时，也能对瑞典自身的状况有所帮助。我们希望这一代的抗争，可以为孩子们赢得一个更自由的网络空间。”
          <br/>
          <br/>
          <strong>
           担忧：抗争诉求能否转变为现实？
          </strong>
          <br/>
          这次的示威行动，尽管在全球范围内获得了较高的网民参与度，加上微软、谷歌、Facebook、Twitter等行业大咖的加入，使反击日的呼吁变得更加振聋发聩；然而，仍有人表示了担忧。
          <br/>
          <br/>
          “这只是一小步。”Verge的评论文章指出，“网络示威可以打破僵局，但这仅仅只是一个开始。”在此前的另一些示威行动中，大多数仅仅止于“示威”，而没有成功地进一步促使政府方面做出改变。
          <br/>
          <br/>
          另一个担忧的问题在于，事实上，示威者们也在疑惑，到底应该去哪里寻找改变，部门之间的踢皮球也常有发生。“当示威者们找到NSA，提出改变的诉求时，NSA会说，‘我们需要来自联邦政府的批准。’但实际上，并没有行动，到了下次示威时，再提一遍这样的‘批准’言论。”
         </p>
        </div>

    </span>
   </li>

  <p>
   <img alt="" class="aligncenter size-full wp-image-13958 jetpack-lazy-image" data-lazy-sizes="(max-width: 1049px) 100vw, 1049px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=1049%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?w=1049&amp;ssl=1 1049w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="700" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=1049%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1049"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13958" data-recalc-dims="1" height="700" sizes="(max-width: 1049px) 100vw, 1049px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=1049%2C700&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?w=1049&amp;ssl=1 1049w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/0-8.jpeg?resize=272%2C182&amp;ssl=1 272w" width="1049"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   去年6月，Telegram 在
   <a class="markup--anchor markup--p-anchor" data-href="https://telegram.org/blog/contacts-local-groups" href="https://telegram.org/blog/contacts-local-groups" rel="noopener noreferrer" target="_blank">
    其网站
   </a>
   上宣布，他们已简化了 “ 交换联系信息 ” 的过程。自从此更新以来，作为开源情报调查人员，您应该有可能根据其位置在 Telegram 上搜索群组和用户。
  </p>
  <p class="graf graf--p">
   这意味着，如果您自己 “在” 一个位置，就会能在搜索结果中找到
   <strong class="markup--strong markup--p-strong">
    同一位置的用户和群组
   </strong>
   的建议。
  </p>
  <p class="graf graf--p">
   您可以简单地检查哪个用户和群组在哪个区域中处于活动状态。
  </p>
  <p class="graf graf--p">
   您不需要坐着飞机到处跑才能实现这点 —— 您可以通过手动欺骗位置来解决这个问题。也就是手动设置。
  </p>
  <p class="graf graf--p">
   aware-online 介绍了如何做到这一点。《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.aware-online.com/telegram-groepen-zoeken-op-basis-van-locatie/" href="https://www.aware-online.com/telegram-groepen-zoeken-op-basis-van-locatie/" rel="noopener noreferrer" target="_blank">
    Telegram groepen zoeken op basis van locatie
   </a>
   》
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    您需要什么
   </strong>
  </p>
  <p class="graf graf--p">
   非常简单，就是：一个虚假手机、Telegram 的移动应用程序、和 Telegram 上的帐户。
  </p>
  <p class="graf graf--p">
   下面的演示将使用 “
   <em class="markup--em markup--p-em">
    Android emulato
   </em>
   r”，这是一种虚拟电话，其功能与普通电话相同。
  </p>
  <p class="graf graf--p">
   您也可以使用
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bignox.com/" href="https://www.bignox.com/" rel="noopener noreferrer" target="_blank">
    NoxPlayer
   </a>
   ，
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bluestacks.com/" href="https://www.bluestacks.com/" rel="noopener noreferrer" target="_blank">
    Bluestacks
   </a>
   或
   <a class="markup--anchor markup--p-anchor" data-href="https://www.genymotion.com/" href="https://www.genymotion.com/" rel="noopener noreferrer" target="_blank">
    Genymotion
   </a>
   。 选择哪种取决于自己的喜好。 例如，您可以在虚拟机
   <a class="markup--anchor markup--p-anchor" data-href="https://www.virtualbox.org/" href="https://www.virtualbox.org/" rel="noopener noreferrer" target="_blank">
    Virtual Box中
   </a>
   创建 Genymotion。或者您也许发现 NoxPlayer 更好或更快。 当然，您也可以使用预付费电话。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    开始 Telegram
   </strong>
  </p>
  <p class="graf graf--p">
   首先您需要下载 Telegram 移动应用程序。
  </p>
  <p class="graf graf--p">
   您可以通过 Android Play 模拟器（带有假的 Gmail 帐户）和 Google Playstore 来执行此操作，这取决于您所使用的设备。
  </p>
  <p class="graf graf--p">
   在此示例中将继续使用 Android Emulator NoxPlayer，因此，如果您使用其他设备，则步骤可能会略有不同。
  </p>
  <p class="graf graf--p">
   步骤1：打开虚拟设备
  </p>
  <p class="graf graf--p">
   步骤2：打开Goog​​le Playstore
  </p>
  <p class="graf graf--p">
   步骤3：下载 Telegram。如果一切正常，您的虚拟手机环境将如下图所示。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13959 jetpack-lazy-image" data-lazy-sizes="(max-width: 601px) 100vw, 601px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?resize=601%2C1002&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?w=601&amp;ssl=1 601w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?resize=180%2C300&amp;ssl=1 180w" data-recalc-dims="1" height="1002" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?resize=601%2C1002&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="601"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13959" data-recalc-dims="1" height="1002" sizes="(max-width: 601px) 100vw, 601px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?resize=601%2C1002&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?w=601&amp;ssl=1 601w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-9.png?resize=180%2C300&amp;ssl=1 180w" width="601"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   步骤4：点击左侧的三个点，然后选择 “ V-loc”；
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13960 jetpack-lazy-image" data-lazy-sizes="(max-width: 663px) 100vw, 663px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?resize=663%2C969&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?w=663&amp;ssl=1 663w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?resize=205%2C300&amp;ssl=1 205w" data-recalc-dims="1" height="969" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?resize=663%2C969&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="663"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13960" data-recalc-dims="1" height="969" sizes="(max-width: 663px) 100vw, 663px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?resize=663%2C969&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?w=663&amp;ssl=1 663w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-7.png?resize=205%2C300&amp;ssl=1 205w" width="663"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   步骤5：选择一个虚拟位置。本示例选择了荷兰的 “鹿特丹” 位置。单击右下角的 “
   <em class="markup--em markup--p-em">
    Move here
   </em>
   ” 以保存位置。
  </p>
  <p class="graf graf--p">
   注意：这不适用于所有 “普通” 手机。如果您不使用 Android 模拟器而是使用其他闲置的普通手机，则可以下载移动应用程序来欺骗位置。请注意在下载过程中需要小心避免遇到恶意软件。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13961 jetpack-lazy-image" data-lazy-sizes="(max-width: 739px) 100vw, 739px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?resize=739%2C631&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?w=739&amp;ssl=1 739w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?resize=300%2C256&amp;ssl=1 300w" data-recalc-dims="1" height="631" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?resize=739%2C631&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="739"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13961" data-recalc-dims="1" height="631" sizes="(max-width: 739px) 100vw, 739px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?resize=739%2C631&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?w=739&amp;ssl=1 739w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/3-3.png?resize=300%2C256&amp;ssl=1 300w" width="739"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   步骤6：打开 Telegram
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13962 jetpack-lazy-image" data-lazy-sizes="(max-width: 601px) 100vw, 601px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?resize=601%2C1002&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?w=601&amp;ssl=1 601w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?resize=180%2C300&amp;ssl=1 180w" data-recalc-dims="1" height="1002" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?resize=601%2C1002&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="601"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13962" data-recalc-dims="1" height="1002" sizes="(max-width: 601px) 100vw, 601px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?resize=601%2C1002&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?w=601&amp;ssl=1 601w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-4.png?resize=180%2C300&amp;ssl=1 180w" width="601"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   步骤7：点击左侧的三个条纹，然后选择 “联系人”；
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13963 jetpack-lazy-image" data-lazy-sizes="(max-width: 539px) 100vw, 539px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?resize=539%2C937&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?w=539&amp;ssl=1 539w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?resize=173%2C300&amp;ssl=1 173w" data-recalc-dims="1" height="937" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?resize=539%2C937&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="539"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13963" data-recalc-dims="1" height="937" sizes="(max-width: 539px) 100vw, 539px" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?resize=539%2C937&amp;ssl=1" srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?w=539&amp;ssl=1 539w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/5-3.png?resize=173%2C300&amp;ssl=1 173w" width="539"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   步骤8：点击 “添加在附近的人”，然后根据您的位置查看群组结果。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-13964 jetpack-lazy-image" data-lazy-sizes="(max-width: 538px) 100vw, 538px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?resize=538%2C936&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?w=538&amp;ssl=1 538w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?resize=172%2C300&amp;ssl=1 172w" data-recalc-dims="1" height="936" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?resize=538%2C936&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="538"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-13964" data-recalc-dims="1" height="936" sizes="(max-width: 538px) 100vw, 538px" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?resize=538%2C936&amp;ssl=1" srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?w=538&amp;ssl=1 538w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/6-4.png?resize=172%2C300&amp;ssl=1 172w" width="538"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   就是这样！您可以变换任何您想要的位置。在整个过程中不要使用您的真实手机。⚪️

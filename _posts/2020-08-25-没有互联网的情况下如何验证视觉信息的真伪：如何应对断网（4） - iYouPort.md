  <p class="graf graf--p">
   人权维护者、调查人员、研究人员、和记者，经常依靠证人拍摄的第一手文件来监测、报告和处理侵犯人权行为。为了确保根据正确的信息行事，就需要采取细致的认证流程以核实收到的每一份文件，这个过程可能是痛苦而耗费时间的。
  </p>
  <p class="graf graf--p">
   作为一个文档工作者，您有一些简单的事情可以做，让便别人可以更容易验证和印证你提供的文档信息，以便于及时有效地使用它们。
  </p>
  <p class="graf graf--p">
   而在断网期间，这几个额外的步骤就更有价值了。于是您应该考虑到：
  </p>
  <p class="graf graf--p">
   1、如果您不能立即上传，那么社交媒体提供的发布日期和位置信息对于显示您的视频是在某个日期、某个位置进行拍摄的帮助就很低。
  </p>
  <p class="graf graf--p">
   2、如果其他人（合作者）也不能上传，则总体上可用于证实您的视频的文档可能会更少。
  </p>
  <p class="graf graf--p">
   3、如果您需要使视频离线传输以到达所需的目的地，其他人可能很难跟踪视频的来源。
  </p>
  <p class="graf graf--p">
   4、如果由于安全性提高或存储容量有限而需要从手机中删除原始视频，或者如果您不得不放弃手机，则可能更难确认视频的真实性。
  </p>
  <p class="graf graf--p">
   5、如果您忘记了有关特定视频的详细信息，并且正在使用的应用无法在没有互联网访问的情况下捕获/记录元数据，那么其他人以后可能无法识别这一证据。
  </p>
  <p class="graf graf--p">
   以下提示可帮助您在断网期间维护视频证据，以最大程度地提高视频证据的可验证性和可用性。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    在视频中直接拍摄出或提供可识别性信息
   </strong>
  </h3>
  <p class="graf graf--p">
   尝试在视频中包含一些细节，以使调查人员或新闻工作者以后更容易利用开源情报思考方式确定时间和地点，例如独特的地标、天际线、路牌、店面、车牌、旗帜、钟表、报纸头版等。
  </p>
  <p class="graf graf--p">
   您也完全可以在拍摄过程中口述基本信息，例如您的姓名和联系信息（如果可以的话）、时间、日期和位置/ GPS坐标（或在纸上写下来然后对着纸面拍照）。
  </p>
  <p class="graf graf--p">
   您所包含的详细信息越多，其他人以后就更容易研究和验证您提供的视频证据，即使他们不认识您或不知道视频来自何处。更多参见下面的技巧：

  </p>

   <strong class="markup--strong markup--p-strong">
    保留备份
   </strong>
  </h3>
  <p class="graf graf--p">
   定期备份手机中的媒体，最好备份到2个单独的存储设备。例如，即使没有计算机，也可以将OTG 或拇指驱动器连接到手机。
  </p>
  <p class="graf graf--p">
   备份将确保您保留视频的副本，以防手机丢失或摔坏，或者需要从手机中删除视频。
  </p>
  <p class="graf graf--p">
   拥有原始视频的安全副本，还可以使调查人员或新闻工作者通过其他方式查看您的视频证据，以便稍后（只要他们能够将其追溯到您）直接从您那里获取原始视频，从而创建一个完整的监管链。

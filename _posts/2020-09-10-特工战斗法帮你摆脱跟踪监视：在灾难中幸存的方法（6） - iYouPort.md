  <p class="graf graf--p">
   老读者会知道，我们的
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">

   </a>
   ，其中收集的内容均为
   <strong class="markup--strong markup--p-strong">
    行动者在现实中与政府间谍过招的技巧
   </strong>
   。您可以用它们来防备任何恶意的侵扰和威胁，包括绑架、抢劫等等。
  </p>
  <p class="graf graf--p">
   在行动之前，恶意者一般都会跟踪、观察或监视他们锁定的目标一段时间。绝大多数绑架、抢劫和入室行窃案的受害人都在此之前受到过监视，时长从几分钟、几小时到几天或几周不等。
  </p>
  <p class="graf graf--p">
   于是您应该知道，
   <strong class="markup--strong markup--p-strong">
    学会摆脱监视、迅速判断自己是否被人跟踪并甩掉跟踪者，是防止侵害的关键
   </strong>
   。首先以最快速度意识到自己被跟踪，这样才能争取到足够时机采取行动以保证安全。
  </p>
  <p class="graf graf--p">
   现在您应该已经了解监视有哪些类型了，最基本的应该知道：跟踪、偷拍、有目的的社交媒体情报收集、包括黑客入侵，有些时候他们不仅针对您本人，还会针对您周边的人，您的亲密关系、家庭成员、要好的同事和朋友等等。
  </p>

  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     敏锐的态势感知
    </strong>
   </span>

  <p class="graf graf--p">
   您需要做的第一步就是以最快速度判断出是否有人在跟踪您。这里必须重申前文中介绍的基本
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b4%bb%e4%b8%8b%e5%8e%bb%e9%9c%80%e8%a6%81%e5%be%88%e5%a4%9a%e7%9f%a5%e8%af%86%e5%92%8c%e6%8a%80%e8%83%bd%ef%bc%9a%e5%9c%a8%e7%81%be%e9%9a%be%e4%b8%ad%e5%b9%b8%e5%ad%98%e7%9a%84%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e6%b4%bb%e4%b8%8b%e5%8e%bb%e9%9c%80%e8%a6%81%e5%be%88%e5%a4%9a%e7%9f%a5%e8%af%86%e5%92%8c%e6%8a%80%e8%83%bd%ef%bc%9a%e5%9c%a8%e7%81%be%e9%9a%be%e4%b8%ad%e5%b9%b8%e5%ad%98%e7%9a%84%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    态势感知
   </a>
   意识。你需要准确判断哪些信号不正常。
  </p>
  <p class="graf graf--p">
   其实并没有那么难。·
  </p>
  <p class="graf graf--p">
   比如您是一位女士，在一家大型商场里选手袋。您忽然发现不远处有一位男士神情奇怪，看起来不像是对手袋感兴趣，他的眼睛一直在瞄向周围的人，包括偷偷瞄向您。
  </p>
  <p class="graf graf--p">
   您会意识到这种奇怪的 “关注”，我们每个人都有这样的直觉，很多时候只是您还没有激发出它、或者您的注意力被恶意操纵了。所以，本
   <strong class="markup--strong markup--p-strong">
    系列内容的阅读顺序很重要，第一篇文章中的内容是后面所有内容的基础
   </strong>
   ，即：
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%b4%bb%e4%b8%8b%e5%8e%bb%e9%9c%80%e8%a6%81%e5%be%88%e5%a4%9a%e7%9f%a5%e8%af%86%e5%92%8c%e6%8a%80%e8%83%bd%ef%bc%9a%e5%9c%a8%e7%81%be%e9%9a%be%e4%b8%ad%e5%b9%b8%e5%ad%98%e7%9a%84%e6%96%b9%e6%b3%95/" href="https://www.iyouport.org/%e6%b4%bb%e4%b8%8b%e5%8e%bb%e9%9c%80%e8%a6%81%e5%be%88%e5%a4%9a%e7%9f%a5%e8%af%86%e5%92%8c%e6%8a%80%e8%83%bd%ef%bc%9a%e5%9c%a8%e7%81%be%e9%9a%be%e4%b8%ad%e5%b9%b8%e5%ad%98%e7%9a%84%e6%96%b9%e6%b3%95/" rel="noopener noreferrer" target="_blank">
    态势感知
   </a>
   。
  </p>
  <p class="graf graf--p">
   在手袋区闲逛的男士本来就不会很多，这是基于常识的信号；但这一信号究竟是否能说明出现了问题？你只需要使用一些技巧来测试它。
  </p>
  <p class="graf graf--p">
   首先假设这一初步判断是准确的。您可以通过快速转到其他购物区来进行观察。如果对方真的瞄准您了，他肯定会跟随您到下一个购物区，甚至再下一个区域。
  </p>
  <p class="graf graf--p">
   这就是
   <strong class="markup--strong markup--p-strong">
    情报界使用的概测法
   </strong>
   ：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    一次=意外事件
   </li>
   <li class="graf graf--li">
    两次=巧合
   </li>
   <li class="graf graf--li">
    三次=敌对行为
   </li>
  </ul>
  <p class="graf graf--p">
   当然，如上面的例子，那个男人有可能是在帮助女朋友或母亲购买礼物；为了排除这点，您就需要采取探测策略，当第三次转换区域后依旧发现他的存在，那就不再是巧合了。
  </p>
  <p class="graf graf--p">
   如我们
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%E5%9C%A8%E4%BD%A0%E7%9A%84%E6%97%A5%E5%B8%B8%E7%94%9F%E6%B4%BB%E4%B8%AD%E5%BA%94%E7%94%A8%E9%9A%90%E8%94%BD%E9%87%8E%E6%88%98%E6%8A%80%E6%9C%AF/" href="https://www.iyouport.org/%E5%9C%A8%E4%BD%A0%E7%9A%84%E6%97%A5%E5%B8%B8%E7%94%9F%E6%B4%BB%E4%B8%AD%E5%BA%94%E7%94%A8%E9%9A%90%E8%94%BD%E9%87%8E%E6%88%98%E6%8A%80%E6%9C%AF/" rel="noopener noreferrer" target="_blank">
    曾经介绍过的那样
   </a>
   ，试图行凶的家伙会从
   <strong class="markup--strong markup--p-strong">
    跟随您的步速
   </strong>
   开始，这是很容易判断的指标。
  </p>
  <p class="graf graf--p">
   一般情况下毫无用心的陌生人会下意识地避免与您同样的步速，您可以稍微留意下这点，如果两个陌生人出现了相同的行进速度，其中一个人必然会变快或者变慢，以错开同步。但是跟踪者却不会，他们将始终追求与您同步 —— 不论是汽车还是步行。
  </p>
  <p class="graf graf--p">
   于是这就是一个很容易判断的指标。
  </p>
  <p class="graf graf--p">
   为了确保不被监视，您需要格外注意、保持警惕。
   <strong class="markup--strong markup--p-strong">
    专业间谍会注意以下情况：
   </strong>
  </p>
  <p class="graf graf--p">
   1、在街上有人靠你太近：正如前面提到的，意图行凶者会跟随您的步速，如果你小步走，他也会小步走以配合你的步伐。您需要注意这点。
  </p>
  <p class="graf graf--p">
   2、有人与你对视：一般正常人不会在街上死盯着另一个人看，当出现目光相对时，正常人会立刻错开目光以避免不舒服；但如果有人这样盯着你，请小心，这可能说明有事要发生。
  </p>
  <p class="graf graf--p">
   3、有人似乎对你太过关注：如果有人多次在你的日常路线的同一地点、同一时间出现（比如下班途中、放学途中、公交车站、健身房等等），监视这个人，注意他的行动。
  </p>
  <p class="graf graf--p">
   不要低估自己的不适感，诸如不安、焦虑、疑惑、怀疑、犹豫和恐惧。
  </p>
  <p class="graf graf--p">
   如果以上情况有一项或多项出现，你就有可能已经被监视了。
  </p>

  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    策略1：停下来转身
   </strong>
  </h3>
  <p class="graf graf--p">
   当有人跟踪您的时候，停下来转身的方法很管用，它既可以帮助您判断是否被监视，也能让对方知道您已经对他的行为有所警觉。
  </p>
  <p class="graf graf--p">
   如果当时您在行走，那只需要停下脚步转过身，假装有事要做 —— 比如查看电话、系携带、或者假装转身找人。
  </p>
  <p class="graf graf--p">
   接着，直视你认为正在跟踪你的那个人。
  </p>
  <p class="graf graf--p">
   一般外行的跟踪者在此时就会方寸大乱。他可能会被吓呆，或者至少表现不自然，因为你在攻其不备。这就很容易识别到了。
  </p>
  <p class="graf graf--p">
   换句话说，跟踪您的人不可能表现得像街上其他路人那般自然。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    策略2：宣告知情
   </strong>
  </h3>
  <p class="graf graf--p">
   正如前面所提到的，一旦判定自己被跟踪，很重要的一点就是，让对方知晓您已经对其行为有所警觉。
  </p>
  <p class="graf graf--p">
   一旦潜在的危险被察觉，多数坏人都会放弃你。记住，坏人是在寻找理想的施暴对象，宣告您已经意识到他们的存在、知晓他们想要加害于你，此举将让您变得不再是那么适合下手的对象。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    策略3：不要当软弱的目标
   </strong>
  </h3>
  <p class="graf graf--p">
   换句话说，您不应该成为 “理想的施暴对象”。您需要展现出强大的一面，挺直腰背走路，把头抬起来。给一个人打电话，谈论有人在跟踪你，声音要大，动作要明显。
  </p>
  <p class="graf graf--p">
   不要担心，把战术防身笔掏出来，准备好使用它。
  </p>
  <p class="graf graf--p">
   坏人在视野内挑选施暴对象时他们选择的都是那些看起来懦弱的、或者看起来掉以轻心的人，比如低着头、塌着肩膀。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    策略4：待在公共场合
   </strong>
  </h3>
  <p class="graf graf--p">
   如果确定被跟踪了，一定不能回家。
  </p>
  <p class="graf graf--p">
   表面上看起来家似乎是安全的地方，可以锁起门来打电话求助，但是，这种情况下一定不要这样做。因为坏人会强行闯入，或者假装离开，其实是隐藏起来，计划着晚点时候闯入。
  </p>
  <p class="graf graf--p">
   如果发现自己被监视，为了确保安全，需要注意以下事项：
  </p>
  <ol class="postList">
   <li class="graf graf--li">
    待在人多的地方，公共场所，打电话求助。比如餐厅、咖啡馆、拥挤的街角等等地方；
   </li>
   <li class="graf graf--li">
    永远不要走入黑暗的小巷或无人的街道，在那里您会与世隔绝，坏人会更容易对您下手；
   </li>
   <li class="graf graf--li">
    如果在公共场合被跟踪，例如杂货店。不要走出店铺返回车上。几乎所有坏人都会等在出口处，等你走出来；接下来他们就会跟踪你的车，或者尾随你回家。
   </li>
  </ol>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    观察对方的鞋子
   </strong>
  </h3>
  <p class="graf graf--p">
   如果您在步行途中遭遇跟踪，请想办法注意对方的鞋子，以确定跟踪您的是不是同一个人。
  </p>
  <p class="graf graf--p">
   因为如果您注意头发、帽子、上衣等部位，这些部位更换是很容易的，我们在
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/xbYXdR/iyp-1" href="https://start.me/p/xbYXdR/iyp-1" rel="noopener noreferrer" target="_blank">
    反监视技巧
   </a>
   中也提出了为什么您应该携带一个小背包，其中装有假发、帽子、眼睛、外套等东西，以便迅速更换并挤入人流中，可以混淆监视者的视线。
  </p>
  <p class="graf graf--p">
   但是鞋子不容易换。所以在判断跟踪的时候您需要反过来使用上述方法：瞄准对方的鞋子，理论上更容易判断一些。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    策略5：减速、混淆视线
   </strong>
  </h3>
  <p class="graf graf--p">
   如果确定有车辆跟踪您，继续驾驶 —— 再次强调，不要回家 —— 改变车速、多转弯，让跟踪者意识到你已经产生了警觉。
  </p>
  <p class="graf graf--p">
   情况安全的时候，请注意跟踪车辆的车牌和汽车型号，记住它们；要能做到在下一次识别它们，并且求助时会非常有用。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     当您遇到高级威胁时
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   高级威胁指的是，威胁来自那些
   <strong class="markup--strong markup--p-strong">
    对您了解太多的人
   </strong>
   。
  </p>
  <p class="graf graf--p">
   也许您不知道恶意者在此前如何对您进行了侦查，因为您不认识他们。除了谨慎使用互联网之外 —— 避免发布任何关于您自己的真实信息 —— 您还可以
   <strong class="markup--strong markup--p-strong">
    通过分身方法来阻断监视侦查的路径
   </strong>
   ，在下面回顾方法：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    这里是完整版《
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e6%82%a8%e5%9c%a8%e4%bb%80%e4%b9%88%e6%97%b6%e5%80%99%e9%9c%80%e8%a6%81%e5%81%87%e5%90%8d%e3%80%81%e5%8c%bf%e5%90%8d%e5%92%8c%e5%85%ac%e5%bc%80%e5%9c%a8%e7%ba%bf%e8%ba%ab%e4%bb%bd%ef%bc%9f/" href="https://www.iyouport.org/%e6%82%a8%e5%9c%a8%e4%bb%80%e4%b9%88%e6%97%b6%e5%80%99%e9%9c%80%e8%a6%81%e5%81%87%e5%90%8d%e3%80%81%e5%8c%bf%e5%90%8d%e5%92%8c%e5%85%ac%e5%bc%80%e5%9c%a8%e7%ba%bf%e8%ba%ab%e4%bb%bd%ef%bc%9f/" rel="noopener noreferrer" target="_blank">
     您在什么时候需要假名、匿名和公开在线身份？– 在线创建和管理身份的思考方式
    </a>
    》其中包含的所有链接都很重要！
   </li>
  </ul>
  <p class="graf graf--p">
   并且您尤其应该注意那些可见的高级威胁，比如愤怒的 Ex、充满怨恨的前同事或雇员、欠下巨额债务的熟人 …… 或者，每次锻炼完健身房外都有个人在注意你吗？某个人总是和你同时去那家店买东西吗？等等。
  </p>
  <p class="graf graf--p">
   如果按照高级威胁来应对，有一些
   <strong class="markup--strong markup--p-strong">
    重要的附加防范措施
   </strong>
   需要注意。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    改变行动习惯
   </strong>
  </h3>
  <p class="graf graf--p">
   如果担心某个特定的人会跟踪您，那就需要注意您一直采取的习惯性行为模式。
  </p>
  <p class="graf graf--p">
   这一点可能有难度，因为人类都是习惯的动物，很难打破常规。但是为了安全，打乱行为模式以及活动顺序是至关重要的。
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    改变早上出门的时间
   </li>
   <li class="graf graf--li">
    改变通勤路线和时间
   </li>
   <li class="graf graf--li">
    打乱行为。如果每天在同一家店吃早餐，那就换一家，或者选择不同的时间
   </li>
   <li class="graf graf--li">
    去之前从未去过的地方购物和用餐
   </li>
   <li class="graf graf--li">
    下班后挑选不同的回家路线
   </li>
  </ul>
  <p class="graf graf--p">
   需要注意的是，这些策略的确可以让您难以被跟踪，但是不要忽视提高警惕；
   <strong class="markup--strong markup--p-strong">
    不要离开黄色态势感知
   </strong>
   —— 黄色能让您远离不得不正面对抗的红色态势，即 危急情况。
  </p>
  <p>
   <img alt="" class="aligncenter size-full wp-image-14244 jetpack-lazy-image" data-lazy-sizes="(max-width: 1050px) 100vw, 1050px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?resize=1050%2C700&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?w=1050&amp;ssl=1 1050w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?resize=300%2C200&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?resize=1024%2C683&amp;ssl=1 1024w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?resize=768%2C512&amp;ssl=1 768w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?resize=272%2C182&amp;ssl=1 272w" data-recalc-dims="1" height="700" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/06/2.jpeg?resize=1050%2C700&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="1050"/>

   </noscript>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     极端监视
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   普通人无需担心自己遇到极端监视的情况，因为这种情况指的是您面对那些
   <strong class="markup--strong markup--p-strong">
    非常专业的政府间谍或私人侦探
   </strong>
   。除非您是重要的政治敏感人士、让政府焦头烂额的人，否则理论上您遇不到这种情况。
  </p>
  <p class="graf graf--p">
   但是我们的读者中可能包括敏感人士，而且很可能
   <strong class="markup--strong markup--p-strong">
    同时还是
   </strong>
   没有接受过专业训练的普通人，应付起来可能并不容易。但如果您平日有习惯性的观察能力练习，应对复杂的监视跟踪方法也能是有一些门道的。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    平行监视 ——
   </strong>
  </h3>
  <p class="graf graf--p">
   需要的工具：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    两辆或更多的汽车
   </li>
   <li class="graf graf--li">
    道路信息 — 必须有平行道路
   </li>
  </ul>
  <p class="graf graf--p">
   如何行动：第一辆车跟踪目标，保持一定的距离；另一辆车从与目标行驶车道平行的道路上跟踪目标。一旦目标车辆转弯，另外那辆车就能接替监视。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    跳蛙式监视 ——
   </strong>
  </h3>
  <p class="graf graf--p">
   需要的工具：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    多辆汽车
   </li>
   <li class="graf graf--li">
    暗语
   </li>
  </ul>
  <p class="graf graf--p">
   如何行动：这种方法需要多辆汽车和随时保持联系的对讲机。您和您的搭档需要创造出一套用于描述目前状况的暗语，避免对讲机信号被监听。
  </p>
  <p class="graf graf--p">
   正如其名所示那样，这套监视方案需要通知前面的搭档，目标正在靠近。这就意味着第一辆车无须减速或撤退，无须向目标隐藏身份。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    迷惑监视者
   </strong>
  </h3>
  <p class="graf graf--p">
   摆脱监视者最好的方法就是给他们捣乱。
  </p>
  <p class="graf graf--p">
   给完全陌生的人随便传句话，或者走进商店，开始漫无目的地闲聊。这些简单的行为就足以迷惑监视者，甚至甩掉他们。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    团队监视 ——
   </strong>
  </h3>
  <p class="graf graf--p">
   需要的工具：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    一组人
   </li>
   <li class="graf graf--li">
    对讲机
   </li>
   <li class="graf graf--li">
    周边区域信息
   </li>
  </ul>
  <p class="graf graf--p">
   如何行动：实施团队监视需要将一组人马安插在目标可能出现的战略位置。这些人需要通过对讲机进行联系。
  </p>
  <p class="graf graf--p">
   如果行动位置是一个大城市，你可以将组员安排在地铁入口、街角、还可以是目标停留的酒店或咖啡馆。监视团队的每个队员都通过对讲机提前通知彼此目标的位置。
  </p>
  <p class="graf graf--p">
   👌 非常建议您练习一下想象力，想象自己如果在这种环境下被跟踪，你可以如何甩掉目标。如果您身处险境，知道该如何保护自己也能让人充满力量。
  </p>
  <p class="graf graf--p">
   从现在开始，对身边人保持警觉；注意那些在商店里和你一起排队的人；观察餐馆里盯着你看的人；对那些与您*保持*着同一路线的车辆不要掉以轻心。
  </p>
  <p class="graf graf--p">
   意识到这些人的存在，正是保证自己的安全需要走出的第一步，保持自信和警觉，这是能救命的。
  </p>
  <p class="graf graf--p">
   今天就是这样，希望您已经学到知识。在下一集中我们将汇总一些斗心智的主要方法 —— 避免圈套、以及给被人下套的方法。下次见！⚪️
  </p>

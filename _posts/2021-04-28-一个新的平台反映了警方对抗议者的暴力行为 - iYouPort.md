  <p class="graf graf--p">
   去年的BLM抗议活动从美国开始蔓延到全球各地，人们聚集在一起大声疾呼反对
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%9e%84%e6%96%ad%e4%b8%8e%e6%b0%91%e4%b8%bb%ef%bc%9a%e5%a6%82%e4%bd%95%e7%bb%93%e6%9d%9f%e9%95%80%e9%87%91%e6%97%b6%e4%bb%a3/" href="https://www.iyouport.org/%e5%9e%84%e6%96%ad%e4%b8%8e%e6%b0%91%e4%b8%bb%ef%bc%9a%e5%a6%82%e4%bd%95%e7%bb%93%e6%9d%9f%e9%95%80%e9%87%91%e6%97%b6%e4%bb%a3/" rel="noopener" target="_blank">
    系统性的种族压迫
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/do-not-resist%ef%bc%9a%e8%ad%a6%e5%af%9f%e5%86%9b%e4%ba%8b%e5%8c%96%e7%9a%84%e5%8d%b1%e6%9c%ba/" href="https://www.iyouport.org/do-not-resist%ef%bc%9a%e8%ad%a6%e5%af%9f%e5%86%9b%e4%ba%8b%e5%8c%96%e7%9a%84%e5%8d%b1%e6%9c%ba/" rel="noopener" target="_blank">
    警察的军事化
   </a>
   。虽然绝大多数抗议者都是在和平地表达自己的意见，进行
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e5%86%b7%e6%88%98%e5%88%b0%e5%8f%8d%e6%81%90%e6%88%98%e4%ba%89%ef%bc%9a%e4%b8%93%e5%88%b6-%e8%a7%a3%e5%86%b3%e6%96%b9%e6%a1%88-%e5%b8%a6%e6%9d%a5%e7%9a%84%e5%8d%b1%e5%ae%b3%e5%be%80/" href="https://www.iyouport.org/%e4%bb%8e%e5%86%b7%e6%88%98%e5%88%b0%e5%8f%8d%e6%81%90%e6%88%98%e4%ba%89%ef%bc%9a%e4%b8%93%e5%88%b6-%e8%a7%a3%e5%86%b3%e6%96%b9%e6%a1%88-%e5%b8%a6%e6%9d%a5%e7%9a%84%e5%8d%b1%e5%ae%b3%e5%be%80/" rel="noopener" target="_blank">
    非暴力抵抗
   </a>
   ，但当权者却以不成比例的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e9%95%87%e5%8e%8b%e9%83%a8%e9%98%9f%e6%9c%80%e5%b8%b8%e7%94%a8%e5%93%aa%e4%ba%9b%e6%ad%a6%e5%99%a8%e5%af%b9%e4%bb%98%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f-%e6%a6%82%e8%a7%88/" href="https://www.iyouport.org/%e9%95%87%e5%8e%8b%e9%83%a8%e9%98%9f%e6%9c%80%e5%b8%b8%e7%94%a8%e5%93%aa%e4%ba%9b%e6%ad%a6%e5%99%a8%e5%af%b9%e4%bb%98%e6%8a%97%e8%ae%ae%e8%80%85%ef%bc%9f-%e6%a6%82%e8%a7%88/" rel="noopener" target="_blank">
    侵略性镇压行为
   </a>
   来回应。警方还
   <a class="markup--anchor markup--p-anchor" data-href="https://www.bellingcat.com/news/americas/2020/06/05/visualizing-police-violence-against-journalists-at-protests-across-the-us/" href="https://www.bellingcat.com/news/americas/2020/06/05/visualizing-police-violence-against-journalists-at-protests-across-the-us/" rel="noopener" target="_blank">
    对报道抗议活动的记者使用了暴力
   </a>
   。
  </p>
  <p class="graf graf--p">
   Bellingcat 与 Forensic Architecture 一起，对2020年5月以来全美1000多起警察对抗议者的暴力事件进行了识别、核实和存档。这些数据
   <a class="markup--anchor markup--p-anchor" data-href="https://docs.google.com/spreadsheets/d/1-9KKfaEDhgeJ8dz8HjsFNBniDCQj-5ADWbuMkw2hNBU/edit?usp=sharing" href="https://docs.google.com/spreadsheets/d/1-9KKfaEDhgeJ8dz8HjsFNBniDCQj-5ADWbuMkw2hNBU/edit?usp=sharing" rel="noopener" target="_blank">
    可以在这里访问
   </a>
   ，并且可供使用和可视化。
  </p>
  <p class="graf graf--p">
   这些数据已经使用
   <a class="markup--anchor markup--p-anchor" data-href="https://github.com/forensic-architecture/timemap" href="https://github.com/forensic-architecture/timemap" rel="noopener" target="_blank">
    TimeMap
   </a>
   进行了可视化，TimeMap 是由 Forensic Architecture 开发的一个平台，它可以在空间和时间上映射事件。调查者已将这些数据分为不同类型的暴力和不当行为：身体暴力 (攻击、逮捕/拘留、橡皮子弹)、化学武器暴力 (催泪瓦斯、胡椒喷雾)、破坏/没收财产、对极右翼的宽容、攻击法律观察员、攻击医务人员。
  </p>

  <p class="graf graf--p">
   调查者公布了地图和数据集，以便让任何人都可以查询数据并找到模式。分析表明，
   <strong class="markup--strong markup--p-strong">
    暴力镇压行为是非常普遍的、并且是持续的。全国各地的警察，来自不同的城镇和城市，都对抗议者使用暴力和故意的侵犯
   </strong>
   。这绝非某个糟糕的警察部门的案例，相反，暴力镇压就是是全国范围内的协议的一部分。
  </p>
  <p class="graf graf--p">
   这也是在某些情况下，许多城市和部门相对罕见的暴力行为。有一些城市在此之前并没有围绕化学武器使用的协议，因为在这些抗议活动之前，化学武器只在特警队的特定情况下使用。然而，在这场抗议活动中，这些地区的警察部队仍然对抗议者使用了催泪瓦斯。尽管公众的强烈抵抗，但从第一波抗议活动的镇压反应开始，这种暴力已经被正常化了。
  </p>
  <p class="graf graf--p">
   这种常态一直在维持，并继续在全国各地蔓延。
  </p>
  <p class="graf graf--p">
   虽然许多组织正在努力为暴力受害者伸张正义和追究责任，但由于美国独特的 “有条件豁免” 独特概念，反抗者的工作十分复杂。这就是为什么调查人员认为这些行为被存档和可视化是至关重要的，以创建一个暴力镇压抗议活动的历史记录，并揭示参与其中的警方和官员。
  </p>
  <p class="graf graf--p">
   【注：“有条件豁免” 用以保护所有被控滥权的警员免于承担民事责任。】
  </p>

  <p class="graf graf--p">
   那么，如果买不起/买不到一次性电话怎么办？
   <a class="markup--anchor markup--p-anchor" data-href="https://theintercept.com/2020/06/15/protest-tech-safety-burner-phone/?utm_medium=email&amp;utm_source=The%20Intercept%20Newsletter" href="https://theintercept.com/2020/06/15/protest-tech-safety-burner-phone/?utm_medium=email&amp;utm_source=The%20Intercept%20Newsletter" rel="noopener noreferrer" target="_blank">
    Intercept 给了一些建议
   </a>
   。
  </p>
  <p class="graf graf--p">
   诺基亚1.3智能手机需要99美元，购买三个月的预付费服务还需要40美元。如果这对您来说太贵了，您可能还有其他选择：
  </p>
  <p class="graf graf--p">
   如果您有老旧的手机还没扔掉，只要它仍然可以工作并且电池仍可以充电，就可以将其用作一次性备用电话，而不用购买新手机。
  </p>
  <p class="graf graf--p">
   您只需要一张新的无关您身份的SIM卡。
   <strong class="markup--strong markup--p-strong">
    在准备抗议之前，请确保将手机恢复出厂设置。
   </strong>
  </p>
  <p class="graf graf--p">
   如果您也没有老旧的手机，
   <strong class="markup--strong markup--p-strong">
    并且您决定带上自己的主要手机，请采取以下步骤以使其更安全 —— 私人数据最终落入警方手中的可能性较小
   </strong>
   ：
  </p>
  <p class="graf graf--p">
   1、
   <strong class="markup--strong markup--p-strong">
    禁用
   </strong>
   指纹和面部解锁功能，改为输入密码或口令来解锁手机。这样一来，解锁手机需要的时间就更长了，也大大增加了警察未经你同意就解锁手机的难度。
  </p>
  <p class="graf graf--p">
   2、确保您的密码不容易被猜到。如果您使用的是数字密码，那么至少应该有6位数字，但
   <strong class="markup--strong markup--p-strong">
    越长越好
   </strong>
   。
  </p>
  <p class="graf graf--p">
   3、
   <strong class="markup--strong markup--p-strong">
    设置SIM卡密码
   </strong>
   ，这样可以防止警察从你的手机上取下SIM卡插入另一个手机，这样他们就可以接管你的电话号码。
  </p>
  <p class="graf graf--p">
   这里是
   <a class="markup--anchor markup--p-anchor" data-href="https://support.apple.com/en-us/HT201529" href="https://support.apple.com/en-us/HT201529" rel="noopener noreferrer" target="_blank">
    iPhone
   </a>
   和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.howtogeek.com/259360/how-to-set-up-sim-card-lock-for-a-more-secure-android-phone/" href="https://www.howtogeek.com/259360/how-to-set-up-sim-card-lock-for-a-more-secure-android-phone/" rel="noopener noreferrer" target="_blank">
    Android
   </a>
   的相关说明。如果您设置了SIM PIN，除了密码之外，每次重启手机时还需要输入SIM PIN。
  </p>
  <p class="graf graf--p">
   如果您使用的是 Android，请确保手机的存储空间已加密（所有 iPhone 均默认加密）。在大多数 Android 手机上，您可以在“设置”应用中的“安全”&gt;“加密手机” 下查找手机加密设置。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    禁用所有不必要的智能手机功能，例如Wi-Fi、蓝牙和定位服务
   </strong>
   。您也可以在不需要使用网络时将手机
   <strong class="markup--strong markup--p-strong">
    保持在飞行模式
   </strong>
   。这将使您的手机泄漏的信息更少，那是那些警察可以用来跟踪您的信息。
  </p>

  </p>
  <p class="graf graf--p">
   您可能会因为担心被警察抢走手机而丢失了所有拍摄的抗议现场照片和视频，倾向于选择将这些文件上传到谷歌相册，以便万一警察强行删除了手机中的记录，记录也不会丢失。但是，不要忘了，
   <strong class="markup--strong markup--p-strong">
    谷歌相册是不加密的，如果警察和政府当局提出要求，谷歌就会出卖你。
   </strong>
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    如果你被捕
   </strong>
  </p>
  <p class="graf graf--p">
   即使你是完全和平的，没有违反任何法律，警察也可能在抗议活动中逮捕你。
   <strong class="markup--strong markup--p-strong">
    如果您察觉到自己即将被逮捕，那么您应该迅速这样做：关闭你的手机电源。这样警察就很难进入你的手机了
   </strong>
   。
  </p>
  <p class="graf graf--p">
   除非有搜查令，否则警察无权在未经你同意的情况下搜查你的手机。如果警察要求你解锁手机，或者抢走你的手机要求你解锁，请告诉他们，“我不同意被搜查手机”、“我拒绝被搜查手机。我需要和律师谈谈” ——
   <strong class="markup--strong markup--p-strong">
    就算您还没有律师，也没关系，但是你必须明确说出这句话
   </strong>
   。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    警察可能会没收你的手机，无论如何都会非法搜查你的手机，但是，此刻您最好明确表示您不同意，这样才能在今后有可能的起诉中避免警察反咬你
   </strong>
   。
  </p>
  <p class="graf graf--p">
   警察也有可能试图偷走或砸烂你的手机，特别是如果他们注意到你拍摄了他们的暴力行为。如果发生任何这类情况，至少您能保证那只是您的廉价一次性手机，上面有最少的数据，而不是一个昂贵的手机，里面充满了关于你的私密生活和政治行动的敏感信息。⚪️
  </p>

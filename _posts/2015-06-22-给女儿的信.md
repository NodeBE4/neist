         <h1 class="page-title">
          给女儿的信
         </h1>
        </div>
       </div>

       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          亲爱的风风：
          <br/>
          <br/>
          欢迎你来到这世界。踏入第十个月，从睡姿变换成坐姿对你已经没难度。你学会爬行，尝试扶着物件站起来，又开始模彷爸爸妈妈牙牙学语。用姆指和食指拿起一粒米、黄豆你都应付自如，力气更大。风风，别忘了能做到这一切，是因为你用上十个月的时间，每天不断反覆努力学习。世界和环境四周对你来说开始有意义，好奇心更使你愈来愈不想花时间睡觉。你的动作表情古灵精怪，妈妈便经常为你拍照，透过网络传送给亲戚和在海外的朋友观看，经常换来大堆“赞好”。
          <br/>
          <br/>
          虽然你还未学会文字，但已经认得妈妈手上的智能电话。和你同代的小孩，有不少甚至抢着父母的电话看动画，玩游戏。如果跟你说，妈妈在中学五年级才有第一部电脑，再过一两年才有自己的手提电话，你可能不会相信，以为妈妈在胡说。
          <br/>
          <br/>
          风风，妈妈和你很幸运。我们在香港出世，这是一个出版、言论和网络尚算自由的地方。在这里，妈妈可以把你的照片上载于社交网络，浏览外国的新闻趣事，自由发表对政治和社会时局的看法，还可以以文字支持一些遭遇不公义对待的人。你知道吗？世界上有其他小朋友，他们生活在极权政府的高压统治，他们不能畅所欲言，上网也只能浏览政府准许的网站。有些时候，他们的父母为着正义敢言，拒绝向不公义低头，或者声援其他受欺压者，结果遭拘捕、软禁、流放海外，被逼和挚爱的子女骨肉分离。身为母亲，岂会不知道这是何等难过的事。但这些父母坚持去冒大不讳，就是希望社会变革，下一代不用再受这些无理的对待。
          <br/>
          <br/>
          要是生长在中国古代，妈妈应该不能写这一封信给你，因为我们是女性，连入学读书的机会也不会有。风风，你属于网络世代。妈妈相信，不用多久你会学懂上网，整个社会世代都迫不及待要你和网络接轨。今天享有的言论和网络自由，其实是前人冒险争取回来，我们正享受着他们艰苦得来的成果。直至今天，世界上仍有许多地方未能享受这种自由，许多哥哥姐姐、叔叔姨姨为此努力。正如不久前美国有位斯诺登哥哥，他牺牲自己的工作前途，甚至被通辑，不能回家见父母，就是为了揭穿美国政府庞大而不合理的监控民众和别国政府的计划。有些网站、工具软件和项目计划，就是为了帮助没有网络自由的人民突破封锁。所以，妈妈希望你能珍惜这份自由，并且在有需要时捍卫它。
          <br/>
          <br/>
          有些时候，政府总爱说网络充斥各种不良资讯，影响小朋友，于是政府出师有名，要发起各种“淨网行动”，审查和过滤有害网站。
          <span>
           网络自由，资讯百花齐放，自然会莨莠不齐，树大自然有枯枝。有些
          </span>
          <span>
           有(內)
          </span>
          <span>
           容，可能妈妈都不想你看
          </span>
          到，怕你做恶梦，或者误入歧途，这真是很大的挑战。不过，除非我们已经在基督教所说的天国，或者一套老电影《真人Show》，主角的生活都只是按照剧本安排。不然，有坏人，有好人，毕竟才是真实的世界。妈妈相信，从小教你正确的人生价值观，学懂分辨是非好歹，让你以自由意志选择拒绝某些资讯，才是可取之道。
          <br/>
          <br/>
          最后，妈妈还有一件事要告诉你。当你在网络自由而行，希望你谨记，要为自己的一言一行负责任，别忘了怜悯和公义的心。网络欺凌已成为你和同代人不能迴避的问题，所谓的网络公审会为受害人带来莫大创伤，甚至了结自己的性命。当你发表和转发任何有特定对象的讯息时，先问问自己，这和公众利益有没有关系？还只是无聊以别人取乐子？三思而后行，似乎和以光纤飞行的网络世界特质相矛盾，却可能挽救不必要的悲剧。此外，网络上还有些人，因着政治或经济利益发佈有违良心的讯息，颠倒是非，指鹿为马。妈妈希望你千万别成为这些人，但凡留言，都要对得起自己的良知。
          <br/>
          亲爱的女儿，妈妈就此搁笔。祝你身体健康，快高长大。
          <br/>
          <br/>
          爱你的妈妈上
         </p>

         <h1 class="page-title">
          国家黑客“整治"港人
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/11" class="username" datatype="" href="/author/11" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        阿蔼
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <strong>
           （泡泡特约撰稿） 人大常委会为香港行政长官普选提名门槛落闸，要求有意竞逐者要得到过半数提名委员会支持才能成为候选人，远远超出目前选举委员会八份之一的提名要求，很明显是为了全面控制行政长官人选。
          </strong>
          <br/>
          <br/>
          为了制作违反民意的说词，中央官员多番以"国家安全"作为借口。
          <br/>
          <br/>
          面对强权，以和平占中作为揭幕的公民不合作运动势在必行。为了抹黑这场运动，国家出动了黑客和特务来展开资讯战。这场"战争"涉及盗取资讯、散播监控软件、手机应用程式及虚假讯息，简值无所不用其极。
         </p>
         <h2>
          导演"外国势力"介入的剧情
         </h2>
         <p>
          黑客资讯战的序幕是
          <a href="https://pao-pao.net/article/117" rel="nofollow">
           《苹果日报》电脑系统被入侵
          </a>
          ，七一前后，国家级的黑客一边以DDoS攻击壹集团电脑主机，一边大规模下载办公室内所有数码资讯，最后在数码影印机的存档里找到集团创办人黎智英向泛民政党以及和平占中运动的捐款记录，并以网上泄密的形式向各大媒体发布。
          <br/>
          <br/>
          其实，黎智英向泛民政党捐钱，一直是政圈及媒界公开的秘密，且无法例禁止。然而，亲北京的媒体，以黎智英私人助理Mark Simon曾在美国海军服役作为材料，暗指黎智英受托于美国特报机关，要以"政治黑金"把香港变成"颠覆基地"。
          <br/>
          <br/>
          正如黎智英的自辩，黑客既然成功入侵，并大规模盗取公司的数据，迄今却无半点证据指向黎与美国情报机关有连系。至于Mark Simon虽于年青时服役美军，并没有证据证明他现在仍然为美国政府办事。在一个强调个人独立自主的社会，人的选择不会一成不变，再加上美国是募兵制，参军对于很多人来说是一份工作，泄露美国领事馆密电的Chelsea Manning，也是美国空军的情报分析员。
          <br/>
          <br/>
          然而，在毫无证据下，黑客泄密被香港各大主流媒体炒成“外国势力”，制造“国家安全”的“危机”，成为人大常委违反全港民意的选举方案的说词。
         </p>
         <h2>
          占中会议记录外泄导演三子操控学生
         </h2>
         <p>
          同样的手法，用于抹黑占中三子——陈健民、戴耀廷和朱耀明。
          <br/>
          <br/>
          自从香港专上学生联会及中学生组织学民思潮 ​​宣布罢课后，又有匿名人以“民主真兄弟”的网名用泄密的形式上传民间占中运动协调会议的会议记录，以导演三子操控学生罢课的情节。组织者怀疑用作会议文书的电脑被黑客入侵并盗取相关档案。
          <br/>
          <br/>
          其实这类民间协调会议很常见，目的是让不同团体分享部署，互相配合，使运动能顺利进行。而会议记录一般不会逐字逐句记下，只写重点，这让匿名发布者可以“断章取义”的方法，制造三子操控学生的情节。
          <br/>
          <br/>
          然而，有一点政治和社运常识的人都知道，学生组织一直较占中三子更为激进，上两个月的七一占中预演，也是由学生主动牵头。此外，占中三子迟迟未行动，令学生燥动，引发罢课的决定。与其说三子操控学生，不如说学生希望透过罢课击起战鼓，推三子尽早行动。
          <br/>
          <br/>
          当然，大部份人对政治漠不关心，很容易会循着“小孩子容易被教唆误入歧途”的“常理”去理解三子与学生的关系，这也是匿名发布者及亲北京媒体想要编写的情节。
         </p>
         <h2>
          虚假会议记录泡制港版六四
         </h2>
         <p>
          更无耻的资讯战，是制作虚假的会议记录，去导演“外国势力”介入、炮制港版六四的情节。
          <br/>
          <br/>
          继占中协调会议的记录流出后，公民党英语组一场“年轻一代如何看待香港重大政治运动发展”的讨论内容，也以泄密的形式被“民主真兄弟”上传到互联网。然而，这所谓的“会议记录”内容纯属虚构。
          <br/>
          <br/>
          虚构会议记录里引述一位大学外籍教授Dan Garrett说，“美国会保护学生领袖，包括赴外留学、定居”。又引述积极统筹“罢课不罢学”的大学教授陈允中说，“香港要争取民主，就必须暴力、‘企硬’，必须违法甚至流血，制造港版‘六四’。”更炮制一位伊斯兰教中东难民说：“如条件成熟会在港成立极端伊斯兰组织。”
          <br/>
          <br/>
          虽然虚构的内容极其夸张，但仍有一些人以“半信半疑”的态度质询与会者是否真有其事。可见，在政治对立的状态下，一般人抵御谣言的智力会下滑。
         </p>
         <h2>
          冒充组织者散播监控程式
         </h2>
         <p>
          上面提到的“虚构泄密”是有针对性的抹黑，再进一步的资讯战，涉及大规模地散播监控程式。
          <br/>
          <br/>
          一直以来，均有冒充某些政党或社运团体名义发出“钓鱼”电邮的情况，实为散播木马程式以破坏或监控开启电邮的电脑。
          <br/>
          <br/>
          因为手机上网越加普及，国家黑客改变策略，从散播钓鱼电邮改为散播钓鱼应用程式。
          <a href="https://pao-pao.net/article/173" rel="nofollow">
           最新的案例是冒充一个由程式编写员组成的组织Code4HK，以协调占中之名，透过Whatsapp散播监控程式。
          </a>
          用户一旦下载，便会被监听电话通讯并被百度地图定位。
          <br/>
          <br/>
          虽然Code4HK很快就澄清事件，但事件令人害怕日后建立无线网状网路(Wireless Mesh Network)作游行示威通讯时，会出现更多的安全隐忧。
          <br/>
          <br/>
          以上种种可见，“国家级”的反占中资讯战，无孔不入，然而这场战争的敌人，并不单是占中的组织者，它以监控、谣言、谎言、抹黑等手法制造恐惧，破坏整个香港社会。它也让大家看到，这个国家为了铲除异见，不惜以黑客、情报特务来（整）“治”港人。
         </p>
        </div>
       </div>

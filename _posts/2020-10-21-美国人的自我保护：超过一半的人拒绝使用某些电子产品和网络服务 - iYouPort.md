   美国人的自我保护：超过一半的人拒绝使用某些电子产品和网络服务

    <span style="color: #00ccff;">
     <em>
      <strong>
       我们经常听到一句中文，将美国称为 “灯塔”。其实这很好，因为灯塔意味着前进的方向。但这个灯塔照耀的绝不能仅仅有当权者朝着棱镜的 “前进”……
      </strong>
     </em>
    </span>
   </li>
  </ul>
  <p>
   <img alt="" class="aligncenter size-full wp-image-14740 jetpack-lazy-image" data-lazy-sizes="(max-width: 960px) 100vw, 960px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=960%2C549&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?w=960&amp;ssl=1 960w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=300%2C172&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=768%2C439&amp;ssl=1 768w" data-recalc-dims="1" height="549" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=960%2C549&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="960"/>

    <img alt="" class="aligncenter size-full wp-image-14740" data-recalc-dims="1" height="549" sizes="(max-width: 960px) 100vw, 960px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=960%2C549&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?w=960&amp;ssl=1 960w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=300%2C172&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/0-10.jpeg?resize=768%2C439&amp;ssl=1 768w" width="960"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   根据皮尤研究中心在2019年6月3日至17日进行的一项调查显示，当许多美国人认为自己的个人信息不太安全、并关注寡头公司和政府如何使用公民的个人数据时，有相当一部分公众因为隐私问题而选择不使用数字产品或服务。
  </p>
  <p class="graf graf--p">
   大约一半（52%）的美国成年人表示，他们最近决定不使用某项数字产品或服务，因为他们担心自己的个人信息会被收集得太多。
  </p>
  <p class="graf graf--p">
   某些美国人之间有一些区别。例如，在过去12个月内亲历的数据泄露经历是一个因素。那些说有人试图用他们的名字开立信用额度或申请贷款的人，比那些没有经历过这种情况的人更有可能表示自己出于对隐私的担忧而决定不使用某项产品或服务（64% vs. 51%）。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-14738 jetpack-lazy-image" data-lazy-sizes="(max-width: 840px) 100vw, 840px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=840%2C1004&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?w=840&amp;ssl=1 840w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=251%2C300&amp;ssl=1 251w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=768%2C918&amp;ssl=1 768w" data-recalc-dims="1" height="1004" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=840%2C1004&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="840"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-14738" data-recalc-dims="1" height="1004" sizes="(max-width: 840px) 100vw, 840px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=840%2C1004&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?w=840&amp;ssl=1 840w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=251%2C300&amp;ssl=1 251w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/1-2.png?resize=768%2C918&amp;ssl=1 768w" width="840"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   比较那些曾经被恶意者未经允许就入侵了自己的私人社交媒体或电子邮件账户的人和那些没有经历过这种情况的人，也存在类似的模式（63% vs. 51%）。
  </p>
  <p class="graf graf--p">
   而那些表示有人在他们的借记卡或信用卡上进行欺诈性收费的人比没有经历过这种情况的人更有可能有这种担忧（56% vs. 51%）。
  </p>
  <p class="graf graf--p">
   总的来说，经历过上述三种数据泄露事件的成年人比没有经历过这些的人更有可能出于对隐私的考虑而避免使用数字产品或服务（57% vs. 50%）。
  </p>
  <p class="graf graf--p">
   教育水平和家庭年收入的不同也存在一些差异。约55%的大学毕业生和受过一定大学教育的成年人表示，他们因为隐私问题而决定不使用某项产品或服务，而高中或以下学历的人中，45%的人有同样的担忧。
  </p>
  <p class="graf graf--p">
   家庭年收入在75000美元或以上的成年人中，54%的人有同样的顾虑，而年收入在 30000 美元或以下的人中，有48%的人有同样的顾虑。
  </p>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-14739 jetpack-lazy-image" data-lazy-sizes="(max-width: 840px) 100vw, 840px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=840%2C1636&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?w=840&amp;ssl=1 840w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=154%2C300&amp;ssl=1 154w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=526%2C1024&amp;ssl=1 526w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=768%2C1496&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=789%2C1536&amp;ssl=1 789w" data-recalc-dims="1" height="1636" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=840%2C1636&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="840"/>
   <noscript>
    <img alt="" class="aligncenter size-full wp-image-14739" data-recalc-dims="1" height="1636" sizes="(max-width: 840px) 100vw, 840px" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=840%2C1636&amp;ssl=1" srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?w=840&amp;ssl=1 840w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=154%2C300&amp;ssl=1 154w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=526%2C1024&amp;ssl=1 526w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=768%2C1496&amp;ssl=1 768w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/07/2-3.png?resize=789%2C1536&amp;ssl=1 789w" width="840"/>
   </noscript>
  </p>
  <p class="graf graf--p">
   该调查还包括一个分为两部分的开放式问题。
  </p>
  <p class="graf graf--p">
   出于个人信息的考虑而决定不使用产品或服务的成年人，请描述最近作出这一决定时发生了什么。然后询问他们具体是什么产品或服务，以及发现什么问题。
  </p>
  <p class="graf graf--p">
   大约五分之一的美国人（21％）表示，他们决定不使用特定的网站。一小部分人（11％）表示，他们决定不使用某些电子产品。类似数量的人们表示正在避免使用某些社交媒体或特定服务，例如DNA检测、财务工具或医疗保健应用程序。
  </p>
  <p class="graf graf--p">
   至于不使用这些东西的原因，最引人关注的担忧是这些应用程序必须共享个人信息才能访问其产品或服务（15％的人这样认为）。
  </p>
  <p class="graf graf--p">
   第二大担忧是产品或服务不可信（9％）。相似比例的受访者（8％）认为监视是一个大问题。较小比例的受访者提到了其他担忧，例如提供付款信息、潜在的第三方监视、以及垃圾邮件的风险。
  </p>
  <p class="graf graf--p">
   一些受访者阐述了他们的推理。例如，一名64岁的男子说：“一封电子邮件随信附上文章链接。当我单击链接时，该网站要求提供有关我的个人身份信息。我拒绝提供这些信息。”
  </p>
  <p class="graf graf--p">
   一位29岁的女士在解释自己对电子产品的担忧时写道：“我的智能电视要求允许收集音频数据和其他类型的数据进行研究。我对此不满意，所以我拒绝了。”
  </p>
  <p class="graf graf--p">
   一名27岁的女士说：“我最近对基因分析感兴趣，以便更好地了解我的祖先。但是经过研究，我发现公司实际上保留了这些信息并在其他情况下使用它。这使我无法继续使用他们的服务。老实说，我对此类服务的情况并不清楚，但这正是让我警惕使用此类服务​​的原因。” ⚪️
  </p>

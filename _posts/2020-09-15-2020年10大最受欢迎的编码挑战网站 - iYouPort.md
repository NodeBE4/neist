  <p class="graf graf--p">
   学习编码的一种提高技能的好方法是解决编码难题。解决各种类型的挑战和难题可以帮助您领会技巧、学习编程语言的复杂性、准备工作面试、学习新算法等等。
  </p>
  <p class="graf graf--p">
   freecodecamp 推荐了一些很好的资源，以帮助您达成上述目标。
  </p>
  <p class="graf graf--p">
   这里是一些受欢迎的编码挑战网站的列表，并简要说明了每个网站所提供的功能。
  </p>
  <h3 class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.topcoder.com/challenges/?pageIndex=1" href="https://www.topcoder.com/challenges/?pageIndex=1" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     1. TopCoder
    </strong>
   </a>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-14114 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-13.png?resize=800%2C133&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-13.png?w=800&amp;ssl=1 800w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-13.png?resize=300%2C50&amp;ssl=1 300w, https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-13.png?resize=768%2C128&amp;ssl=1 768w" data-recalc-dims="1" height="133" src="https://i2.wp.com/www.iyouport.org/wp-content/uploads/2020/05/1-13.png?resize=800%2C133&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   TopCoder 是在线竞技编程原始平台之一。它提供了一系列过去的算法
   <a class="markup--anchor markup--p-anchor" data-href="https://www.topcoder.com/community/how-it-works/" href="https://www.topcoder.com/community/how-it-works/" rel="noopener noreferrer" target="_blank">
    挑战
   </a>
   ，你可以使用他们的代码编辑器直接在线完成自己的挑战。
  </p>
  <p class="graf graf--p">
   他们流行的单轮比赛每月都会在特定的时间提供多次，您可以和其他人竞争，以最快的速度和最好的分数解决挑战。
  </p>
  <p class="graf graf--p">
   TopCoder 上
   <a class="markup--anchor markup--p-anchor" data-href="https://www.topcoder.com/tc?module=AlgoRank" href="https://www.topcoder.com/tc?module=AlgoRank" rel="noopener noreferrer" target="_blank">
    排名靠前的用户
   </a>
   都是非常优秀的程序员，经常参加编程比赛。
  </p>
  <p class="graf graf--p">
   排名最高的用户维护着自己的博客名为
   <a class="markup--anchor markup--p-anchor" data-href="http://petr-mitrichev.blogspot.com/" href="https://petr-mitrichev.blogspot.com/" rel="noopener noreferrer" target="_blank">
    Algorithms
   </a>
   ，由 Petr Mitrichev 每周撰写，他在其中提供了许多有关编码竞赛，算法，数学等主题的文章。
  </p>
  <h3 class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.coderbyte.com/" href="https://www.coderbyte.com/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     2. Coderbyte
    </strong>
   </a>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-14115 jetpack-lazy-image" data-lazy-sizes="(max-width: 800px) 100vw, 800px" data-lazy-src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-12.png?resize=800%2C288&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-12.png?w=800&amp;ssl=1 800w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-12.png?resize=300%2C108&amp;ssl=1 300w, https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-12.png?resize=768%2C276&amp;ssl=1 768w" data-recalc-dims="1" height="288" src="https://i1.wp.com/www.iyouport.org/wp-content/uploads/2020/05/2-12.png?resize=800%2C288&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="800"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   Coderbyte 提供了200多个编码挑战，您可以使用10种编程语言中的任何一种直接在线解决挑战（看看这个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.coderbyte.com/information/First%20Factorial" href="https://www.coderbyte.com/information/First%20Factorial" rel="noopener noreferrer" target="_blank">
    例子
   </a>
   ）。挑战的范围从简单的（找到字符串中最大的单词）到困难的（打印图形的最大基数匹配）。
  </p>
  <p class="graf graf--p">
   他们还提供了一系列
   <a class="markup--anchor markup--p-anchor" data-href="https://www.coderbyte.com/algorithm/tree-traversal-algorithms" href="https://www.coderbyte.com/algorithm/tree-traversal-algorithms" rel="noopener noreferrer" target="_blank">
    算法教程
   </a>
   、入门视频和
   <a class="markup--anchor markup--p-anchor" data-href="https://www.coderbyte.com/course/google-interview-questions" href="https://www.coderbyte.com/course/google-interview-questions" rel="noopener noreferrer" target="_blank">
    面试准备课程
   </a>
   。与 HackerRank 和其他类似网站不同的是，除了 Coderbyte 发布的官方解决方案外，您还可以查看其他用户提供的关于任何挑战的解决方案。
  </p>
  <h3 class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://projecteuler.net/" href="https://projecteuler.net/" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     3. Project Euler
    </strong>
   </a>
  </h3>
  <p class="graf graf--p">

  <p class="graf graf--p">
   Project Euler 提供了大量的计算机科学和数学领域的挑战。这些挑战通常涉及到编写一个小程序来计算出一个巧妙的数学公式或方程的解法，例如找到一个数列中每个数字前面的
   <a class="markup--anchor markup--p-anchor" data-href="https://projecteuler.net/problem=551" href="https://projecteuler.net/problem=551" rel="noopener noreferrer" target="_blank">
    所有数字之和
   </a>
   。
  </p>
  <p class="graf graf--p">
   您不能直接在网站上用编辑器编码，所以您需要在自己的电脑上编写解决方案，然后在他们的网站上提供它。
  </p>
  <h3 class="graf graf--p">
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackerrank.com/domains" href="https://www.hackerrank.com/domains" rel="noopener noreferrer" target="_blank">
    <strong class="markup--strong markup--p-strong">
     4. HackerRank
    </strong>
   </a>
  </h3>
  <p class="graf graf--p">
   <img alt="" class="aligncenter size-full wp-image-14118 jetpack-lazy-image" data-lazy-sizes="(max-width: 796px) 100vw, 796px" data-lazy-src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-3.jpeg?resize=796%2C416&amp;is-pending-load=1#038;ssl=1" data-lazy-srcset="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-3.jpeg?w=796&amp;ssl=1 796w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-3.jpeg?resize=300%2C157&amp;ssl=1 300w, https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-3.jpeg?resize=768%2C401&amp;ssl=1 768w" data-recalc-dims="1" height="416" src="https://i0.wp.com/www.iyouport.org/wp-content/uploads/2020/05/4-3.jpeg?resize=796%2C416&amp;ssl=1" srcset="data:image/gif;base64,R0lGODlhAQABAIAAAAAAAP///yH5BAEAAAAALAAAAAABAAEAAAIBRAA7" width="796"/>

   </noscript>
  </p>
  <p class="graf graf--p">
   HackerRank 提供了多个不同领域的挑战，如算法、数学、SQL、函数式编程、AI等。
  </p>
  <p class="graf graf--p">
   您可以直接在线解决所有的挑战（看看
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackerrank.com/challenges/simple-array-sum" href="https://www.hackerrank.com/challenges/simple-array-sum" rel="noopener noreferrer" target="_blank">
    这个例子
   </a>
   ）。他们为每一个挑战提供了讨论和排行榜，大多数的挑战都会有一篇评论，解释了更多关于挑战的内容，以及如何做得更好，从而提出解决方案。
  </p>
  <p class="graf graf--p">
   除了评论之外，您目前无法在 HackerRank 上查看其他用户的解决方案。
  </p>
  <p class="graf graf--p">
   HackerRank 还通过解决
   <a class="markup--anchor markup--p-anchor" data-href="https://www.hackerrank.com/jobs/common-app" href="https://www.hackerrank.com/jobs/common-app" rel="noopener noreferrer" target="_blank">
    公司赞助的编码挑战
   </a>
   为用户提供了提交应用程序和申请职位的能力。
  </p>
  <h3 class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    5、
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.codechef.com/" href="https://www.codechef.com/" rel="noopener noreferrer" target="_blank">

     CodeChef
    </strong>
   </a>
  </h3>

   </noscript>
  </p>
  <p class="graf graf--p">
   CodeChef 是一个位于印度的竞争性编程
   <a class="markup--anchor markup--p-anchor" data-href="https://en.wikipedia.org/wiki/CodeChef" href="https://en.wikipedia.org/wiki/CodeChef" rel="noopener noreferrer" target="_blank">
    网站
   </a>
   ，提供数百个挑战项目。
  </p>
  <p class="graf graf--p">
   您可以在他们的在线
   <a class="markup--anchor markup--p-anchor" data-href="https://www.codechef.com/ide" href="https://www.codechef.com/ide" rel="noopener noreferrer" target="_blank">
    编辑器
   </a>
   中编写代码，并根据您自己的技能水平，查看不同类别的挑战集（看看这个
   <a class="markup--anchor markup--p-anchor" data-href="https://www.codechef.com/problems/TREEFUNhttps://www.codechef.com/problems/TREEFUN" href="https://www.codechef.com/problems/TREEFUNhttps://www.codechef.com/problems/TREEFUN" rel="noopener noreferrer" target="_blank">
    例子
   </a>
   ）。
  </p>

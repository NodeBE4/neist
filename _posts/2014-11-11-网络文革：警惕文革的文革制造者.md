         <h1 class="page-title">
          网络文革：警惕文革的文革制造者
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <span class="username" datatype="" property="foaf:name" typeof="sioc:UserAccount" xml:lang="">

       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <br/>
          读者投稿 （辽宁鞍山   H.S）
          <br/>
          <br/>
          10月1日《麦兜故事》的最新电影上映了，我这个从小就喜欢这个系列电影（里）的当代中国人，当然要在第一时间从网上“非法下载”，再“非法观看”一下了。我找到了常用的一个盗版电影下载网站，正准备像平时一样“违法”的时候，底下的评论却吸引了我的目光——一个网友留言“看到黄秋生就不想看了”。黄秋生，他在以前的麦兜动画中为麦兜的父亲配过音，我又向上看了一下演员表单，他果然在其中。
          <br/>
          <br/>
          9月，由于中共政府的长期倒行逆施、不回应香港公众的民主需求，在至少两年的争取和平改良与理性通道的诉求无果之后，
          <a href="https://pao-pao.net/tags/161" rel="nofollow">
           香港人走向了街头
          </a>
          ，开始了时至今日的雨伞革命。港人争取真普选，要求689（梁振英）下台和取消功能组别，很多知名艺人纷纷表达了自己对学生和公民大众的支持，中共中央坐不住了。当局垂帘听政的红色贵族元老们大为不满，习皇帝和御用文人们在惊恐与恼怒中拿出了自家的看家本事——一众被中共收买的商贾巨富和明星文人们粉墨登场，开始了对港人雨伞革命的舆论攻击。
          <br/>
          <br/>
          由于中国国家网络防火长城的存在，中国人的信息渠道一直是被堵死的。这次当局像应对以往民众不满时一样，开足宣传机器，对事实进行了极其可笑的一些列歪曲报道。一瞬间，中国人仇外无知的气氛被炒到了又一个新的顶点。于是，不管是普通网民还是拿钱干活的五毛，无论是失去未来和希望的年轻人，还是已经被中世纪权贵政治祸害了半辈子的中年人，中国人在网上对香港人几乎都是普遍性的不满。甚至有的人还会觉得“他们又有什么了不起，还不是被美国利用的反动分子，晾着他们，要冷处理！”
         </p>
         <div class="half-right">
          <div class="media media-element-container media-full">
           <div class="ds-1col file file-image file-image-jpeg view-mode-full clearfix">
            <img alt="" class="media-element file-full" height="383" src="https://pao-pao.net/sites/pao-pao.net/files/styles/large/public/byzfwjycmaajple.jpg_large.jpg?itok=HGj5BwAK" title="" typeof="foaf:Image" width="480"/>
            <div class="field field-name-field-image-source field-type-link-field field-label-hidden">
             <div class="field-items">
              <div class="field-item even">
               <a href="https://pbs.twimg.com/media/ByzfWJyCMAAJpLe.jpg:large">
                黄秋生Facebook声援占中示威者 图/推特
               </a>
              </div>
             </div>
            </div>
           </div>
          </div>
          <br/>
         </div>
         <p>
          于是，诸多挺港人的艺人被中共的国家级宣传机器人身攻击，这种毁谤直接导致他们失去了名声和在大陆的地位。本来黄秋生是一位非常受大陆人认可和尊敬的艺人，结果，今天到处是骂他、声讨他是汉奸走狗的人。
          <br/>
          <br/>
          我跟同龄人一样喜欢大型网络游戏，我的一个《魔兽世界》QQ群里，群友们依旧在讨论着“爱恨情仇”和“生离死别”，无病呻吟与故作愁态不知愁，这是当代青年的写照。每个时代都会有人担心下一代人不能承担起这个社会的担子，但这只是一种不必要的多虑。然而，在中国却又有了不一样的地方——虽然年轻人不懂最基本的自由经济，不知道权贵们剥夺掉价格的一般均衡所致的纳什均衡会毁灭掉无权无势的平民子弟的人生，虽然他们不懂得自己已经失去了未来，但是，他们依旧在维护自己的统治者，他们不敢哪怕有一丁点越界。他们在“守王法”上成熟得惊人！一句话：中国的青年人，同他们的父母们一样，永远无法搞懂法治和依法人治的区别，就仅仅是因为信息的封锁——没有言论自由。
         </p>
         <div class="half-left">
          <div class="media media-element-container media-full">
           <div class="ds-1col file file-image file-image-jpeg view-mode-full clearfix">
            <img alt="" class="media-element file-full" height="480" src="https://pao-pao.net/sites/pao-pao.net/files/styles/large/public/10468058_488116921327720_6608804555092898784_n.jpg?itok=yDmx6yjK" title="" typeof="foaf:Image" width="360"/>
           </div>
          </div>
         </div>
         <p>
          <br/>
          QQ群里的朋友们和我在大陆“局域网”上看到的网友们，一致声讨港人的民主运动，他们认为这是一种文革。
          <br/>
          <br/>
          文革，在大陆同其他几乎每种事物一样，都是被当局精心调整过的概念，不但去除了威胁中共统治地位的东西，还大大加强了极权统治。
          <br/>
          <br/>
          在历史上，跟文革最为相似的是
          <a href="https://zh.wikipedia.org/wiki/%E4%B9%89%E5%92%8C%E5%9B%A2%E8%BF%90%E5%8A%A8" rel="nofollow">
           庚子拳乱
          </a>
          。义和团声称自己是西太后的“红卫兵”，要“保大清、灭洋教”。毛遮洞*的造反将领们也是如此，他们永远忠于领导、忠于党。实际上，什么是文革呢？支持自家的皇帝呗！就是我们要拥护自己的统治者、统治阶级，打倒一切外来者。排外、无知、恐惧，这些才是文革的本来面貌。(作者注：“毛遮洞”，据前辈们所言，是六十年代民间的暗语。表面上，它是一种对女性生殖器官的侮辱性称谓，实则在当时，是被“反动分子”用于咒骂皇帝的发泄词。）
          <br/>
          <br/>
          但是，今天的中共政府已经将它彻底歪曲了，而且歪曲得非常成功。如今中国人认为，你搞游行、你争人权，你就是在搞乱中国，你就是文革余孽。他们恰恰弄反了一切！他们不知道自己的这种拥护独裁政权的愚昧，才是彻头彻尾的文革——
          <strong>
           网络文革，是今天中国最大的暴行
          </strong>
          ！
          <br/>
          <br/>
          而这一切，都是共产党用中国人的钱、资源，一点一点暗中培养、明里利诱形成的。阴谋、阳谋合到一起耍，党经过三十年的半开放式培养，终于让中国人成为了选择性遗忘的精神病人。于是我们看到了：新时代的中国人——警惕文革的文革制造者，这种矛盾、有趣的存在。这样的罪孽，真是罄竹难书啊……
         </p>
        </div>

        <div class="field-item even" property="dc:title">
         <h1 class="page-title">
          懒人包、Line与台湾反黑箱服贸协议运动
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/54" class="username" datatype="" href="/author/54" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        Agnes Poe
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          <strong>
           台湾学生反服贸协议已堂堂迈入第11天。这个被称为太阳花学运的运动，是为了反对执政党违反逐条审议两岸服务贸易协议的先前协定，3月18日学生占领立法院，24日凌晨遭强制驱离行政院的血腥洗礼，但反抗声势亦没有衰颓的迹象。总指挥林飞帆、陈为廷等人更号召全民3月30日走上街头，包围总统府，让总统马英九与朝野党团看见人民的意志。
          </strong>
          <br/>
          <br/>
          太阳花学运除了写下台湾国会第一回被抗议群众占据的历史，也展现了网络、脸书与微信等新科技对公民运动的影响力。
          <br/>
          <br/>
          <strong>
           什么是懒人包
          </strong>
          <br/>
          在台湾近年来十分流行的“懒人包”，原是将常用软件与相关文件放在一起的压缩档，因此只要下载一个档案就可以找到许多有用的程序，故名之“懒人包”。后来热心网友在遇到重要事件时，将相关新闻链结、信息整理条列出来的网页，亦称为懒人包。懒人包在台湾最近几年的公民运动里，逐渐地扮演起非常重要的角色，从苗栗大埔强徵土地争议、文林苑都更强拆事件与华光社区拆迁事件，当媒体开始报导冲突事件，原本对这件事情不甚了解的人民，可以快速地通过懒人包了解事情的前因后果，迅速决定是否加入抗争。
          <br/>
          <br/>
          到了反黑箱服贸运动，懒人包又再度演化，资料来源的原始链结不再重要，内容多半是已经消化过、具明显立场的简报式文件。相较以往事件只有一两个流通的懒人包，反黑箱服贸运动则经历了懒人包的信息大爆炸，网络上流通的懒人包数以千百计。其中有教授专家制作的懒人包，也有一般网友自行整理的简化版；有反应特定懒人包的懒人包，也有针对单一项目做讨论的懒人包。反服贸与支持服贸者的懒人包通常立场鲜明，就如同它们分别的支持者一般，少有沟通与彼此理解的可能性。
          <br/>
          <br/>
          <strong>
           南辕北辙
          </strong>
          <br/>
          懒人包中最早、也可能是最广泛受到引用的，是台湾大学经济系教授郑秀玲于去年七月制作的懒人包《两岸服贸协议对我国的冲击分析》。她主张虽然中国对台湾开放80项、台湾对中国开放64项，貌似对台湾有利，实则影响的产业高达数千种；她也批评两岸经济规模不对等，中国整合型的大型企业，将对台湾以中小企业为主的经济体系造成严重的伤害。然而最强烈的指控，其实在她对出版与移民两项的批评：她认为，服贸将会严重损害台湾出版业与言论自由，而中国有钱人将藉此假投资真移民。这套懒人包不仅广泛流传于网络上，也是反服贸人民根本的理论依据。然而反”反服贸”的懒人包则批评她立场偏颇、证据不足，充满个人偏见与恐中心理。
          <br/>
          <br/>
          赞成服贸的这一方，则多半以经济部长达122页的《服贸Q＆A》为主要依据，或是从了解中国现状的经验谈角度入手，指出中国目前的薪资条件优于台湾，大可不必担心大陆劳工大举侵台，并强调台湾若不及时签署服贸，将落后韩国，逐渐被边缘化。赞成与反对两方的意见南辕北辙，然而共同的问题是，除了少数对特定议题有了解的专业人士外，没有人能对复杂的服贸协议提出一个更中肯的解读，只能在两方学者的意见间选边站。
          <br/>
          <br/>
          <strong>
           网络动员与集资
          </strong>
          <br/>
          服贸虽然是问题的核心，却不是太阳花学运的主题。学生们多次强调，他们反的不是服贸，而是审理服贸过程间的黑箱作业。这个诉求显然远比53页的服贸协议来得简洁易懂，也迎合了民间普遍对马英九政府的不信任，因此得到了近七成人民的支持（根据民进党民调中心民调）。然而这些据守立法院的学生是怎么突破台湾蓝绿媒体壁垒分明的局面，将自己的主张与意见直接传播出去的呢？
          <br/>
          <br/>
          首先他们通过脸书开辟了几个粉丝专页，如“
          <a href="https://www.facebook.com/portestallover" rel="nofollow">
           遍地开花！反黑箱服贸协议
          </a>
          ”、“
          <a href="https://zh-cn.facebook.com/lslandnationyouth" rel="nofollow">
           黑色岛国青年阵线
          </a>
          ”、“
          <a href="https://zh-tw.facebook.com/WatchOutTW" rel="nofollow">
           沃草（Watchout）
          </a>
          ”等页面，成为他们对外发声的渠道；架起IPad通过Ustream立刻就建立起实况转播的渠道，效果不比媒体耗资千万的SNG系统来得差。
          <a href="http://line.me/en/" rel="nofollow">
           Line（连我）
          </a>
          （类似微信的app）也成为学生串连动员的最佳利器，不仅可以分享文件、信息，群组联络的即时通讯，尤其在23日占领行政院与遭驱除时扮演了极为重要的角色。当媒体被抗暴警察阻挡在外，在场学生与人民纷纷用自己的相机与手机纪录镇压的实况，将警察打人的影像立即传上脸书与Youtube。
          <br/>
          <br/>
          支持学运的三千多名人民更通过网络串连组成“公民合购团队”，先是在25日买下《苹果日报》与《自由时报》的头版半版广告，宣扬反黑箱服贸的观念，并将在29日起连续两天在《纽约时报国际版》刊登广告。将近20万人民币的广告费仅花半天的时间就募集成功。
          <br/>
          <br/>
          <strong>
           公民审服贸
          </strong>
          <br/>
          然而在这次的公民觉醒运动里，网络最有建设性的功能才刚要浮现。过往几天，几个汇整梳理服贸内文、并提供论坛与投票功能的网站接连成立，标榜“公民自审”、强调信息透明公开，虽然有的仍设有立场，但大体而言仍将问题回归到条文本身，主张让人民自行决定。也有团体建设网站平台，开始汇集各种产业对服贸的意见。虽然服贸的条文与其法律规范似乎对一般人民来说仍过度专业，也无法保证在这些网站上某人发表的意见是否真的代表了整个产业的见解，但其将服贸议题透明化、公开化的努力，还是令人耳目一新。
          <br/>
          <br/>
          <strong>
           网络时代的真理
          </strong>
          <br/>
          后行动网络时代的公民运动，在信息传播与动员方面，无疑地远比以往更有效率也更直接。尤其当人民对媒体与朝野政党的信任不再，对他们传布的信息充满疑惑之时，网络为他们杀出了一条血路。
          <br/>
          <br/>
          然而这些即时、往往也未经查证的信息，也可能冒着错谬的风险，诸如脸书上曾经流传一张在24日凌晨强行驱除占领行政院学生时，抗暴警察提脚往学生踹下的画面。这张照片出现后立即遭到疯狂转载，随后有人找到现场的动态影像画面，证实画面中的警察只是要跨过静坐学生罢了。澄清虽然也来得非常迅速，但它强烈的影像效果仍盘桓在人们的心中，成为警察暴力的象征。
          <br/>
          <br/>
          这张照片以及仍处在两极的服贸论述，让我们看见了网络时代真理的局限与其价值。一方面，如果公民欠缺记录与传播的渠道，而拍下照片并给予诠释的是媒体或官方，事情的真相或许将永远石沈大海。当传播成为公民的权利与工具之一，我们也必须担负起求证与探索真相的责任。这正是太阳花运动给我们的最重要的启示：真正的民主需要公民的参与，被动地等待官方给予说法，或毫不质疑地接受媒体报导，形同放弃了身为公民的权利。或许，台湾民主真正黄金时代才要来临呢！
         </p>

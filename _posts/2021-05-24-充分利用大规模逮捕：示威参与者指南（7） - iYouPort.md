  <p class="graf graf--p">
   2017年1月20日，美国发生了历史上最激烈的抵抗总统就职典礼的抗议事件。当天，有远超100名示威者在12号和L号交叉路口被警察包围并被逮捕。这是自15年前的
   <a class="markup--anchor markup--p-anchor" data-href="http://flag.blackened.net/atb/en/october2002_strike_call.html" href="http://flag.blackened.net/atb/en/october2002_strike_call.html" rel="noopener" target="_blank">
    人民大罢工
   </a>
   以来，华盛顿最大规模的大规模逮捕行动，而2020年的就职典礼示威活动中的具体统计数字还没有出来。
  </p>
  <p class="graf graf--p">
   📌 这次逮捕使用的是水壶战术，我们曾经介绍过这种抓捕方式见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e8%80%85%e5%a6%82%e4%bd%95%e5%af%b9%e4%bb%98%e8%ad%a6%e5%af%9f%ef%bc%9a%e8%ad%a6%e6%83%95%e6%b0%b4%e5%a3%b6%e6%88%98%e6%9c%af/" href="https://www.iyouport.org/%e6%8a%97%e8%ae%ae%e8%80%85%e5%a6%82%e4%bd%95%e5%af%b9%e4%bb%98%e8%ad%a6%e5%af%9f%ef%bc%9a%e8%ad%a6%e6%83%95%e6%b0%b4%e5%a3%b6%e6%88%98%e6%9c%af/" rel="noopener" target="_blank">
    抗议者如何对付警察：警惕”水壶”战术
   </a>
   》。在下面看到当时的场景视频 —— 注意在警察的围捕 “水壶” 开始形成的最初，一些抗议者很有组织地号召人们冲锋，举着雨伞地抵挡警方的反击。最终只有这些人逃脱了，后面没有跟上来的100人都被捕了：
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="659" src="//player.vimeo.com/video/200490022?title=0&amp;byline=0&amp;portrait=0&amp;color=8dc7dc" width="800">
   </iframe>
  </p>
  <p class="graf graf--p">
   这批抗议者与经历过大规模逮捕的前辈之间的差距显而易见。
   <strong class="markup--strong markup--p-strong">
    有些人无意中让警察收集到了有关他们的情报；许多人对逮捕过程的基本知识很是缺乏
   </strong>
   。
  </p>
  <p class="graf graf--p">
   当下正是群众抗争和大规模逮捕的新阶段，本文将尝试梳理一些历史事件中的经验教训，以便帮助未来的行动者尽可能避免被逮捕、利用被逮捕、尤其是在拘留所里保护自己。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1. 如果您拿定主意破釜沉舟，那就必须非常快速地行动
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   周五的水壶围困中，有些人确实逃过了逮捕这一劫。一场勇敢冲锋让一些反抗者挣脱了束缚（如上面的视频），并在当天余下的时间里在街头继续战斗。
  </p>
  <p class="graf graf--p">
   他们成功的关键是，他们冲锋的时候，最大部队的警察力量还守在12号和L号路交叉口。
  </p>
  <p class="graf graf--p">
   📌 被水壶围困的时间越长，就越难突围：警察会快速建立起自己的势力和监控，被围困的抗议者会迅速降低士气。所以，如果您想尽可能摆脱这种围困逮捕，
   <strong class="markup--strong markup--p-strong">
    最好的机会就是在水壶战术开始实施的最初突破防线
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     2. 利用团队的中间环节，让自己得以摆脱敏感物品
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   一旦确定被包围，你就没有逃跑的机会了，这时候你就应该立即更换服装；
   <strong class="markup--strong markup--p-strong">
    尽量在不被拍到的情况下更换服装
   </strong>
   。
  </p>
  <p class="graf graf--p">
   📌 在任何一个行动组织中，穿一套最普通的衣服在里层，都是很重要的 —— 我们曾经介绍过这点，见第5集《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e9%9b%a8%e4%bc%9e%e5%88%b0%e6%bf%80%e5%85%89%e3%80%81%e6%b0%94%e7%90%83%e5%88%b0%e7%94%b5%e5%8a%a8%e5%b7%a5%e5%85%b7%ef%bc%9a%e7%a4%ba%e5%a8%81%e5%8f%82%e4%b8%8e%e8%80%85%e6%8c%87%e5%8d%97/" href="https://www.iyouport.org/%e4%bb%8e%e9%9b%a8%e4%bc%9e%e5%88%b0%e6%bf%80%e5%85%89%e3%80%81%e6%b0%94%e7%90%83%e5%88%b0%e7%94%b5%e5%8a%a8%e5%b7%a5%e5%85%b7%ef%bc%9a%e7%a4%ba%e5%a8%81%e5%8f%82%e4%b8%8e%e8%80%85%e6%8c%87%e5%8d%97/" rel="noopener" target="_blank">
    从雨伞到激光、气球到电动工具
   </a>
   》，因为只有这样你才可以
   <strong class="markup--strong markup--p-strong">
    迅速转变自己的面貌
   </strong>
   。
  </p>
  <p class="graf graf--p">
   在遭遇大规模逮捕的情况下，
   <strong class="markup--strong markup--p-strong">
    最稳妥的方法是移动到人群中心位置，隐藏在人群中看不到你的地方，脱掉任何可疑的物品
   </strong>
   — — 比如行动标识、与这场行动相关的装束等，换上你的便装，然后移动到人群的外围，为那些还没有换服装的人提供掩护。
  </p>
  <p class="graf graf--p">
   📌 也就是说换衣服需要有组织地交替进行，不能是一下子所有人都想要获得换衣服的机会，那将没有任何人能拥有这个机会。
   <strong class="markup--strong markup--p-strong">
    必须有足够多的人愿意为同伴做掩护。
   </strong>
  </p>
  <p class="graf graf--p">
   在换衣服的时候，
   <strong class="markup--strong markup--p-strong">
    要认识到可能会让你暴露的 *所有* 细节
   </strong>
   ，比如你的鞋子或背包。例如，你可以把早前背着的包包放在另一个更大的包包里面，或者放在你的黑色装备下面（外面套一个外套），或者用黑色胶带遮住鞋子上的可识别性标志。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     3. 不要让警察的工作变得简单
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   当被扣留在12号和L号路口的抗议者人群被要求把手举起来时，大多数人都举起了手。这不是他们唯一能做的事。
  </p>
  <p class="graf graf--p">
   📌 请记住，
   <strong class="markup--strong markup--p-strong">
    我们在囚禁中能找到的安全感永远不会来自于服从命令，而是来自于我们如何利用警方要求的任何合作，而这些合作仍在我们的控制之下
   </strong>
   。
  </p>
  <p class="graf graf--p">
   如果他们让你做什么，可能是因为这能让他们的工作更轻松。理想的情况是，被拘留的人群应该建立一个程序，共同决定如何回应警察的命令。
  </p>
  <p class="graf graf--p">
   非暴力
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e5%88%86%e6%ad%a5%e6%8c%87%e5%8d%97%ef%bc%9a%e5%ae%83%e6%98%af%e4%bb%80%e4%b9%88%e3%80%81%e5%ae%83%e6%9c%89%e4%bb%80%e4%b9%88%e7%94%a8%e3%80%81%e5%a6%82%e4%bd%95/" href="https://www.iyouport.org/%e7%9b%b4%e6%8e%a5%e8%a1%8c%e5%8a%a8%e5%88%86%e6%ad%a5%e6%8c%87%e5%8d%97%ef%bc%9a%e5%ae%83%e6%98%af%e4%bb%80%e4%b9%88%e3%80%81%e5%ae%83%e6%9c%89%e4%bb%80%e4%b9%88%e7%94%a8%e3%80%81%e5%a6%82%e4%bd%95/" rel="noopener" target="_blank">
    直接行动
   </a>

  </p>

   社会里，
   <strong class="markup--strong markup--p-strong">
    日常情况下几乎所有人都是孤立的，长期缺乏理解、信赖和同情，当审讯者施加伪造的理解和同情时，很多人会难以抗住
   </strong>
   。避免这种情况下发生的唯一办法就是，
   <strong class="markup--strong markup--p-strong">
    您的队伍中必需用爱包围起来，用互爱连接所有人，不能让任何队友被边缘
   </strong>
   。
   <strong class="markup--strong markup--p-strong">
    只有熟悉被爱的人才能准确及时地识别审讯者制造的虚假善意。
   </strong>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     8. 不要和警察说话
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 必需不断重复这点  — — 你对警察说什么都不会有任何帮助，请记住这点。您只需要说：“对不起，警官，我的律师告诉我，我不应该告诉警察任何事，除了我的名字之外”。就这么简单。
  </p>
  <p class="graf graf--p">
   一旦你进了监狱，警察就会竭尽全力让你开口。其手段范围从比较无辜的样子，比如 “那么，下面发生了什么事？” 到更明显的陷阱，比如 “如果有人对自己的罪名感到紧张，我们可以去谈谈你的罪名和接下来会发生的事 …… 我们只需要你放弃保持沉默的权利”。
   <strong class="markup--strong markup--p-strong">
    即使你确定自己没有做任何违法的事，你所说的话也会让其他人陷入危险之中  — — 警察越是把你看成是一个愿意和他们交谈的人，他们就会给你更大的压力。看似不相关的谈话，都可以用来确定被拘留者之间的关联，以及以后用来对付你
   </strong>
   。
  </p>
  <p class="graf graf--p">
   请专注于度过难关，直到你被释放。之后，在律师的指导下，你将有足够的时间减压。
  </p>
  <p class="graf graf--p">
   📌 请记住，你很可能在牢房里被录音和/或录像。在拘留室内，由于身边似乎 “没有警察“，许多被捕者会感觉下意识地放松，开始公开谈论自己的生活、自己的组织工作、当天的事件以及许多其他话题，但却忘了很可能有隐藏的监视器存在。这些话题即使不直接入罪，也会给你的对手提供有关你们的行动项目和相互联系的信息，这只会伤害我们。
  </p>
  <p class="graf graf--p">
   警察和管教人员很可能会有不公正的行为。
   <strong class="markup--strong markup--p-strong">
    他们是想激怒你
   </strong>
   。你可能会发现自己想告诉他们到底应该去哪里，以及其他与他们相似的卑鄙人物。还是省省吧。
   <strong class="markup--strong markup--p-strong">
    如果你自己憋不住，就想想其他和你一起被捕的人，为他们憋着。与警察的任何一种互动都不能改善你的案子或你周围人的案子
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     9. 不要相信警察说的任何话
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 一旦被捕，你就处于一种不确定的状态，在这种状态下，你对自己的处境几乎没有控制权  — — 这种不确定性可能导致的焦虑心理就是警察对付你的主要武器之一。
   <strong class="markup--strong markup--p-strong">
    如果你能在被拘留期间努力保证自己暂时不对潜在的结果做出情绪反应，你就能夺走警察对付你的大部分权力
   </strong>
   。
  </p>
  <p class="graf graf--p">
   警察是很会欺负人的。他们会说谎话只是为了吓唬你：说你会面临几年的牢狱之灾，说你在监狱里会被殴打，说其他被捕者跟他们说过 “你做了什么” 。
   <strong class="markup--strong markup--p-strong">
    不要相信任何一个字，一切都是假的
   </strong>
   。你的案子的证据以后会很清楚，不要听信警察说的任何话。
   <strong class="markup--strong markup--p-strong">
    在你被逮捕的时候，要集中精力确保警察从你那里得到的信息尽可能地少，这非常重要！
   </strong>
  </p>
  <p class="graf graf--p">
   即使你莫名其妙地遇到了一个真心实意的好警察，他们也无论如何不能做出任何决定或承诺。他们可能对实际发生的事知之甚少。他们不发号施令，他们只是向上面传递信息。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    所有人要齐心协力熬过被逮捕关押的过程
   </strong>
   ，而不是去纠结警察的话，那没有任何意义。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     10. 保持士气 (我们需要新的歌谱)
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   自从被水壶围困之后，一些被捕者都抱怨过那些行动中使用的歌曲。大部分只是类似 “永远团结” 的合唱。这些歌曲在某些情况下是不错的，但这一次人们见证了华盛顿特区15年来最大规模的逮捕，而这些歌曲与十年前大规模逮捕时人们所唱的歌曲简直一模一样。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    我们需要新的歌曲，让人感到新鲜和现代
   </strong>
   。当然，要找到一首大家都喜欢的歌曲是不可能的，但至少可以通过扩大选择范围来保持它的趣味性。集体歌集应该包括针对各种情绪和环境的赞歌：欢欣鼓舞的曲目，没错，但也包括凝聚力量、严肃的团结、嘲笑警察，以及在不确定的情况下传播决心的力量。
   <strong class="markup--strong markup--p-strong">
    人们还需要其他方式来维持士气，同时一起打发时间，比如做游戏、讲故事和说笑话
   </strong>
   。
  </p>
  <p class="graf graf--p">
   抗议者特别感谢所有在集会对面围观的人们，他们通过挥手和大喊声援来保持抗议者的精神状态  — — 但到目前为止，最令人振奋的支持是两个街区外的燃烧的豪华轿车从第13街传来的烟雾和橡胶轮胎燃烧的气味。这些勇气的传染，
   <strong class="markup--strong markup--p-strong">
    街道上其他地方的激烈抵抗声比任何歌曲都要更加甜美
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     11. 一分预防胜过十分治疗
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   📌 所有这些建议只要稍加预想就会更加有效。例如，如果事先稍作计划，也许被拘留者的发言就会更有说服力；上述绝食的提议也是如此。在您的亲密团队中
   <strong class="markup--strong markup--p-strong">
    提前计划好一旦被围困，你们会怎么做
   </strong>
   — — 是否要突围，如何迅速摆脱可能被定罪的一切物品，谁会携带一次性电话，以及如何处理它们。
  </p>
  <p class="graf graf--p">
   在反全球化抗议运动中，监狱声援战术取得了一些胜利，原因之一就是在组织大规模抗议活动的过程中，这些战术都是事先公开策划的。从理论上讲，你有可能在被捕后讨论并采用同样的团结战术，但在如此短的时间内很难得到其他人的认同。
  </p>
  <p class="graf graf--p">
   📌 如果您忘记了战术策划和战略制定的方法，请回顾我们的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" href="https://www.iyouport.org/%e7%a1%ae%e5%ae%9a%e5%8f%98%e5%8c%96%e7%9a%84%e6%9d%a0%e6%9d%86%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%882%ef%bc%89-%e6%88%98%e7%95%a5%e6%95%88/" rel="noopener" target="_blank">
    信息行动主义
   </a>
   》指南系列，目前进行的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e4%bb%8e%e6%b8%b8%e6%88%8f%e4%b8%ad%e8%8e%b7%e5%be%97%e5%9b%a2%e9%98%9f%e5%8d%8f%e4%bd%9c%e8%83%bd%e5%8a%9b%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%886/" href="https://www.iyouport.org/%e4%bb%8e%e6%b8%b8%e6%88%8f%e4%b8%ad%e8%8e%b7%e5%be%97%e5%9b%a2%e9%98%9f%e5%8d%8f%e4%bd%9c%e8%83%bd%e5%8a%9b%ef%bc%9a%e4%bf%a1%e6%81%af%e8%a1%8c%e5%8a%a8%e4%b8%bb%e4%b9%89%e6%8c%87%e5%8d%97%ef%bc%886/" rel="noopener" target="_blank">
    第6集
   </a>
   。
  </p>
  <p class="graf graf--p">
   最重要的是，
   <strong class="markup--strong markup--p-strong">
    要在情感上和精神上为被拘留做好准备
   </strong>
   。熟悉自己的权利和典型的警察谎言和诱导伎俩。练习一句话，比如：“在我有律师之前，我只会给你我的名字和地址”。或者最自信地表达你不会和警察说话的原则。排练一下。让自己习惯于说出来。
  </p>
  <p class="graf graf--p">
   <iframe allowfullscreen="allowfullscreen" height="449" src="//www.youtube.com/embed/-4PPjladbsg" width="800">
   </iframe>
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     12. 最好任何时候都不要和警察说话
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   在你被释放后，你不必和给你打电话的警察、侦探或地区检察官交谈。向你的律师和你的队友们报告任何警察对你的接触或骚扰。
  </p>
  <p class="graf graf--p">
   📌 另外，不要做任何 “实际上等于” 和警察说话的事：不要在获释后和媒体交谈  — — 无论他们发表什么有你名字的文章，对警察来说都和与你直接交谈一样有用。在签署任何集体诉讼之前，先和你的律师谈谈 — — 如果你的刑事案件还在进行中，有些案件可能会要求取证，强迫你作证，这可能会成为对你不利的证据。
  </p>
  <p class="graf graf--p">
   📌 一定践行良好的《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e7%9a%84%e5%9b%a2%e9%98%9f%e9%99%b7%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%e5%92%8c%e8%a2%ab%e6%b8%97%e9%80%8f%e7%9a%84%e6%81%90%e6%85%8c%e4%b8%ad%ef%bc%8c%e5%8f%aa%e6%98%af/" href="https://www.iyouport.org/%e5%a6%82%e6%9e%9c%e6%82%a8%e7%9a%84%e5%9b%a2%e9%98%9f%e9%99%b7%e5%9c%a8%e8%a2%ab%e7%9b%91%e8%a7%86%e5%92%8c%e8%a2%ab%e6%b8%97%e9%80%8f%e7%9a%84%e6%81%90%e6%85%8c%e4%b8%ad%ef%bc%8c%e5%8f%aa%e6%98%af/" rel="noopener" target="_blank">
    安全文化
   </a>
   》！并考虑如果警方没收了你的智能手机，可能会获得哪些关于你和你的团队的信息。更改你的密码。尽快把你的手机号码换成新的，这样来电就不会一直打到警察持有的号码上。
  </p>
  <p class="graf graf--p">
   以上这些建议会为您提供一个起点  — —
   <strong class="markup--strong markup--p-strong">
    但不要完全照搬
   </strong>
   。新的抗议者可以问问前几代的抗议者，他们应对大规模逮捕的经历。时代已经变了，但其中许多教训仍然是相关的 — — 包括一些我们无法在这样的描述的东西。
  </p>
  <p class="graf graf--p">
   <strong class="markup--strong markup--p-strong">
    一定要随机应变！您了解的可能性越多，您的应变计划就越丰富，您就越容易做到冷静沉着
   </strong>
   。好运。⚪️
  </p>
  <p class="graf graf--p">
   —— 未完待续 ——
  </p>

         <h1 class="page-title">
          2014中国互联网的英雄：“编程随想”博客
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/714" class="username" datatype="" href="/author/714" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        萧强
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          博客在中文的网络空间兴起，已有超过十年的历史了。在起起落落的中文博客中，我一直非常钦佩的，是一个叫做
          <a href="https://d2x2nunjal7182.cloudfront.net/" rel="nofollow">
           “编程随想”的博客
          </a>
          。这个博客在墙外开了至少五年多了，打开其主页，右边的侧栏上就有作者对其博客主要内容的自我介绍：“提升思维能力，普及政治常识，各种翻墙姿势，党国的阴暗面，网络安全知识，软件开发技术”。而作者自己推荐一些帖子，有纯粹信息技术类的“如何选择开源项目？”，也有介绍突破网络封锁的的“如何翻墙(全方位入门扫盲)”，普及网络安全的“如何防止黑客入侵(系列)”和“如何保护隐私(系列)”，关于阅读和思考的“扫盲逻辑谬误——以五毛言论为反面教材”以及政治类的评论文章“回顾"六四"(系列)”，“俺整理的《太子党关系网络》”等等。除了和互联网、信息技术有关的帖子，这位匿名作者对大量的网络政治文化信息，有独特而又清晰的整理，归纳和分析，拥有了大量翻墙而来的中国国内读者，以至于墙内的“知乎”网站，也有网友开了“编程随想时一个怎样的人？”的词条。
          <br/>
          <br/>
          所以，当泡泡的编辑请我推荐一位2014年中国互联网的“英雄”的时候，我第一个想到的，就是这位“编程随想”的作者。因为不知道他的真实名字，我愿意在这里尊称他为“随想君”。随想君在他的文字里，对于一党专制及其网络审查制度有犀利的批判，而更为难得的，是他数年如一日的坚持和过人的勇气。在中文互联网上，过去十年中表达同样观念，传播类似信息的网友不下于成百上千万。随想君用一个固定的博客和网名，持续以恒地搜集，整理，分享这些信息，并且以自己广博的阅读，清晰的思考，专业的知识，透过明了亦风趣的文字地表达和传播，并且和大量阅读的网友互动互答，可以说以一人之力，建立了一个对中国的审查制度和网络封锁抵抗的社区。这样的努力和扩散的影响力，确实很难有其他中文博客可以比肩。
          <br/>
          <br/>
          随想君不仅有渊博的知识和敏锐的思考能力，他同时还是一位精通网络和信息技术的专业好手。他发表的文字和分享的信息，因为直指审查制度甚至一党专制意识形态的命门要害，不可能在墙内公开传播，一旦被中国的警察特务机关找出了作者真实的身份，更会有人生安全的风险甚至牢狱之灾。这个结论，只要看看那些在网上以实名向专制进行挑战而受到种种形式迫害的长长名单就一目了然：刘晓波，艾未未，许志永，
          <a href="https://pao-pao.net/node/312" rel="nofollow">
           李承鹏
          </a>
          ...... 。和这些名字不同的是，编程随想君使用的是固定的笔名，同时发挥了自己精通网络安全技术的优势，透过技术手段来保护自己不被专制政权的镇压机构追踪的真实身份。
          <br/>
          <br/>
          随想君能数年如一日和国家机器做这样的周旋，当然让人信服其技术能力。但我更加钦佩的，是他在其言行背后的巨大勇气。因为这样匿名的抵抗形式，绝对不比实名的风险来得更小。透过技术手段保护自己，同时又采用固定网名，发表大量专制者决不能容忍的信息的做法，等于是在观念和技术两个层次上，同时对专制者的权威，做出了公开的挑战和嘲弄。这是中国的专制者绝不能容忍的。作者真实身份一旦被拥有巨大技术资源的专制机构发现，遭到的惩罚可能比起实名的挑战者更加严重。因为公众不知道作者的实名，专制者可以使用最卑鄙阴暗的手段。随想君是一个思想者，尽管“艺高人胆大”，也不会不懂自己这样做巨大的风险所在。他能够一直坚持犀利的写作，一定在明了风险的前提下，对自己选择的所言所行，有为信念而承担的决心。正因为此，我愿意推荐他为2014年中国互联网的“英雄”。
          <br/>
          <br/>
          2014年的岁末，中国的互联网仍然处于审查的严冬。怀着对“随想君”的钦佩，我写下这几句介绍的话。因为我相信，人心必然向往自由，有审查就有对审查的抵抗。信息技术的发展本身，也会不断地让专制者为审查制度付出日益高昂的经济和政治代价。在不是那么遥远的未来一天，人们定会看到专制之墙最后的崩塌。我希望未来的人们，会记住为了这自由一天的到来，在所有那些实名的，匿名的，无名的点滴抵抗行动的背后，一个个真实的生命，以及他们的智慧，勇敢，坚持和牺牲。而“编程随想”和它的作者，即使今天我们还不知道他的真名实姓，打开他的一篇篇博文，每一个读者都会看到，这就是21世纪的中国人，在网络上抵抗专制，争取自由的真切证明。
         </p>

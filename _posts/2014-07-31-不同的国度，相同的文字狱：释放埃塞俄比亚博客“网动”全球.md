         <h1 class="page-title">
          不同的国度，相同的文字狱：释放埃塞俄比亚博客“网动”全球
         </h1>
        </div>
       </div>
      </div>
      <div class="field-name-author">
       <div class="label-inline">
        文/
       </div>
       <a about="/author/11" class="username" datatype="" href="/author/11" property="foaf:name" title="查看用户资料" typeof="sioc:UserAccount" xml:lang="">
        阿蔼
       </a>
      </div>
      <div class="field field-name-body field-type-text-with-summary field-label-hidden">
       <div class="field-items">
        <div class="field-item even" property="content:encoded">
         <p>
          今天是全球呼吁释放埃塞俄比亚 #FreeZone9Bloggers 的行动日。
          <br/>
          <br/>
          在翻看9位博客背景和被捕因由，与中国大陆的情况几乎一模一样，忽然想起，埃塞俄比亚是中国友邦，过去几年，中国政府提供了大量奖学金给非洲，让他们的大学生和公务员到大陆学习"政务"，以推广"中国模式"。去年中国国家行政学院与埃塞俄比亚管理学院就签订了合作备忘录，要深入交流管治之述。专制统治不断的结连，人民之间却缺乏了解与关怀。
          <br/>
          <br/>
          <strong>
           博客与反恐法
          </strong>
          <br/>
          这次9位被捕的博客和记者是Befegadu Hailu、Abel Wabela、Atnaf Berahane、Mahlet Fantahun、Zelalem Kibret、Natnael Feleke、Asmamaw Hailegeorgis、Tesfalem Waldyes 和 Edom Kassaye，前6位是一个名为 Zone 9 的集体博客平台的成员。这个平台于2012年成立，透过经营一个集体平台发表社会政治评论文章，推动公民介入社会。
          <br/>
          <br/>
          在今年4月25至26日，埃塞俄比亚的中央调查办公室（类似大陆的国保部门）拘捕了这9位 Zone 9 的成员，指他们与"境外的人权活跃份子合作"，"利用社交媒体制造国家不稳定"（using social media to destabilize the country)。若埃塞俄比亚决定以反恐法起诉他们，最高的刑罚为死刑。这条反恐法例于2009年通过，而在短短4年间，已经有超过200人因这法例被捕，当中有35个反对派政治人物与记者被判参与恐怖主义活动。
          <br/>
          <br/>
          这次被捕的9名博客与记者中，Tesfalem Waldeys 就曾经担任一家独立经营的报纸 (Addis Neger)的编辑，该报馆在反恐法通过后，被迫令关闭，很多编辑记者都逃出国外，但 Tesfalem 坚持留下从事独立记者工作。他早在2012年就指出，反恐条例通过后，报导也会变成恐怖主义：
         </p>
         <blockquote>
          <p>
           恐惧与自我审查在埃塞俄比亚的记者圈裡蔓延。很多记者朋友都感到沮丧，他们看不到这个行业的未来，他们当中有些正打算出走过流亡的日子，有少数则已经有随时被捕的心理准备。没有人知道下一个将是谁。
          </p>
         </blockquote>
         <p>
          <strong>
           温饱与民主
          </strong>
          <br/>
          正如“
          <a href="https://advocacy.globalvoicesonline.org/2014/05/02/statement-global-voices-calls-for-the-release-of-nine-journalists-in-ethiopia/" rel="nofollow">
           全球之声倡议项目”的一篇文章
          </a>
          指出，埃塞俄比亚打压互联网言论的做法非常奇怪，因为这个国家61%成年人文盲，互联网的渗透率只有1.5%，言论又怎可能对政权构成实质威胁？
          <br/>
          <br/>
          所谓“与外国人权份子合作”的博客与记者，他们只在言论的层次，对政府的一套只重视温饱的发展观提出异议，他们认为非洲贫穷国家把人民描述成只追求温饱的一群本身，是内化殖民者对非洲的刻板印象，非洲人民亦有权享有民主和法治，但当权者却指这些人被西方观念“洗脑”。
          <br/>
          <br/>
          “与外国势力合作”、“温饱权”、“制造不稳定”这些概念，我们都很熟悉吧！同一串话语，在不同的地区，发挥同一样的效果，协助专权者摧毁一切求变的声音。
          <br/>
          <br/>
          在香港，廿三条尚未立法，主流媒体自我审查靠边站，支持民主的《
          <a href="https://pao-pao.net/node/139" rel="nofollow">
           主场新闻》突然结束
          </a>
          ，网络反对力量失去了一个平台。
          <br/>
          <br/>
          我们的言论和新闻自由与中国大陆、埃塞俄比亚等专权越拉越近，可能在不久的将来，我们也要向外求救，呼吁释放我们的博客与记者。
          <br/>
          <br/>

           </a>
          </em>
         </p>

  <p class="graf graf--p">
   和平集会是受到宪法保护的公民权利 —— 在全球大多数国家的宪法中都有此条款，包括中国。中国宪法第35条写道，“中华人民共和国公民有言论、出版、集会、结社、游行、示威的自由。”
  </p>
  <p class="graf graf--p">
   但是，如您所知，
   <strong class="markup--strong markup--p-strong">
    完全和平的抗议活动同样在全世界范围内都是暴力镇压的主要受害者，
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%b0%91%e8%af%b4%e8%af%9d%ef%bc%8c%e5%a4%9a%e5%8a%9e%e4%ba%8b-%e5%bd%93%e5%9d%8f%e8%9b%8b%e6%89%8b%e6%8c%81%e5%ae%aa%e6%b3%95/" href="https://www.iyouport.org/%e5%b0%91%e8%af%b4%e8%af%9d%ef%bc%8c%e5%a4%9a%e5%8a%9e%e4%ba%8b-%e5%bd%93%e5%9d%8f%e8%9b%8b%e6%89%8b%e6%8c%81%e5%ae%aa%e6%b3%95/" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     宪法几乎 “顶个球”
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    。您可以在我们的
   </strong>
   <a class="markup--anchor markup--p-anchor" data-href="https://start.me/p/1kod2L/iyp-direct-action-5" href="https://start.me/p/1kod2L/iyp-direct-action-5" rel="noopener" target="_blank">
    <strong class="markup--strong markup--p-strong">
     “维稳面面观” 板块
    </strong>
   </a>
   <strong class="markup--strong markup--p-strong">
    中看到非常多镇压武器和技术的调查曝光，它们都来自于那些是具有冠冕堂皇的宪法的国家。
   </strong>
  </p>
  <p class="graf graf--p">
   人类历史上的几乎每一次抗议都会遭到镇压，抗议者常常得不到任何有效的援助。对那些不甘于忍耐的人们来说，抗议是一种政治工具，涵盖了广泛的形式，从和平抗议和静坐、到大声疾呼的示威，经常会导致与镇压部队发生冲突。
  </p>
  <p class="graf graf--p">
   我们在今天发布的另一篇文章中已经介绍了宪法为什么不等于宪政，见《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e5%b0%91%e8%af%b4%e8%af%9d%ef%bc%8c%e5%a4%9a%e5%8a%9e%e4%ba%8b-%e5%bd%93%e5%9d%8f%e8%9b%8b%e6%89%8b%e6%8c%81%e5%ae%aa%e6%b3%95/" href="https://www.iyouport.org/%e5%b0%91%e8%af%b4%e8%af%9d%ef%bc%8c%e5%a4%9a%e5%8a%9e%e4%ba%8b-%e5%bd%93%e5%9d%8f%e8%9b%8b%e6%89%8b%e6%8c%81%e5%ae%aa%e6%b3%95/" rel="noopener" target="_blank">
    少说话，多办事
   </a>
   》，对伪民主的抵制是印度抗议活动的重点。印度人口与中国最为近似，于是印度的抗议规模可作为中国可能的抗议规模的一个最接近的参考。
  </p>
  <p class="graf graf--p">
   不论是2019年初
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/386-%e8%8b%b1%e9%87%8c%e7%9a%84%e4%ba%ba%e8%82%89%e9%95%bf%e5%9f%8e-%e7%be%8e%e5%9b%bd%e5%aa%92%e4%bd%93%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a7%86%e8%80%8c%e4%b8%8d%e8%a7%81%ef%bc%9f/" href="https://www.iyouport.org/386-%e8%8b%b1%e9%87%8c%e7%9a%84%e4%ba%ba%e8%82%89%e9%95%bf%e5%9f%8e-%e7%be%8e%e5%9b%bd%e5%aa%92%e4%bd%93%e4%b8%ba%e4%bb%80%e4%b9%88%e8%a7%86%e8%80%8c%e4%b8%8d%e8%a7%81%ef%bc%9f/" rel="noopener" target="_blank">
    386英里的人肉长城
   </a>
   ，还是去年底有史以来最强大、最广泛的抗议活动 —— 抗议者进行了长达一个多月的静坐，以反对现政府的法西斯主义政策  — — 莫迪政府推行的国家公民登记册和公民身份修正法；印度公民的抗议热情应该引起全世界反抗者的关注。
  </p>
  <p class="graf graf--p">
   在中国，我们最常听到的哀怨就是 “
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%89%93%e7%a0%b4-matrix-%e7%9a%84%e8%b7%af%ef%bd%9e%e4%b8%8eiyp%e5%af%b9%e8%af%9d2020%ef%bc%9a%e5%85%b3%e4%ba%8e%e4%b8%ad%e5%9b%bd%e5%92%8c%e7%be%8e%e5%9b%bd/" href="https://www.iyouport.org/%e6%89%93%e7%a0%b4-matrix-%e7%9a%84%e8%b7%af%ef%bd%9e%e4%b8%8eiyp%e5%af%b9%e8%af%9d2020%ef%bc%9a%e5%85%b3%e4%ba%8e%e4%b8%ad%e5%9b%bd%e5%92%8c%e7%be%8e%e5%9b%bd/" rel="noopener" target="_blank">
    中国人联合不起来
   </a>
   ”；中国人口并不少，而且平均受教育程度要高于印度，那么为什么会这样？印度活动家又是如何动员到如此多的人参加抗议活动的？
  </p>
  <p class="graf graf--p">
   要回答这个问题，不如先来看看，人们为什么拒绝参加抗议活动。
  </p>
  <p class="graf graf--p">
   大致可能会有几种最常见的原因。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     1、认为 “抗议不会带来任何改变”
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   异议的过程 —— 尤其是针对当权政府的异议 ——
   <strong class="markup--strong markup--p-strong">
    绝不是短跑，而是马拉松
   </strong>
   。抗议不会自动或立即导致压迫性政策的改变或撤销，但它们是动员群众和明确运动要求的第一步。
  </p>
  <p class="graf graf--p">
   这为积极人士向政治代表发出呼吁、并就现行政策的妥协和改变开始对话，提供了动力。抗议不会立即产生变化，但如果没有抗议，变化是绝对不可能的。
  </p>
  <p class="graf graf--p">
   我们在去年底的访谈中强调过这点。中国朋友表示 “虚无在弥漫”，人们追求一次性的成功，从而拒绝甚至抵制任何不能一次性成功的提议。这完全是不现实的。这是对抗议活动的理解性错误。
  </p>
  <p class="graf graf--p">
   我们在去年的
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e7%a7%8b%e5%90%8e%e4%ba%86-%e6%8a%97%e8%ae%ae%e7%a9%b6%e7%ab%9f%e6%9c%89%e6%b2%a1%e6%9c%89%e7%94%a8%ef%bc%9f%e7%ad%94%e6%a1%88%e5%8f%af%e8%83%bd%e4%b8%8d%e5%9c%a8%e7%9c%bc%e4%b8%8b/" href="https://www.iyouport.org/%e7%a7%8b%e5%90%8e%e4%ba%86-%e6%8a%97%e8%ae%ae%e7%a9%b6%e7%ab%9f%e6%9c%89%e6%b2%a1%e6%9c%89%e7%94%a8%ef%bc%9f%e7%ad%94%e6%a1%88%e5%8f%af%e8%83%bd%e4%b8%8d%e5%9c%a8%e7%9c%bc%e4%b8%8b/" rel="noopener" target="_blank">
    “通过BLM” 系列第4集
   </a>
   中专门讲述了这点：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    抗议活动当然有效，但通常不是以许多人所认为的方式和时间框架体现的。抗议活动有时在短期内看似失败，但抗议活动的大部分力量在于其对抗议者本身和社会其他方面的长期影响。
   </li>
   <li class="graf graf--li">
    在短期内，抗议可以在一定程度上发挥作用，人们可以震慑当局以令当局改变其行为。抗议是一种信号 — — “我们很不高兴，我们不会忍受这种方式。”
    <strong class="markup--strong markup--li-strong">
     但要想抗议奏效，“我们不会忍受” 这部分必须是可信的 —— 组织难度、人数规模和坚持不屑的精神，都可以体现这种可信度。
    </strong>
   </li>
   <li class="graf graf--li">
    抗议之所以有效，还因为它们改变了抗议者本身，使一些人从偶然的参与者变成了终生的活动家，这反过来又改变了社会。
    <strong class="markup--strong markup--li-strong">
     集体行动是一种改变生活的体验。置身于要求积极的社会变革的人海中，是一种增强能力和令人振奋的体验。
    </strong>
   </li>
   <li class="graf graf--li">
    抗议之所以有效，是因为在那一刻，抗议者心中的问题不再是短期有效还是长期有效，而是 — — 在严重的不公正现象展开的时候，人们是否可以袖手旁观多一天。而这也许就是抗议活动最有力的手段：当问题如此之大以致于抗议者不再去计算是否有效，而是感觉到在道义上不得不参与其中。
   </li>
  </ul>
  <p class="graf graf--p">
   抗议真的有效吗？只有当人们不再思考这个问题的时候，你就会看到希望。
  </p>
  <p class="graf graf--p">
   历史上有大量的例子可以证明这点：1913年美国的妇女选举权游行，当时有8000名妇女为争取选举权而加入抗议，掀起了一场更广泛的选举权运动，最终才促成了妇女享有选举代表的权利；1930年圣雄甘地的 “食盐游行” 成为一场更大范围的公民抗命运动的关键事件，最终促使印度从英国殖民统治中获得独立。去年底，印度全国各地反 NRC-CAA 抗议浪潮除了静坐和包围等战术之外，还采取了向最高法院请愿的形式，对这些政策提出质疑，最高法院不得不答应举行听证会，以解决抗议者的关切。
  </p>
  <p class="graf graf--p">
   再一次，
   <strong class="markup--strong markup--p-strong">
    指望一次性的 “成功” 是不合理的，您所挑战的恶行越是深厚，就越是如此；某一场胜利也并非终点站，很多时候您必须反复多次赢得同样的战斗。持衡能力最强大的反抗者，将是最终赢家
   </strong>
   。
  </p>
  <h3 class="graf graf--p">
   <span style="color: #339966;">
    <strong class="markup--strong markup--p-strong">
     2、“抗议只是具有相同信念的人之间的回声室”
    </strong>
   </span>
  </h3>
  <p class="graf graf--p">
   我们在中国也听到过类似的观点和说法，比如 “反贼圈子”，就好像异议人士是个小圈子，和外人互不相关？事实上并非如此。反抗者的工作与所有人相关，他们的努力是在为所有人争取未来。
  </p>
  <p class="graf graf--p">
   活动家必须能找到真正的动员触发点，这就是为什么我们在 《
   <a class="markup--anchor markup--p-anchor" data-href="https://www.iyouport.org/%e6%89%93%e7%a0%b4-matrix-%e7%9a%84%e8%b7%af%ef%bd%9e%e4%b8%8eiyp%e5%af%b9%e8%af%9d2020%ef%bc%9a%e5%85%b3%e4%ba%8e%e4%b8%ad%e5%9b%bd%e5%92%8c%e7%be%8e%e5%9b%bd/" href="https://www.iyouport.org/%e6%89%93%e7%a0%b4-matrix-%e7%9a%84%e8%b7%af%ef%bd%9e%e4%b8%8eiyp%e5%af%b9%e8%af%9d2020%ef%bc%9a%e5%85%b3%e4%ba%8e%e4%b8%ad%e5%9b%bd%e5%92%8c%e7%be%8e%e5%9b%bd/" rel="noopener" target="_blank">
    打破matrix的路
   </a>
   》中特别强调
   <strong class="markup--strong markup--p-strong">
    知识分子要像活动家那样思考
   </strong>
   —— 活动家从知识分子的论述中获得动员能量，如果知识分子不能将论述主题放在有利于调动最广泛的支持度和行动力的角度上，活动家的动员效果肯定大打折扣。
  </p>
  <p class="graf graf--p">
   当然，活动家之间肯定具有相同或相似的信念，有共同目标，并具备足够的朝着这一目标努力的热情，这会让他们看起来 “内部抱团”；但这并不是因为反抗运动是 “这一小群人自己的事”。这是异议知识分子应该出现的地方，准确地告诉人们：
  </p>
  <ul class="postList">
   <li class="graf graf--li">
    抵制面部识别的努力
    <a class="markup--anchor markup--li-anchor" data-href="https://www.iyouport.org/%e8%a2%ab%e6%b5%81%e8%a1%8c%e8%af%ad%e6%8e%a9%e7%9b%96%e7%9d%80%e7%9a%84-%e6%8a%80%e6%9c%af%e4%b8%93%e5%88%b6%e4%b8%bb%e4%b9%89-%e6%97%b6%e4%bb%a3/" href="https://www.iyouport.org/%e8%a2%ab%e6%b5%81%e8%a1%8c%e8%af%ad%e6%8e%a9%e7%9b%96%e7%9d%80%e7%9a%84-%e6%8a%80%e6%9c%af%e4%b8%93%e5%88%b6%e4%b8%bb%e4%b9%89-%e6%97%b6%e4%bb%a3/" rel="noopener" target="_blank">
     并不只是技术本身的问题
    </a>
    ，也不是仅仅为了保护敏感人士；它是
    <strong class="markup--strong markup--li-strong">
     所有人的基本人权和尊严的问题，是全社会的反压迫行动
    </strong>
    ；
   </li>
   <li class="graf graf--li">
    <a class="markup--anchor markup--li-anchor" data-href="https://www.patreon.com/posts/lai-zi-zhong-guo-38773296" href="https://www.patreon.com/posts/lai-zi-zhong-guo-38773296" rel="noopener" target="_blank">
     反抗警察暴力
    </a>
    是反抗私刑，每一个不想蹲冤狱、不想被殴打和刑讯逼供的人都应该加入这场抗议，爱党爱国老实听话并不能保护你；
   </li>
   <li class="graf graf--li">
    抵抗996、为富士康

   ，不要强调一次性的胜利，更不要鼓励牺牲；只要有足够多的人保持清醒的头脑和拒绝服从，最终就一定能获得成功。⚪️
  </p>

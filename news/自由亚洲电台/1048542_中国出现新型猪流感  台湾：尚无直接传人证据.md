[中国出现新型猪流感  台湾：尚无直接传人证据](https://www.rfa.org/mandarin/yataibaodao/huanjing/hx2-06302020111459.html)
------

<p> </p><p>美国一份科技期刊上日前刊登论文说，在中国研究发现一种新型猪流感，一旦突变，可能出现人传人，引发全球大流行。台湾疾管署高度重视，召开记者会指出，尚未有猪流感传人的直接证据，但是必须持续监测。</p><p>台湾中央通讯社报导，美国《国家科学院学报》（PNAS）29日刊登研究论文指出，中国研究人员发现一种新型猪流感，一旦突变，可能出现人传人，引发全球大流行。这种猪流感病毒被称为“G4 EA H1N1”，是从造成2009年大流行的H1N1病毒株基因演变而来，带有“会高度适应以感染人类的一切重要特征”。</p><p> </p><ul><li><b><a class="external-link" href="http://www.rfa.org/mandarin/yataibaodao/huanjing/yl-04242019103021.html"><span 中央社报导还指出，撰写研究论文的中国疾病预防中心科学家表示，2011至2018年间，研究人员从中国10个省的屠宰场与一间兽医院中，采集了3万头猪只的鼻腔拭子，并从中分离出179种猪流感病毒。这些病毒大多属于一种自2016年起就在猪只间流行的新病毒。研究人员观察到G4具高度传染性，会在人类细胞中复制，在雪貂身上造成的症状，也比其他病毒更为严重。根据血液检测，已有10.4%的养猪场工作人员感染G4，另有4.4%的一般民众似也感染，因此可知这种病毒或会从动物传染给人，但目前尚无证据显示会人传人。</p><p> </p><p><div 英媒BBC报导指出，这个新型猪流感病毒被称为“G4 EA H1N1”，可以在人类呼吸道内的细胞中生长和繁殖。研究团队发现，最近感染的证据始于中国屠宰场和养猪业人士。现有的流感疫苗似乎无法抑止新病毒，但疫苗仍可以根据需要进行调整。<br/> <br/><b>中国称血清检测有一成养猪场员工感染G4 </b><br/> <br/> 从中国大陆传出的新冠肺炎尚在全球流行，这一则中国科学家在中国发现新型猪流感的新闻，在台湾引发高度关注。台湾疾管署30日召开记者会，副署长庄人祥表示，目前没有人类感染个案的相关报告。<br/> <br/> 疾管署医师邹宗珮在记者会上也说，目前还没有动物传人的直接证据。“他们是由血清流行病学间接证明这些猪场工作者可能有曝露到这些病毒产生抗体反应，所以目前并没有有症状的业者经由PCR阳性确诊这个病毒，所以没有动物传染的证据，当然人传人的证据也就更缺乏。”<br/> <br/><b>台湾农政单位：病毒不会跑到猪肉里 可安心食用</b><br/> <br/> 邹宗珮说，这份研究报告也未提及血清有抗体反应者会出现什么症状。她还说，重要新兴病毒血清检验方法，并没有标准化，在报告里有些抗体非常高，有些不达标准，很微弱。</p><p> </p><p><div class="image-inline captioned" style="width:640px;"><div style="width:640px;"><img 邹宗珮并指，新兴传染病当中， 动物来源是主要的威胁，持续对动物来源病毒做监测有其必要，也呼吁禽畜工作者必须做好个人防护，如戴口罩、穿防护衣、勤洗手、戴护目镜，都可以降低被传染的机率。<br/> <br/> 对中国研究人员发现新型猪流感病毒（G4 EA H1N1），台湾主管农政事务的农业委员会防疫检疫局表示，猪流感非属世界动物卫生组织（OIE）表列疾病，也非台湾法定动物传染病。本次中国发现的猪流感病毒，仅利用血清学资料推判有猪传人的风险，并无直接证据显示可由猪只传给人类或人传人。<br/> <br/> 台湾防检局指出，依照OIE资料显示，猪流感仅是猪只轻微疾病，病毒不会跑到猪肉里，吃猪肉感染的风险可以被忽略。民众无需担心，可放心享用猪肉美食。</p><p> </p><p><div class="image-inline captioned" style="width:622px;"><div style="width:622px;"><img 台湾大学兽医系名誉教授、有“猪博士”称号的赖秀穗，接受自由亚洲电台采访表示，对于目前传出所谓“G4”的情况他并不了解，也不明白为何以此命名，一般猪流感会命名“Ｈ”多少“N”多少。猪流感、人流感、禽流感很容易互相传染，症状大都是跟流感一样，如发烧、咳嗽、打喷嚏，上呼吸道症状。<br/> <br/> 根据报导，中国发现的新型猪流感，是从造成2009年大流行的H1N1病毒株基因演变而来。赖秀穗回顾2009年大流行的H1N1说：“2009年发现猪流感在墨西哥，再传到美国，这个猪流感血清型是H1N1，主要是感染小孩子为主，小孩子都没有抗体，老年人、年长者很少部份有抗体，大部份的人都没有抗体，才会几个月的时间就流行到全世界。流感（致死率）大都百分之零点几，但是那个时候猪流感大部份人都没有抗体，造成全世界20几万人死亡，光美国就死掉2万多人。”<br/> <br/> 赖秀穗提及，一般禽流感比猪流感更复杂、变异重组情况更严重。他呼吁民众先不要恐慌，如果真正造成猪流感大流行，防范的方式就跟现在武汉肺炎一样，戴口罩、勤洗手，减少出入公共场所。<br/> <br/> <br/> 自由亚洲电台记者 夏小华  台北报导  责编 胡力汉 申铧  网编 瑞哲</p>
